Causal Forcing++: Scalable Few-Step Autoregressive Diffusion Distillation for Real-Time Interactive Video Generation（因果驱动++：面向实时交互视频生成的可扩展少步自回归扩散蒸馏）

实时交互式视频生成需要低延迟、流式处理和可控的滚动生成。现有自回归扩散蒸馏方法通过将双向基础模型蒸馏为少步自回归模型，在分块4步机制中取得了显著成果，但受限于粗粒度的响应和不可忽略的采样延迟。本文研究了一项更具挑战性的设置：帧级自回归，仅需1-2步采样。在此机制下，我们识别出少步自回归学生模型的初始化是关键瓶颈：现有策略要么目标不对齐，要么无法完成少步生成，要么成本过高难以扩展。我们提出Causal Forcing++，一种原则性强且可扩展的流水线，采用因果一致性蒸馏（causal CD）进行少步自回归初始化。核心思想是，因果一致性蒸馏学习与因果ODE蒸馏相同的自回归条件流映射，但通过相邻时间步之间的单个在线教师ODE步骤获取监督，从而避免预计算和存储完整PF-ODE轨迹。这使得初始化更高效且更易于优化。由此产生的流水线\ours在帧级2步设置下，于VBench总分、VBench质量和VisionReward上分别超出当前最优分块4步因果驱动方法0.1、0.3和0.335，同时将首帧延迟降低50%，阶段2训练成本减少约4倍。我们进一步将该流水线扩展到基于Genie3理念的动作条件世界模型生成。项目页：https://github.com/thu-ml/Causal-Forcing 和 https://github.com/shengshu-ai/minWM 。

结论：受蒙特卡洛启发的分析先验在Kvasir-Capsule上于6个种子中提供了微小但方向一致的宏AUC提升，最大的稳健单类提升在淋巴管扩张症上；蒸馏变体可在纯3通道RGB上运行，并产生免费的、可解释的热力图。

PreFT: Prefill-only finetuning for efficient inference（PreFT：仅预填充微调实现高效推理）

尽管当前文生图（T2I）模型进步迅速，但主要依赖单步生成范式，难以处理复杂语义，且参数规模扩展带来的收益递减。近期多步推理方法虽有前景，但受限于缺乏验证的无依据规划幻觉、单一的事后反思、长上下文优化不稳定以及极高的推理延迟。为克服这些瓶颈，本文提出闭环视觉推理（CLVR）框架，将视觉-语言逻辑规划与像素级扩散生成深度耦合。CLVR引入了具备步骤级视觉验证的自动化数据引擎，用于合成可靠的推理轨迹；并提出代理提示强化学习（PPRL），通过将交错的多模态历史蒸馏为显式奖励信号以实现精确因果归因，从而解决长上下文优化不稳定问题。此外，为缓解迭代去噪带来的严重延迟瓶颈，本文提出Δ-空间权重合并（DSWM）方法，将对齐权重与现成的蒸馏先验融合，将每步推理成本降至仅4个NFE，且无需昂贵的再蒸馏。大量实验证明，CLVR在多个基准上优于现有开源基线，并接近专有商业模型的性能，为复杂视觉生成解锁了通用的测试时缩放能力。

Let Robots Feel Your Touch: Visuo-Tactile Cortical Alignment for Embodied Mirror Resonance（让机器人感受你的触碰：用于具身镜像共振的视触皮层对齐）

该论文提出了一种名为“镜像触觉网络（Mirror Touch Net）”的计算框架，旨在赋予机器人镜像触觉能力。其核心在于通过多层级约束（语义、分布和几何对齐），强制视觉与触觉表征在结构上对应（即皮层层面对齐）。利用该框架，机器人能够仅凭RGB图像预测其机械手上1140个触觉传感器上的毫米级触觉信号。研究表明，这些约束重塑了视觉表征，使其与触觉流形的几何结构保持一致，从而降低了跨模态映射的复杂度。将这一对齐框架扩展至人类手部的跨域观察后，机器人能够实现触觉预测并对观察到的触碰产生反射式反应。该工作将视触共振的神经原理与机器人感知联系起来，为实现预见性触碰和共情式人机交互提供了可解释的途径。

SR-Prominence: A Crowdsourced Protocol and Dataset Suite for Perceptually-Weighted Super-Resolution Artifact Evaluation（基于众包协议和数据集套件的感知加权超分辨率伪影评估）

现代图像超分辨率方法虽能生成细节丰富、视觉上吸引人的结果，但常引入视觉伪影（不自然纹理和扭曲），降低感知质量。不同伪影的感知影响差异很大（有些几乎不可见，有些则非常刺眼），但现有检测方法对所有伪影一视同仁。本文提出将“伪影显著性”作为评估目标，定义为认为高亮区域包含明显伪影的观众比例。我们设计了众包标注协议，并构建了SR-Prominence数据集套件，包含来自DeSRA、Open Images、Urban100及无真实参考的Urban100-HR场景的3,935个伪影掩码及其显著性标注。重新标注DeSRA发现，其48.2%的实验室二元伪影未被多数观众注意到。基于该套件，我们评估了SR伪影检测器、图像质量指标及SR方法，发现经典全参考指标（尤其是SSIM和DISTS）能提供出人意料的强局部显著性信号，而无参考IQA方法和专用伪影检测器常难以跨数据集和参考设置泛化。SR-Prominence附带客观评分协议，新指标无需额外众包即可在该套件上基准测试。数据和协议共同推动SR伪影评估从二元缺陷存在性转向感知影响。数据集地址：https://huggingface.co/datasets/imolodetskikh/sr-artifact-prominence

Evidential Reasoning Advances Interpretable Real-World Disease Screening（证据推理推动可解释的真实世界疾病筛查）

当前医学影像筛查模型可解释性差、性能欠佳，且缺乏引用历史病例或提供透明推理路径的机制。为此，本文提出了EviScreen——一个基于证据推理的筛查框架，它利用来自历史病例的区域级证据。EviScreen通过从双知识库中检索到的区域证据，提供回顾性可解释性。随后，基于该证据机制，证据感知推理模块结合当前病例与历史病例的证据进行预测，从而提升筛查性能。此外，EviScreen不依赖事后显著性图，而是利用对比检索得到的异常图来增强定位可解释性。该方法在自建的真实世界疾病筛查基准上取得了更优性能，在临床级召回率下实现了显著更高的特异性。代码已公开在：https://github.com/DopamineLcy/EviScreen。

Quantitative Video World Model Evaluation for Geometric-Consistency（几何一致性视频世界模型的量化评估）

生成式视频模型常被研究为隐式世界模型，但评估其生成的物理合理3D结构与运动仍具挑战。现有评估依赖人工判断或学习型评分器，存在主观性强且难以诊断几何缺陷的问题。本文提出PDI-Bench（透视畸变指数），一个用于审计生成视频几何一致性的量化框架。该框架通过分割与点跟踪获取对象级观测，利用单目重建将其提升至3D世界坐标，并计算一组射影几何残差，涵盖三个失效维度：尺度-深度对齐、3D运动一致性与3D结构刚性。为支持系统评估，我们构建了PDI-Dataset，包含多种旨在考验这些几何约束的场景。评估表明，PDI能揭示当前最先进视频生成器中常见的几何特定失效模式，这些模式无法被常见感知指标捕捉，并为推动物理真实视频生成与物理世界模型发展提供了诊断信号。代码与数据集：https://pdi-bench.github.io/。

ICED: Concept-level Machine Unlearning via Interpretable Concept Decomposition（ICED：通过可解释概念分解实现概念级机器遗忘）

视觉-语言模型（VLM）中的机器遗忘通常在图像或实例层面进行，导致难以在不影响无关语义的情况下精确移除目标知识，因为单张图像常包含多个纠缠的概念。本文提出一种可解释的概念级遗忘框架，利用多模态大语言模型从遗忘集构建紧凑的任务特定概念词汇。视觉表示被分解为稀疏、非负的语义概念组合，从而为细粒度知识操控提供显式接口。基于此分解，该方法将遗忘形式化为概念级优化：选择性抑制目标概念，同时保留实例内的非目标语义及全局跨模态知识。在域内和域外遗忘设定下的广泛实验表明，与现有VLM遗忘方法相比，该方法能实现更全面的目标遗忘，更好保留同一图像中的非目标知识，并维持有竞争力的模型效用。

AnyBand-Diff: A Unified Remote Sensing Image Generation and Band Repair Framework with Spectral Priors（AnyBand-Diff：一种融合光谱先验的统一遥感图像生成与波段修复框架）

现有扩散模型虽在生成逼真图像方面取得显著进展，但直接应用于遥感数据时常忽视其内在物理规律，导致光谱失真与辐射不一致性，严重限制了生成数据的科学用途。为解决此问题，本文提出AnyBand-Diff——一种新颖的、以光谱先验引导的扩散框架，专为鲁棒的光谱重建而设计。其核心包括：1）一个集成双重随机掩码策略的条件扩散掩码主干，使模型能从不完整波段子集中恢复完整光谱信息；2）一种物理引导采样机制，利用可微物理模型的梯度显式引导去噪轨迹收敛于物理合理的解空间；3）一种多尺度物理损失函数，在像素、区域和全局层面联合施加严格约束。大量实验证明，AnyBand-Diff在生成可靠图像与实现精确光谱重建方面具有有效性，推动了面向地球观测的物理感知生成方法的发展。

ATLAS: Agentic or Latent Visual Reasoning? One Word is Enough for Both（ATLAS：代理式还是潜视觉推理？一个词足以兼顾两者）

视觉推理常与中间视觉状态交织，已成为有前景的研究方向。直接通过统一模型在推理过程中生成图像虽直观，但计算成本高且架构复杂。近期替代方案包括通过代码或工具调用的代理式推理，以及基于可学习隐藏嵌入的潜推理。然而，代理式方法因外部执行导致上下文切换延迟，潜推理则缺乏任务泛化性且难以通过自回归并行化训练。为结合两者优势并克服局限，本文提出ATLAS框架：一个名为“功能标记”的单一离散词，同时作为代理操作和潜视觉推理单元。每个功能标记关联一个内化的视觉操作，无需视觉监督，且作为分词器词汇表中的标准标记，可通过下一词预测生成。该设计避免了冗长的中间视觉内容生成，同时保持了与标准可扩展有监督微调和强化学习的兼容性，无需架构或方法上的修改。为解决强化学习中功能标记稀疏性问题，我们引入潜锚定GRPO（LA-GRPO），通过静态加权辅助目标锚定功能标记，提供更强的梯度更新，从而稳定训练。大量实验和分析表明，ATLAS在挑战性基准上取得卓越性能，同时保持清晰的可解释性。我们希望ATLAS能为未来视觉推理研究提供新范式。

A Unified Knowledge Embedded Reinforcement Learning-based Framework for Generalized Capacitated Vehicle Routing Problems（面向广义容量车辆路径问题的统一知识嵌入强化学习框架）

容量车辆路径问题（CVRP）是基于NP难的基础问题，广泛应用于物流运输领域。现实中的CVRP常包含多种目标与复杂约束（如时间窗或回程需求），因而需要统一求解框架。现有强化学习方法虽在组合优化中表现优异，但因采用端到端学习且缺乏显式问题求解知识，导致解质量受限。本文提出一种受"先规划路径、再聚类"启发式方法启发的知识嵌入框架。该框架在两层嵌入知识：（1）将CVRP分解为路径优先与聚类次优子问题；（2）利用动态规划求解次优子问题，其输出结果指导基于强化学习的构造性求解器解决优先子问题。为缓解问题分解造成的部分可观测性，我们引入了统一的历史增强上下文处理模块。大量实验表明，该框架相比最先进的基于学习方法实现了更优的解质量，与经典启发式方法的差距更小，在多种CVRP变体上展现出强泛化能力。

Agentic Recommender System with Hierarchical Belief-State Memory（具有层次化信念状态记忆的智能推荐系统）

记忆增强的大语言模型代理虽已推动了个性化推荐技术的发展，但现有方法普遍采用扁平化记忆表示，混淆了瞬时信号与稳定偏好，且均未提供完整的记忆演化生命周期管理。我们提出的MARS（记忆增强智能推荐系统）框架将推荐视为部分可观测问题，维护结构化的信念状态，逐步将含噪声的行为观测抽象为用户偏好的紧凑估计。该信念状态分为三层：事件记忆缓冲原始信号，偏好记忆维护具有显式强度与证据追踪的细粒度可变块，画像记忆将所有偏好提炼为连贯的自然语言描述。基于大语言模型规划器自适应调度六个完整生命周期操作（提取、强化、弱化、整合、遗忘与重构），而非采用固定间隔启发式方法。在四个InstructRec基准域上的实验表明，MARS在HR@1和NDCG@10指标上分别较最强基线平均提升26.4%和10.3%，且在动态演化场景中通过智能调度获得进一步增益。

NeuroAtlas: Benchmarking Foundation Models for Clinical EEG and Brain-Computer Interfaces（神经图谱：为临床脑电图和脑机接口的预训练基础模型建立基准测试）

本文介绍NeuroAtlas，这是迄今为止规模最大的脑电图（EEG）基准测试，包含42个数据集和26万小时数据，覆盖临床脑电图（癫痫、睡眠医学、脑年龄估计）和脑机接口领域。研究提出了三项核心发现：一，专门针对EEG的预训练基础模型并未持续优于通用时间序列模型，后者既没有采用以EEG为中心的架构，也未在EEG数据上预训练；二，标准机器学习指标不足以评估临床效用，因此研究在癫痫、睡眠和脑年龄领域分别评估了事件级决策质量、睡眠脑电图衍生特征和脑年龄差距等更适宜的指标；三，同一领域内模型排名和性能可能存在显著差异。结论认为，现有预训练模型的表现大体持平，仅有少数模型具有狭窄优势，目前尚未能实现即插即用的统一EEG模型。NeuroAtlas揭示了这一差距，并为下一代统一EEG预训练模型提供了数据集和评估指标。

Boosting Reinforcement Learning with Verifiable Rewards via Randomly Selected Few-Shot Guidance（通过随机选择的少量示例引导提升基于可验证奖励的强化学习）

基于可验证奖励的强化学习（RLVR）在数学和编程等任务中成功开发了大语言模型（LLM），但在困难问题上样本效率较低，难以生成正确轨迹。已有研究通过演示引导的RLVR解决此问题，即在RL失败时进行监督微调（SFT），但SFT通常需要大量昂贵数据。本文提出FEST算法，一种基于少量演示引导的RLVR方法，仅需从SFT数据集中随机选取128个演示即可取得显著效果。研究发现三个关键成功要素：监督信号、在策略信号以及防止多轮训练过拟合的衰减权重。在多个基准测试中，FEST使用远少于完整数据集的SFT数据即可超越基线，甚至匹配其性能。

EnergyLens: Predictive Energy-Aware Exploration for Multi-GPU LLM Inference Optimization（EnergyLens：面向多GPU大语言模型推理优化的预测性能量感知探索）

本文提出了 EnergyLens，一个用于大语言模型推理优化的端到端能量感知框架。随着LLM规模扩大，预测和降低其能耗对可持续性至关重要，但现有方法要么需要生产级代码和高成本分析，要么无法准确捕捉多GPU能量行为。因此，当全面分析不可行时，实践者缺乏工具来优先选择优化方案或部署配置。EnergyLens 通过一个直观的、基于einsum的接口解决了这一问题，该接口捕捉LLM的规格（包括融合、并行化和计算通信重叠），并结合了负载不均衡感知的MoE建模以及经验驱动的多GPU通信能耗模型。我们在 Llama3 和 Qwen3-MoE 上，针对张量并行和专家并行配置进行了验证，多GPU预填充和解码能耗的平均绝对百分比误差在9.25%至13.19%之间，在Megatron风格的重叠配置中SM分配的平均绝对百分比误差为12.97%。我们的能量驱动探索显示，不同配置下的预填充和解码效率能耗变化高达1.47倍和52.9倍，这推动了分布式服务的应用。我们还发现，仅凭直觉难以优化计算通信重叠，但 EnergyLens 能正确识别出帕累托最优的重叠配置。

Learning from Language Feedback via Variational Policy Distillation（通过变分策略蒸馏从语言反馈中学习）

针对可验证奖励的强化学习（RLVR）中稀疏结果信号导致的复杂推理探索瓶颈问题，现有在线自蒸馏方法依赖固定的被动教师解读语言反馈，但随学生策略提升，教师的零样本评估能力停滞，从而终止学习。为此，我们提出变分策略蒸馏（VPD）框架，将语言反馈学习形式化为变分期望最大化（EM）问题：E步通过自适应信任域更新主动优化教师策略，将文本反馈转化为动态改进的目标令牌分布；M步让学生在自身策略轨迹上内化该密集分布引导。通过持续提升教师从文本批评中提取可操作信号的能力，VPD克服了被动蒸馏的局限。在科学推理和代码生成任务的多样化诊断反馈评估中，VPD持续优于标准RLVR及现有自蒸馏基线。最后，通过在严格数学推理和冷启动场景下的压力测试，我们揭示了相较于纯环境驱动RL，反馈驱动自蒸馏的根本性边界。
