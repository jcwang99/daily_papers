Adaptive Coarse-to-Fine Subgoal Refinement for Long-Horizon Offline Goal-Conditioned Reinforcement Learning（面向长周期离线目标条件强化学习的自适应由粗到细子目标精化）

Learning When to Think While Listening in Large Audio-Language Models（大型音频语言模型中的“边听边思考”时机学习）

最近大型音频语言模型（LALMs）的进展使得实时流式语音交互越来越实用。在这种场景下，推理质量与响应速度紧密耦合：延迟推理至语音结束可提高答案质量，但会带来用户可见的响应延迟；过早回答则可能在决定性证据到来前做出错误判断。

本文提出了一种针对LALMs的可学习的“等待-思考-回答”控制模式。受人类对话的渐进性启发，该控制器基于部分音频证据来决定何时等待、何时外化紧凑的推理更新以及何时回答。以Qwen2.5-Omni-7B为基础模型，我们从语音推理数据中构建了对齐的“等待-思考-回答”轨迹，通过监督微调（SFT）训练控制器，随后应用解耦剪辑与动态采样策略优化（DAPO）。

针对大语言模型在线强化学习中探索与利用失衡导致优化不稳定及性能次优的问题，本文提出以下核心贡献：

3. 提出信息瓶颈驱动的树形策略优化（IB-TPO）框架：将IB-Score作为细粒度优化目标，并设计IB-guided树形采样策略——在相同token预算下多生成50%的轨迹，同时复用树结构实现高效IB-Score蒙特卡洛估计。
4. 在标准基准测试中，该方法比GRPO基线提升2.9%-3.6%，并超越其他最先进的在线RL方法。

多模态检索增强生成（MMRAG）通过融合外部视觉、文本与结构知识，已成为增强多模态大语言模型在知识密集型问答中能力的强大范式。然而，现有MMRAG框架存在检索噪声大且相关性差、跨模态语义错位、缺乏自适应推理以及局部与全局上下文生成不连贯等关键局限。我们提出CogniVerse——一种新颖的MMRAG框架，通过受认知启发且数学严谨的方法解决上述挑战。该框架模拟人类推理，整合三个协同模块：（1）认知反思模块，动态评估检索必要性并过滤相关多模态内容，减少噪声和计算开销；（2）多模态检索模块，利用信息几何在黎曼流形中对齐嵌入，并通过谱图论精炼知识图谱，确保检索精确连贯；（3）层次化生成模块，采用基于最优传输的损失函数，平衡词元级别准确性与全局语义连贯性。大量实验表明，CogniVerse在准确性和连贯性上均显著优于现有最优系统，同时降低了检索延迟。

Parameter-Efficient Generative Modeling with Controlled Vector Fields（基于受控向量场的参数高效生成建模）

受Chow-Rashevskii定理启发，本文提出了一种连续时间生成建模框架。该框架通过一组固定的向量场和学习的标量控制来构建表达性流，而非学习无约束的高维向量场。当固定场满足括号生成条件时，其李代数张成整个空间，从而仅需少量学习控制通道即可实现表达性传输，提供了一种参数高效的几何替代方案。这种解耦设计使得学习的标量输出通道数独立于环境维度，形成了结构化、可解释的生成模型。本文提出了一个表达性原理，证明在适当的可控性与适定性假设下，此类受控流能将源分布传输至目标分布。模型通过连续归一化流似然目标进行训练，并在合成分布上进行了概念验证实验。

A Unified and Reproducible Experimentation Framework for Speech Understanding（一种统一且可复现的语音理解实验框架）

现有的语音基础模型和语音大语言模型推动了语音理解的发展，但在模型选择时，由于后处理不匹配导致评估结果不可比，且训练结果难以跨数据规模和流程复现，从而受到阻碍。我们提出了SURE，一个统一的实验框架，用于标准化预测格式、归一化和评分。SURE在代表性任务中，在真实的声学和语言压力下，评估了从传统流程到语音大语言模型的多种范式中的强系统。除评估外，SURE引入了一种智能体辅助的训练转换流程，将论文和代码映射为在统一协议下、基于匹配的开源数据子集中的版本化、可运行的训练流程。总之，SURE提升了面向部署的评估的可比性和可复现性。

MemTrace: Tracing and Attributing Errors in Large Language Model Memory Systems（MemTrace：大语言模型记忆系统中的错误追踪与归因）

记忆对于大语言模型实现长程推理至关重要，但现有记忆系统仍然不可靠且难以调试。本文研究了大语言模型记忆系统中的错误追踪与归因这一新问题。我们提出了一种新颖框架，将记忆流水线转化为可执行的记忆演化图，从而实现对操作信息流的细粒度追踪。随后构建了MemTraceBench基准数据集（涵盖Long-Context、RAG、Mem0、EverMemOS等代表性记忆系统），用于系统研究记忆失败模式。我们还引入了一种自动归因方法，通过迭代追踪操作子图来定位任何失败案例的根本原因。分析表明，记忆失败具有系统性，源于操作层面问题（如信息丢失和检索错位）。关键的是，我们利用这些细粒度归因信号指导下游提示优化，建立了一个闭环系统，可自动修复故障并将端任务性能提升高达7.62%。代码将在https://github.com/zjunlp/MemTrace 发布。

ForestHG-Trace: Traceable Long-Horizon Ecological Reasoning over Large-Scale Forest Scenes（ForestHG-Trace：大规模森林场景下可溯的长时序生态推理）

遥感问答（RS-QA）在大规模森林场景中需超越直接语义预测，进行多步筛选、数值聚合、邻域推理及可验证证据分析。ForestHG-Trace框架将多模态NEON森林场景表示为生态超图，支持树实例、空间单元、语义组及邻域关系的高阶推理。LLM引导的智能体调用确定性工具执行读取、过滤、扩展、聚合、比较及审计操作，生成可回放的执行轨迹与紧凑证据记录。研究构建了可执行基准ForestTraceQA。实验表明，ForestHG-Trace在答案准确性和执行忠实度上显著优于单步基线及场景图智能体，而执行深度是长时序生态问答的主要瓶颈。

Is Backpropagation Optimal? When Synthetic Gradients Improve Sample Efficiency（反向传播是最优的吗？合成梯度何时提升样本效率）

反向传播是人工神经网络的默认学习规则，在可微条件下常被视为既定方法。本研究从样本效率的理论角度重新审视这一惯例。我们提出了一个用于计算图上基于损失和基于奖励学习的统一向量化反馈框架，其中合成梯度成为反向传播的自然替代方案。我们刻画了合成梯度能达到比反向传播更低梯度估计均方误差的条件，并构造示例表明这种样本效率优势可以任意大。针对上下文赌博机和强化学习任务的实验验证了我们理论发现的潜力。

HarmoVid: Relightful Video Portrait Harmonization（HarmoVid：令人愉悦的视频人像重光照调和）

我们提出了一种方法，用于调和前景视频的光照以匹配目标背景场景，调整阴影、色调和照明强度（重光照调和）。与图像不同，为视频获取标注数据（即在不同光照条件下记录相同动作）实际上不可行且不可扩展。虽然创建此类配对数据的一种方式是逐帧应用现有的基于图像的调和模型，但结果通常会出现严重的时序抖动。我们通过引入一种新颖的照明去闪烁模型来克服这一问题，该模型能稳定全局和局部的照明闪烁伪影。我们的视频扩散模型利用这些升级后的去闪烁数据（包含大量真实与合成视频）进行学习，从而生成高质量的视频调和结果。我们进一步提出了一种非对称的Alpha蒙版条件技术，以从真实视频中学习清晰的边界。实验表明，与先前的基于图像和基于视频的调和方法相比，我们的模型实现了强时序一致性、自然度、更清晰的边界以及物理上合理的照明行为，同时保持了强大的重光照表现力。

When Think-with-Image Meets Safety: What Determines Multimodal Jailbreak Robustness?（当图文推理遇上安全：什么决定了多模态越狱鲁棒性？）

图文推理正成为大型视觉-语言模型的新推理范式，但其安全性尚不明确。现有系统包含多种流程设计，包括直接生成回答、纯文本前置回合、视觉状态操纵以及显式外部图像工具调用。本文探究哪种范式能提升多模态越狱鲁棒性及其原因。实验表明，显式图像工具交互在所有评估模型中取得了最低的攻击成功率，平均相对降低了约30%的越狱成功率。这一发现起初令人惊讶：即使手动覆盖返回的图像工具输出或该输出本身不安全，攻击成功率仍保持低位；但在纯文本前置回合控制下，攻击成功率回升至接近直接问答的水平。结果表明，低攻击成功率并非由返回图像的安全语义或仅凭文本图像工具轨迹所能解释。为此，本文引入了图像工具安全向量框架，将图像工具调用建模为隐藏表征向安全相关方向的残差偏移。表征层分析与激活干预支持了这一解释。总体而言，我们的结果表明，显式图像工具交互是提升越狱鲁棒性的一种有前景的设计模式，同时也推动了针对特定流程的安全性评估。

PILOT: A Data-Free Continual Learning Approach for Real-Time Semantic Segmentation via Boundary Guidance（PILOT：一种基于边界引导的无数据实时语义分割持续学习方法）

本文针对实时语义分割模型在动态环境中增量学习新类别时面临的灾难性遗忘问题，提出了一种新颖的持续学习框架PILOT。该框架专为先进实时分割模型PIDNet设计，通过冻结原网络参数并引入一个并行导数分支来捕捉新类别的高频边界信息，实现了一种实时且轻量化的学习策略。该方法仅需新类别数据即可适应新语义类别，同时保留旧知识，显著降低了训练开销。实验结果表明，PILOT在有效分割新类别的同时，对原始基类保持了较高的平均交并比，性能优于该领域所有主流持续学习方法，并显著缓解了灾难性遗忘，对推理延迟影响极小，从而维持了实时性能。

Computational Modeling of Antibody-Antigen Complexes: PLM-Based and MSA-Based Approaches（抗体-抗原复合物的计算建模：基于PLM与基于MSA的方法）

本论文探究了抗体相关计算任务为何更具挑战性，并沿两个互补方向提出了改进方法。首先，针对基于蛋白质语言模型（PLM）的抗体与抗体-抗原结构预测方法，使用多种PLM嵌入在抗体单体预测中取得了最佳CDR-H3精度；但该方法在复合物预测中无法泛化，因为缺乏抗体与抗原间的共进化信号，单序列PLM表示无法可靠识别结合界面。其次，针对抗体-抗原复合物预测，开发了两种基于多重序列比对（MSA）的干预措施：一是结合CDR聚焦过滤与从更大序列数据库中恢复深度的MSA精炼；二是收敛感知循环，选择稳定的中间循环状态用于最终扩散采样。这些干预措施在预留抗体-抗原测试集上相比AlphaFold3基线取得了一致提升。由于该方法仅修改了MSA构建与循环行为而非模型参数，因此无需重新训练或获取权重即可应用。

AnyMo: Scaling Any-Modality Conditional Motion Generation with Masked Modeling（AnyMo：利用掩码建模实现任意模态条件运动生成）

条件人体运动生成是计算机视觉与机器人领域的基础挑战。现有方法受限于固定模态配置和任务特定架构，跨模态交互及多模态条件合成的扩展规律尚未充分探索。关键瓶颈在于大规模模态对齐运动数据的匮乏，限制了模型对不同控制信号的泛化能力。为此，我们提出大规模高质量数据集OmniHuMo，包含超过5000小时运动和320万条序列，并配有精确对齐的多模态标注（如文本、语音、音乐和轨迹）。基于OmniHuMo，我们提出统一多模态框架AnyMo，该框架结合基于残差FSQ的运动分词器与可扩展的掩码建模变换器，能够在任意模态组合下实现高质量运动合成。大量实验表明，AnyMo在实现高保真合成的同时，可对空间和风格属性进行灵活控制。

The Compressive Knowledge Graph Hypothesis: Which Graph Facts Matter for Scientific Hypothesis Generation?（压缩知识图谱假设：哪些图谱事实对科学假设生成至关重要？）

知识图谱（KG）能为语言模型提供结构化的科学上下文，但尚不清楚哪些图谱事实真正影响生成的假设。本研究针对电池材料领域，在Mistral-7B、Llama-3.1-70B和Gemini 2.5 Flash模型上，通过改变密度、本体丰富度、拓扑和控制结构对局部KG进行扰动，并采用提供的图谱与固定参考指标评估输出。跨模型来看，KG的效用具有选择性和模型依赖性：图谱上下文会改变输出，但无KG输出的模型先验也能恢复大量图谱内容。紧凑的top-k子图常能近似完整KG行为，甚至在保留声称的结果三元组时也是如此。同时，压缩并非唯一依赖于某种语义排序规则，随机和基于拓扑的子集也能恢复大量信号。这些结果支持冗余感知的压缩KG假设：有用的KG信号往往能从紧凑、科学结构化的子图中获取，而非需要完整的局部图谱。

BenHalluEval: A Multi-Task Hallucination Evaluation Framework for Large Language Models on Bengali（BenHalluEval：面向孟加拉语大语言模型的多任务幻觉评估框架）

尽管孟加拉语是全球第六大语言，但此前尚无系统性评估大语言模型在孟加拉语中幻觉现象的研究。本文提出BenHalluEval，一个针对孟加拉语的细粒度幻觉评估框架，涵盖四项任务：生成式问答（GQA）、孟加拉语-英语混合问答、摘要生成和推理。我们利用GPT-5.4，基于三个现有孟加拉语数据集，针对十二种特定任务的幻觉类型构建了12,000个幻觉候选样本。在双轨协议下，对七种大语言模型（包括推理导向、多语言和孟加拉语中心三类模型）进行评估，该协议独立测量真实实例上的假阳性率（轨迹A）与幻觉候选样本上的幻觉检测率（轨迹B）。为联合惩罚两种失效模式并防止因均匀响应偏差导致的分数虚高，我们提出BenHalluScore双轨校准指标。该指标在模型与任务间的得分范围为7.72%至55.42%，揭示了幻觉校准的显著差异。作为缓解策略应用的思维链提示，虽然改变了响应分布，但并未一致提升幻觉判别能力。BenHalluEval建立了首个孟加拉语专用幻觉基准，并凸显了双轨及仅依赖提示的评估方法在低资源语言场景中的不足。数据集与代码见：https://anonymous.4open.science/r/BanglaHalluEval-EB77
