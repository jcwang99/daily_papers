Geometry-Aware Offline-to-Online Learning in Linear Contextual Bandits（几何感知的线性上下文摇臂离线到在线学习）

本研究聚焦线性上下文摇臂（Linear Contextual Bandits）中的离线到在线学习问题，考虑了有偏的离线回归数据：离线参数无需与在线参数匹配，因此历史数据不应仅被当作单一的热启动。通过引入偏移证书（shift certificate）$(M_{\mathrm{shift}},ρ)$ 和离线岭估计（ridge estimation），模型建立了一个几何感知的置信区域（geometry-aware confidence region），而非各向同性的半径。提出了Ellipsoidal-MINUCB算法，该算法将标准在线分支与离线信息驱动的池化分支相结合，仅在离线信息能够缩小不确定性时加以利用。高概率下，遗憾界（regret bound）取标准SupLinUCB风格退路项与池化项（将统计宽度与证书加权偏移惩罚分离）两者的最小值。在简单的对齐条件下，池化项进一步简化为基于离线几何诱导的有效维度（effective dimension）的速率。研究表明，纯欧几里得（标量）偏移界限本身无法确定哪些特征方向是可迁移的。在固定证书的基础上，还展示了如何在有限次刷新时刻从数据中学习数据驱动的证书，并建立了具有逐轮（epoch-wise）学习证书的Ellipsoidal-MINUCB的高概率遗憾界。实验验证了主要预测：当离线覆盖与可迁移性对齐时，收益在中长期阶段最为显著，否则该方法将退回到安全的在线基线。

Poster: ClawdGo: Endogenous Security Awareness Training for Autonomous AI Agents（ClawdGo：面向自主AI代理的内生安全认知训练）

在OpenClaw等平台上部署的自主AI代理面临提示注入、内存投毒、供应链攻击和社会工程学威胁，但现有防御仅关注平台边界，未能训练代理自身的威胁判断能力。本文提出ClawdGo框架，实现内生安全认知训练：在推理阶段教会代理从内部识别和推理威胁，无需修改模型。主要贡献包括：TLDT（三层领域分类法）在自我保护、所有者保护和企业安全三个层面组织12个可训练维度；ASAT（自主安全认知训练）是一种自对弈循环，代理在“最弱优先”课程调度下交替扮演攻击者、防御者和评估者角色；CSMA（跨会话记忆积累）通过四层持久记忆架构和公理结晶促进机制增强技能积累；SACP（安全认知校准问题）形式化了内生训练引入的精确率-召回率权衡。实验表明：最弱优先ASAT在16个会话中将TLDT平均得分从80.9提升至96.9，比均匀随机调度高6.5分，覆盖12个维度中的11个。CSMA能保持跨会话的全部增益；冷启动消融仅恢复2.4分，留下13.6分差距。E模式生成覆盖全部12个维度的32个TLDT兼容场景。SACP表现为经过重度训练的代理将合法能力评估误判为提示注入（30/160）。

DepthKV: Layer-Dependent KV Cache Pruning for Long-Context LLM Inference（面向长上下文大语言模型推理的层级依赖型KV缓存剪枝）

长上下文推理是大语言模型（LLMs）的关键能力，但自回归推理依赖的键值（KV）缓存随序列长度线性增长，造成内存瓶颈。现有KV缓存剪枝方法通常对所有层采用统一的剪枝比例，隐含假设各层对模型性能贡献相同。本研究证明该假设是次优的，因为不同层对剪枝的敏感度存在显著差异。我们提出DepthKV，一种基于层级依赖的剪枝框架，根据各层对剪枝的敏感度分配固定的全局KV预算，而非采用统一分配。在多个模型和任务中，DepthKV在相同全局剪枝比例下始终优于统一剪枝方法，通过层级依赖的分配方式实现了KV缓存预算的更有效利用。

Diffusion-Guided Feature Selection via Nishimori Temperature: Noise-Based Spectral Embedding（基于西森温度的扩散引导特征选择：噪声谱嵌入）

本文提出了噪声谱嵌入（NBSE），一种无需贪婪搜索即可从高维数据中选择信息特征的物理启发框架。NBSE在样本上构建稀疏相似图，并识别西森温度β_N（贝蒂海森矩阵变得奇异的临界逆温度）。相应的最小特征向量捕获了本质上具有度数校正的扩散过程的主导模式，自然地重新加权节点以防止枢纽支配。通过转置数据矩阵并在特征空间中应用NBSE，获得一维谱嵌入，揭示冗余或语义相关维度组；然后通过平衡分箱从每组中选择一个代表性特征。本文证明，有色高斯扰动最多使β_N移动O(σ̄²)，确保了对测量噪声的鲁棒性。在MobileNetV2和EfficientNet-B4的ImageNet嵌入实验表明，即使在高压缩下NBSE也能保持分类精度：仅保留30%特征时，EfficientNet-B4的精度下降低于1%，比ANOVA F检验和随机选择高出6.8%。

本文提出了一种基于视觉的术中医疗图像导航系统，通过单RGB摄像头捕捉手势实现非接触式交互。该系统无需额外硬件或用户特定训练，利用MediaPipe Hands进行实时2.5D手部地标跟踪。简单直观的手势被映射为平移、旋转和缩放命令，实现与图像查看器的连续自然交互。系统架构独立于可视化软件，本研究将其与PyVista集成。性能评估通过帧级日志及延迟、稳定性和交互鲁棒性量化分析完成。实验结果表明，系统具备低延迟、稳定控制的实时行为，满足流畅交互需求。该低成本非接触式方案验证了术中医疗图像访问的可行性，为未来临床评估奠定了基础。

All That Glitters Is Not Audio: Rethinking Text Priors and Audio Reliance in Audio-Language Evaluation（闪光的未必是音频：重新思考音频-语言评估中的文本先验与音频依赖）

大型音频-语言模型在语音和音频基准测试中表现持续提升，但高分并不代表真实的听觉感知能力。如果模型能在不处理音频信号的情况下回答问题，则基准测试无法衡量听觉理解能力。我们提出一个诊断框架，包含两个维度：文本先验（仅凭文本和常识即可回答问题的程度）和音频依赖（对音频信号的实际依赖程度）。在三个基准测试中对八个LALM进行评估后发现，即使没有音频输入，模型仍能保留60-72%的全音频分数。此外，在需要音频回答的项目中，仅3.0-4.2%需要完整音频片段，其余大部分可通过局部片段解决。这些发现挑战了“基准测试成绩等同于稳健音频理解”的假设，最后我们提出了提高评估可靠性和基准设计质量的实用指南。

MEMCoder: Multi-dimensional Evolving Memory for Private-Library-Oriented Code Generation（MEMCoder：面向私有库的多维演化记忆代码生成）

大型语言模型（LLM）在通用代码生成上表现优异，但在依赖企业私有库（未出现在公开训练语料中）时性能急剧下降。尽管检索增强生成（RAG）通过提供静态API文档实现了无需微调的替代方案，但该类文档通常仅提供孤立定义，存在根本性的知识缺口：LLM在任务层面缺乏API之间的协调模式，并在API层面误解参数约束与边界条件。为此，我们提出MEMCoder框架，使LLM能够自主积累并演化两个维度的“使用指南”。该框架引入一种多维演化记忆，从模型自身的解题轨迹中提取精炼经验。在推理阶段，MEMCoder采用双源检索机制，将静态文档和相关的历史指南同时注入上下文。它通过自动闭环运作，利用客观执行反馈反思成功与失败，解决知识冲突并动态更新记忆。在NdonnxEval和NumbaEval基准上的广泛评估表明，MEMCoder显著增强了现有RAG系统，平均绝对pass@1提升达16.31%。此外，与现有基于记忆的持续学习方法相比，MEMCoder展现出极强的领域特定适应能力。

Tuna-2: Pixel Embeddings Beat Vision Encoders for Multimodal Understanding and Generation（Tuna-2：像素嵌入在多模态理解与生成中超越视觉编码器）

Benchmarking Source-Sensitive Reasoning in Turkish: Humans and LLMs under Evidential Trust Manipulation（土耳其语中源敏感性推理的基准测试：证据信任操纵下的人类与大语言模型）

本文提出FDIM，一种基于特征距离的通用视频质量评价指标，适用于传统及神经视频编解码器（NVC），覆盖标准动态范围（SDR）和高动态范围（HDR）格式。FDIM采用混合架构，融合深度特征与手工特征：深度特征部分学习多尺度表征，以捕捉从结构、纹理保真度退化到高级语义偏差的失真；手工特征部分提供稳定的补充线索，提升整体泛化能力。该模型在包含超过1.6万视频序列的大规模主观质量数据集（DCVQA）上训练，数据集由传统混合视频编解码器与端到端感知优化的神经编解码器编码而成。在十个涵盖多种未见编解码器的SDR/HDR视频质量评估（VQA）数据集上的实验表明，FDIM具有强泛化能力，且与主观评估高度相关。

Credal Concept Bottleneck Models for Epistemic-Aleatoric Uncertainty Decomposition（面向认知-偶然不确定性分解的可信概念瓶颈模型）

概念瓶颈模型（CBMs）通过人类可解释的概念进行预测，但其输出的点概率通常混淆了认知不确定性（可减少的模型欠规格）与偶然不确定性（不可减少的输入模糊性），导致概念层面的不确定性难以解释，更难以据此采取行动。我们提出CREDENCE（可信集成概念估计），一种通过构造分解概念不确定性的CBM框架。CREDENCE将每个概念表示为可信预测（概率区间），从多个不同概念头的分歧中推导认知不确定性，并通过专门训练的模糊性输出（匹配标注者分歧）估计偶然不确定性。所得信号支持规范性决策：对低不确定性情形自动化，优先收集高认知不确定性情形的数据，将高偶然不确定性情形交由人工审查，当两者均高时主动弃权。在多个任务中，我们证明认知不确定性与预测误差呈正相关，而偶然不确定性紧密追踪标注者分歧，提供了超越误差相关性的指导。代码实现见：https://github.com/Tankiit/Credal_Sets/tree/ensemble-credal-cbm
