Barnamala: Parameter-Efficient Handwritten Devanagari Recognition at Benchmark Saturation（高效参数的手写梵文识别在基准饱和下的表现）

我们构建了一个紧凑的卷积网络（仅 111 万参数）用于 46 类 DHCD 手写梵文识别，达到了 99.73% 的准确率，是此前最优模型参数量的 1/15.6。所有测试模型（包括大型教师集成）均触及相同的 11 错误固有下限，在精确 McNemar 检验和 Wilson 置信区间下，没有任何配置能获得统计上的显著优势。即使不使用知识蒸馏，我们的学生模型也匹配了最大的基线模型（1732 万参数；McNemar p=0.345）。在 DHCD 之外，对 CMATERdb 数字的零样本识别率为 76.6%，微调后达到 97.8%；在鲁棒性方面，我们的模型远优于大型基线（平均噪声准确率 75.7% 对比 38.7%）。所有代码已在 GitHub 开源。

TCAM-Diff: Triplane-Aware Cross-Attention Medical Diffusion Model（三平面感知交叉注意力的医学扩散模型）

我们提出TCAM-Diff，一种新颖的3D医学图像生成模型，能大幅降低编码和生成高分辨率3D数据所需的内存。该模型使用仅解码器的自编码器方法，从密集体数据中学习三平面表示，并利用泛化操作防止过拟合。随后，采用三平面感知的交叉注意力扩散模型，有效学习并整合这些特征。扩散模型生成的特征可通过预训练解码器模块快速转换为3D体数据。我们在三种不同规模的医学数据集（BrainTumour 128x128x128、Pancreas 256x256x256和Colon 512x512x512）上进行了实验，结果出色。我们使用MSE和SSIM评估重建质量，并利用W-GAN判别器评估生成质量。与现有方法对比表明，在具有相似大小潜空间的编码器-解码器方法中，我们的方法在重建和生成效果上均更优。

When Bots Join the Team: Bot Adoption and the Institutional Fabric of Open-Source Software Projects（当机器人加入团队：机器人采用与开源软件项目的制度结构）

AI代理加入人类团队引发了一个基本问题：当自动化代理成为常规参与者时，群体组织会增强还是削弱？本研究在开源软件中考察这一问题，将机器人视为参与者而非工具，分析了2,991个GitHub项目在首次采用机器人前后各两年的数据。研究测量了制度理论中与持久协调相关的三种能力——重复参与、社会记忆和角色分化，以及两种结果：冲突级联和输出独特性。采用机器人后，出现了更多重复协作、讨论中对特定机器人的更高认可、更少的冲突级联以及更独特的输出。这些变化集中在采用时间点附近，而非逐渐累积。由于缺乏未处理对照组，结果被解释为精确时间点的关联，而非因果效应。两种模式难以用其他解释来说明：能力根据其功能（协调与分化）而非提供者的身份（人类或机器人）来预测结果；人类侧能力解释了机器人-冲突关联，但未解释机器人-独特性关联。研究结果与一种特定解释一致：可预测、基于规则的代理可以成为社区社会基础设施的一部分。机器人是契机；社会组织是机制。

LAPO: Leave-One-Turn Attribution for Self-Generated Process Rewards in Multi-Turn Search Reasoning（LAPO：面向多轮搜索推理中自生成过程奖励的留一轮归因法）

针对多轮搜索推理中强化学习依赖终端结果奖励、无法区分有用、冗余与有害中间交互的问题，本文提出LAPO——一种基于逆向留一轮归因的自生成过程监督方法。该方法对每个搜索轮次，将其与检索观察替换为固定[DELETE]占位符，并测量当前策略在标准答案上平均对数似然的变化。该“答案-似然增益”在保留所有下游交互的同时评估该轮贡献，使早期证据能在完整推理语境中被评价。LAPO还应用符号一致性门控，仅保留方向与原始归因分数一致的归一化过程优势值。该方法无需额外奖励模型、教师、验证器或LLM评判。在七个需本地检索的知识密集型问答数据集上，LAPO平均精确匹配得分为0.326，比最强步骤奖励基线IGPO高出0.053。消融实验表明，逆向归因与符号一致性门控具有互补优势，证明策略导出的回溯归因能为多轮搜索智能体提供有效的过程监督。

ThinkBLOX: 3D Indoor Scene Generation with Progressive Reasoning（ThinkBLOX：基于渐进推理的三维室内场景生成）

针对传统方法多用一次性范式导致编辑需全局重构、布局混乱的问题，本文提出ThinkBLOX。它是一种基于视觉语言模型（VLM）的渐进推理框架，将布局生成视为状态条件驱动的逐步推理与行动过程。为此，作者构建了包含22.5万个程序化放置对、附有多视图场景上下文、显式思维链（CoT）推理及结构化JSON布局数据的ThinkBLOX-Data-200K数据集。通过在此数据集上进行监督微调（SFT），VLM学会了在增量更新下弥合推理与行动之间的差距。此外，针对场景合成本质上的多解任务特性及SFT存在的奖励冲突，作者提出了分层解耦GDPO，这是一种将异构奖励组织成分层、稳定物理有效性、语义合理性及推理行动一致性策略优化的强化学习方案。大量实验表明，ThinkBLOX在物理合理性、语义对齐及交互可编辑性方面显著优于近期的一次性与迭代式基线。同时，它支持全局与局部生成、三维场景重新排列等多种应用。

STOCKTAKE: Measuring the Gap Between Perception and Action in LLM Agents with a Fair Oracle（盘点：利用公平神谕衡量LLM智能体感知与行动之间的差距）

本文提出STOCKTAKE基准，用于区分LLM智能体在长时间跨度决策任务中因“感知错误”还是“知而未行（知道-行动差距）”导致的失败。现有评估无法分离这两种失败，因为它们要么依赖智能体无法获取的特权信息，要么缺乏参考策略。STOCKTAKE被构建为一个具有六种隐藏因子过程的26周供应链补货基准，可计算公平参考策略（基于精确贝叶斯滤波的滚动策略），并通过技能得分（0-1）和逐周推理评分分别衡量状态估计与控制能力。实验表明，Claude Sonnet 5、GPT-5.4、DeepSeek-V4-Pro和Grok 4.5能检测84-88%的隐藏故障（通常在出现后一周内），但技能得分跨度从0.62到-0.23：其中两个模型低于基准水平，且其诊断速度反而略快于表现更好的模型。失败呈现两面性：在持续压力下，每个模型有34-43%的正确诊断周仍导致缺货；技能得分较低的模型在诊断周缺货率最低（反应不足仅为一面），而其轨迹指向另一面——成本超出所保护价值的过度反应。STOCKTAKE可衡量这两类失败方向。

GeoAnchor: Collaborative Reasoning via Latent Decomposition for 3D Spatial Understanding（GeoAnchor：通过潜在分解实现协作推理的3D空间理解）

尽管多模态大语言模型（MLLMs）取得了显著进展，但从2D图像理解3D空间关系仍是关键挑战。现有方法主要依赖符号文本标记，缺乏表示连续几何信息的保真度；而近期使用潜在表示的方法因单一潜在类型无法适应空间任务的多样性，导致复杂几何场景中的错位。为解决这些问题，本文提出GeoAnchor——一个交织文本与潜在表示的推理框架。它将3D空间信息分解为三个互补组件：用于物体定位的位置潜在、用于关系方向的方位潜在以及用于场景结构的几何潜在。这些组件在结构化空间中重组，构建局部证据并捕获全局上下文，实现动态且可解释的推理。此外，本文引入一种协作训练策略，引导模型从局部空间感知走向全面3D理解。在多样且复杂的3D推理任务上的大量实验表明，GeoAnchor超越了现有最先进水平，验证了其有效性和泛化能力。

Is the Statistical Advantage Worth the Cost? An Empirical Comparison of KANs and MLPs for Structured Data Classification（统计优势是否值得付出代价？KANs与MLPs在结构化数据分类中的实证比较）

本研究对Kolmogorov-Arnold网络（KANs）与多层感知机（MLPs）在结构化表格分类任务上进行了实证基准比较。在12个公开数据集（涵盖二分类、多分类、多标签和序数问题）上，两者均采用标准化预处理、固定架构及超参数设置，以测试准确率、F1分数、配对假设检验和效应量评估性能。结果表明，KANs在二分类和多分类领域统计上优于MLPs，且在所有数据集上具有显著的综合优势。然而，中等效应量（d = -0.46）揭示了重要成本效益权衡：KANs虽通过自适应样条映射提供更优泛化能力，但其参数和计算复杂度远高于MLP基线。这表明KANs是高精度应用的首选，而MLPs仍是资源受限环境下的稳健高效选择。未来工作应扩展至其他数据形态以完善架构选择标准。

Nexus: Native Mesh Generation with Diffusion（Nexus：基于扩散的原生网格生成）

本文提出Nexus方法，通过解耦顶点与拓扑生成实现整体网格生成。首先，将网格顶点视为按八叉树组织的稀疏体素，采用扩散模型以从粗到细的方式生成顶点。其次，针对拓扑建模，提出了时空区间（Spacetime Interval）作为时空距离的扩展，将任意边和面拓扑编码为连续的逐顶点嵌入，从而实现全局且高效的复杂拓扑恢复。随后在生成的顶点上使用扩散模型生成连续嵌入。在Objaverse、Toys4K数据集及野外图像上的大量实验表明，该方法优于最先进的自回归和两阶段基线，有效规避了顺序网格建模的内在局限。来自3D从业者的盲测用户研究证实了用户对本方法结果的强烈感知偏好。

From Language to Navigation Goals: A Vision-Language Approach for Semantic Navigation of Mobile Robots Using RGB-D Perception（从语言到导航目标：基于视觉语言与RGB-D感知的移动机器人语义导航方法）

本文提出了一种语言驱动的导航框架，使移动机器人能够理解自然语言指令（如“去邮箱那里”），并自主导航至目标。该框架基于模块化的ROS 2组件构建，通过语义感知识别目标物体、利用RGB-D数据估算其位置，并生成导航目标，最终由Nav2导航栈执行。系统在TurtleBot3 Waffle和Unitree Go2机器人上，经仿真和真实场景验证，能成功解析直接指令与上下文请求，生成有意义的自然语言反馈并精准导航至目标。研究结果表明，融合语义感知与自主导航是实现直观人机交互的可行方案。代码将在论文接收后开源。

Automatic Ordinary Differential Equations Discovery For Biological Systems Using Large Language Model Powered Agentic System（利用大语言模型驱动的智能系统自动发现生物系统的常微分方程）

本文提出了一种名为MEDA的系统，该系统结合大语言模型（LLM）与符号回归（SR），构建智能代理框架，用于自动发现生物及仿生动力系统的常微分方程（ODE）模型。MEDA能够自主检索背景知识、定义变量、生成机制约束、提出候选方程并进行拟合评估。研究在经典模型检索、基于推理的外推以及无/有实验数据的开放式发现等任务中进行了评估。结果表明，MEDA能够正确恢复状态变量，在检索和外推任务中实现强结构恢复，并生成具有生物合理性的发现导向模型。消融与鲁棒性分析进一步揭示，知识引导的形式化与机制约束是关键支撑组件，而单纯依赖数值拟合虽能保持轨迹兼容性，却可能产生生物学上不正确的方程。

DP-BOA: Dirichlet-Process Birth-or-Assign for On-the-Fly Category Discovery（狄利克雷过程生或分配：用于在线类别发现）

在线类别发现需要为每个测试样本决定是将其分配到现有类别还是创建新类别。现有方法通常通过基于匹配的启发式规则（如半径或哈希规则）来实现此决策。虽然实践中有效，但这些方法通常将创建新类别隐式地作为当没有现有类别能够匹配时的后备方案，而非由自身统计证据支持的明确选择。为此，我们提出DP-BOA，一种基于在线狄利克雷过程高斯混合模型（使用正态逆威沙特先验）的后验预测决策框架。训练时，我们用标注数据校准类别高斯分布的共享NIW先验，并预热已知类别的后验。测试时，对于每个样本，DP-BOA比较分配到现有类别的后验预测证据与由DP先验诱导的创建新类别的证据，随后在决策后在线更新类别统计信息。该方法能捕捉各向异性的类别几何结构，并随证据积累自然调整决策置信度。在标准OCD基准测试中，DP-BOA持续超越强基线，在保持竞争力已知类别精度的同时，尤其擅长发现新类别。

Fine-grained CLIP fine-tuning with self-annotated region alignment（使用自标注区域对齐的细粒度CLIP微调）

摘要总结：针对CLIP模型因预训练关注全图-文本匹配而导致细粒度特征表示不足的问题，现有微调方案通常需要额外区域标注，限制了语义多样性并牺牲了全局表征能力。本文提出SFF-CLIP（自标注细粒度CLIP微调），仅利用图像-文本对作为输入，通过运行时区域-短语对齐机制（从输入句子提取概念短语，利用文本特定热力图与对应区域特征对齐），在增强CLIP细粒度表示能力的同时保持原有全局视觉语义一致性。大量实验表明，SFF-CLIP显著提升了细粒度密集特征表示性能，且保持了原CLIP在图像级任务上的表现。代码后续将开源。

Demystifying On-Policy Distillation: Roles, Pathologies, and Regulations（揭秘同策略蒸馏：角色、病理与调控）

同策略蒸馏（OPD）已成为大语言模型后训练的关键范式，但其训练动态尚不清晰。我们系统研究了OPD的角色、病理与调控。首先阐明OPD作为探索催化剂的作用：通过密集的令牌级引导，将学生模型导向正确的推理路径，但不提升能力上限。我们通过验证提示多样性比单问题采样数量更重要，并指出OPD的有效性完全取决于引导信号的质量，确认了该作用。这种依赖性暴露了两个破坏探索的病理：当师生分布差距过大时，引导信号与任务正确性错位，导致探索方向适得其反，即“师生不匹配”；当聚合的令牌级目标产生长度依赖的捷径，使学生通过响应截断或冗余填充操控奖励，探索退化长度模式而非推理策略时，即“长度利用”。为抑制这些病理，我们研究了轻量级信号调控：优势裁剪和对数尺度压缩，确保探索由可靠信号引导。七个基准实验表明，这些调控能缓解长度利用，实现有效蒸馏，稳定超越OPD变体和RLVR基线，从而证实：调控良好的信号质量，而非单纯的教师规模，决定了OPD中成功探索的关键。

Approximation of solutions of parameter-dependent problems by residual neural networks（通过残差神经网络近似参数依赖问题的解）

本文开发了一种基于梯度流的收敛方案，用于训练包含解析激活函数的神经网络。该方案的收敛性由Lojasiewicz理论保证，其主要优点是实现简单。网络的系数通过求解常微分方程组来逼近。作者通过构建参数化问题的残差神经网络近似来测试该方法。结果准确再现了简单常微分方程解对少数参数的依赖关系。即使在对问题严重不适定的区域，也能合理逼近涉及少数参数的波动约束反问题的解。

Early Adoption of Agentic Coding Tools by GitHub Projects（GitHub项目对智能编码工具的早期采用）

本文分析了来自2,361个热门GitHub仓库的25,264个智能体拉取请求（PR），针对智能编码工具在项目层面的采用与管理进行了研究。核心发现如下：1）采用强度集中，中位数项目在三个月内仅产生1-2个智能体PR，广泛采用仅集中于少数项目；2）小型项目（1-5名贡献者）的智能体PR参与率和平均活动水平高于中大型项目；3）智能体PR生产力存在显著差异，少数项目超过行业报告的每参与者36个PR的阈值，但多数项目低于此水平；4）人机协作主要采用单人监督模型（即一名开发者审查/修改智能体贡献），多人协作模式仍不常见。这些早期实证证据表明，开源项目围绕智能编码工具组织人工监督，而智能体贡献的成功整合不仅取决于智能体能力进步，更依赖于治理其使用的人类与组织流程。
