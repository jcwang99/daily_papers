Calibrated Surprise: An Information-Theoretic Account of Creative Quality（校准的惊喜：从信息论视角解析创意质量）

优秀创意写作的本质是“校准的惊喜”：当所有相关维度的约束共同作用时，可行解空间被压缩至狭窄区域，而从无约束视角看，最终留存的选择最不可预测。“校准”具有精确含义——作者意图、读者合理预期以及现实逻辑三者必须一致。当这三个独立判断在每个维度上都达成一致时，可接受的写作选择集就被迫局限在极小的区域内。其数学推论是：全维度准确性与平庸性互斥——它们是同一约束结构的两面，而非两个独立目标。

本文以香农互信息公式 I(X;Y)=H(X)-H(X|Y) 作为分析工具：“校准”对应条件熵趋近于零，“惊喜”对应熵值升高，而互信息正是对两者联合量的精确度量。论证基于两大支柱：静态层面，当人格、叙事、措辞与思想四类约束同时施加时，可行集急剧缩小，其余解在无约束视角下呈现为低概率选择；动态层面，链式法则表明每个写作选择既受先前选择约束，又制约后续选择，宏观决策自然贡献更大比例的信息量，无需人工调参加权。

通过案例研究与轻量级大语言模型对数概率计算，本文证明了该框架兼具分析有效性与可操作性，为“创意质量对齐”（CQA）及专业评估基准奠定了理论基础。

组合泛化是现代神经网络的根本弱点，限制了其在需要分布外推理领域的鲁棒性和应用。神经符号AI中一个核心但未经检验的假设是，组合推理将作为成功符号具身化的副产品自然涌现。本文通过解构具身化与推理的贡献，首次系统性地实证挑战了这一假设。为此，我们引入了迭代逻辑张量网络（$i$LTN）——一种专为多步推理设计的完全可微架构。利用形式化的泛化分类（探测新实体、未见关系及复杂规则组合），我们证明：仅基于具身化目标训练的模型无法泛化；而我们的完整$i$LTN模型在感知具身化和多步推理的联合训练下，在所有任务上实现了高零样本准确率。研究结论提供了确凿证据：符号具身化虽必要，但不足以实现泛化，推理并非涌现属性，而是一种需要明确学习目标的独立能力。

Culturally Aware GenAI Risks for Youth: Perspectives from Youth, Parents, and Teachers in a Non-Western Context（面向青少年的文化敏感型生成式人工智能风险：来自非西方背景下青少年、家长和教师的视角）

本研究聚焦沙特阿拉伯等非西方背景，采用混合方法（分析736条Reddit帖子、1262条X推文，并访谈31名参与者），探讨7-17岁青少年、家长及教师与生成式AI工具的互动及风险认知。主要发现包括：由社区结构和规范形成的隐私与安全具有情境依赖性；青少年向GenAI透露个人及家庭信息的行为，与当地文化中对谦逊、隐私和荣誉的期望相冲突；社会经济因素（如共用ChatGPT账户）进一步加剧风险。研究最终提出了符合文化规范的、情境敏感的家长控制设计启示。

QYOLO: Lightweight Object Detection via Quantum Inspired Shared Channel Mixing（QYOLO：通过量子启发的共享通道混合实现轻量级目标检测）

针对单阶段目标检测器中深层骨干网络计算开销大的问题，本文提出QYOLO框架。其核心是用紧凑的QMixBlock替换骨干网络中最深的两个C2f瓶颈模块（P4/16的512通道和P5/32的1024通道）。QMixBlock通过正弦混合机制实现全局通道重校准，并在两个骨干阶段共享可学习参数，从而强制保持通道重要性一致性，避免为每个阶段设置独立参数集，实现了真正的架构压缩；颈部网络和检测头保持经典结构不变。在VisDrone2019数据集上的评估显示：QYOLOv8n参数减少20.2%（从3.01M降至2.40M），GFLOPs降低12.3%，mAP@50仅下降0.4个百分点；QYOLOv8s参数减少21.8%，精度仅下降0.1个百分点。结合知识蒸馏后，可在不增加压缩代价的情况下恢复全部精度。扩展骨干加颈部的变体虽实现38-41%的参数减少，但精度下降更大，因此最终设计仅选用骨干网络修改方案。

Rule-based High-Level Coaching for Goal-Conditioned Reinforcement Learning in Search-and-Rescue UAV Missions Under Limited-Simulation Training（基于规则的高层指导用于有限仿真训练下搜救无人机任务中的目标条件强化学习）

本文针对有限仿真训练下的搜救无人机任务，提出了一种分层决策框架。该框架将基于固定规则的高层顾问与在线学习的目标条件低层强化学习控制器相结合，并考虑了一种严格的“无预训练”部署模式来测试早期适应能力。高层顾问由结构化任务规范离线定义，编译成确定性规则，提供可解释的、兼顾任务与安全的指导（包括推荐动作、禁止动作及模式相关仲裁权重）。低层控制器从任务定义的密集奖励中在线学习，并通过融入规则衍生元数据的模式感知优先回放机制复用经验。在电池感知的多目标递送与障碍物密集环境中的移动目标递送两项任务上评估，该方法主要通过减少碰撞终止次数，提升了早期安全性和样本效率，同时保留了在线适应场景特定动态的能力。

SplitFT: An Adaptive Federated Split Learning System For LLMs Fine-Tuning（SplitFT：面向大语言模型微调的自适应联邦拆分学习系统）

联邦拆分学习被证实为一种高效方法，既能解决经典联邦学习中客户端的计算资源限制，又能保障分布式模型训练的数据隐私。然而，当这种训练策略应用于大语言模型（LLMs）微调时，面临若干关键挑战：如何跨不同客户端自适应设置切割层以应对数据和设备异构性问题（这会显著影响系统性能），以及如何高效降低微调过程中的通信开销。目前尚无工作尝试解决这些问题。

为弥补这一空白，我们提出SplitFT——一个面向LLMs微调的自适应联邦拆分学习系统。SplitFT允许不同客户端根据自身计算资源和训练模型性能设置不同切割层，同时提出在切割层降低LoRA秩以减少通信开销。除在提出的拆分联邦学习系统中模拟真实应用中的异构数据外，我们还提出一种基于长度狄利克雷分布的方法，将训练数据分配给不同客户端。大量实验结果表明，基于多种流行基准，我们提出的方法在微调时间效率和模型性能方面均优于现有最优方法。

FaaSMoE: A Serverless Framework for Multi-Tenant Mixture-of-Experts Serving（FaaSMoE：面向多租户混合专家服务的无服务器框架）

混合专家模型（MoE）通过为每个输入仅激活少量专家子集，实现了高容量与高效推理。然而，部署MoE模型时所有专家必须常驻内存，导致激活专家所用资源与配置资源之间存在缺口，这种资源利用率不足在多租户场景下更为突出。本文提出FaaSMoE，一种基于函数即服务（FaaS）平台的多租户MoE服务架构。FaaSMoE通过将专家部署为无状态FaaS函数，解耦MoE的控制平面与执行平面，实现跨租户的按需及伸缩至零的专家调用。此外，FaaSMoE支持函数内可配置的专家粒度，以牺牲细粒度弹性换取更低的调用开销。我们基于开源边缘FaaS平台实现原型，并使用Qwen1.5-moe-2.7B模型在多租户负载下进行评估。与完整模型基线相比，FaaSMoE仅需不到三分之一的资源，展示了一条在多租户环境中实现可扩展、资源高效的MoE服务的可行路径。

GIFGuard: Proactive Forensics against Deepfakes in Facial GIFs via Spatiotemporal Watermarking（GIFGuard：通过时空水印对面部GIF中的深度伪造进行主动取证）

现有深度伪造取证工作主要针对静态图像，难以应用于循环短视频媒体GIF。为此，本文首次提出面向GIF深度伪造主动取证的时空水印框架GIFGuard。在嵌入阶段，提出时空自适应残差编码器（STARE），利用3D卷积骨干和自适应通道重校准捕获全局连贯的时间依赖，确保对高级语义篡改的鲁棒性。在提取阶段，设计深度完整性恢复解码器（DIRD），通过配备3D注意力的时空沙漏架构恢复潜在特征，即使面对严重面部篡改也能精确提取水印信号。此外，构建了首个大规模GIF主动取证基准数据集GIFfaces。大量实验表明，GIFGuard在面部GIF上实现了高保真视觉质量和显著抗深度伪造鲁棒性。相关代码与数据集将公开。

LATTICE: Evaluating Decision Support Utility of Crypto Agents（LATTICE: 评估加密智能体的决策支持效用）

本文提出了LATTICE基准，用于评估真实用户场景下加密智能体的决策支持效用。与以往主要基于推理或结果评估的基准不同，LATTICE通过以下方式填补了空白：(1) 定义了六个关键决策支持属性的评估维度；(2) 提出了涵盖端到端加密副驾驶工作流的16种任务类型；(3) 利用大语言模型自动评分智能体输出。关键创新在于，其维度和任务设计支持利用大语言模型进行大规模评估，无需依赖专家标注的真实数据或外部数据源，并且评估标准可随着新维度、任务、标准及人类反馈不断审计和更新，从而实现可靠且可扩展的评估。与通常比较通用框架下基础模型的基准不同，LATTICE评估了实际加密副驾驶产品中的生产级智能体，揭示了流程编排和UI/UX设计对智能体质量的重要性。本文在1200个多样化查询上评估了六款真实加密副驾驶，结果显示多数副驾驶总分接近，但在维度级和任务级性能上差异显著，表明决策支持质量存在实质性的权衡：不同优先级的用户可能更适合不同的副驾驶，而非仅依据总分排名。为支持可重复研究，我们开源了所有代码和数据。

FutureWorld: A Live Environment for Training Predictive Agents with Real-World Outcome Rewards（未来世界：利用真实世界结果奖励训练预测代理的实时环境）

实时未来预测是指对尚未发生的真实世界事件进行预测。该任务正越来越多地通过基于大语言模型的代理系统进行研究，对于构建能够持续从真实世界学习的代理至关重要。正如交互环境常推动代理的进步，推进实时未来预测自然会将其视为一种学习环境。先前的工作从不同方面探索了未来预测，但通常未将其构建为统一的学习环境。该任务对学习具有吸引力，因为它能提供大量基于多样化真实世界事件的预测问题，同时防止答案泄露。为利用实时未来预测的优势，我们提出了FutureWorld——一个实时代理强化学习环境，它闭环了预测、结果实现与参数更新之间的训练循环。在该环境中，我们选取三个开源基础模型并对其进行了连续多天的训练，结果表明训练是有效的。此外，我们基于该环境构建了每日基准测试，并评估了多个前沿代理，为当前代理系统建立了性能基线。

CurEvo: Curriculum-Guided Self-Evolution for Video Understanding（CurEvo：课程引导的自我进化用于视频理解）

针对现有视频理解自我进化框架缺乏结构化引导、优化控制弱、难度进度无序的问题，本文提出CurEvo框架，将课程学习引入自我进化过程。CurEvo根据模型能力动态调节任务难度、优化评估标准并平衡数据多样性，形成课程引导的反馈循环。基于该原理，开发了多维度自适应QA框架，在感知、识别和理解三个维度上联合进化问题生成与答案评估，确保课程进度的连贯性与可测量性。该框架将弱控制的自我进化转化为更结构化的学习过程，在七个骨干网络和四个VideoQA基准测试中，持续提升了基准准确率和基于评估者的语义分数，验证了课程引导自我进化的有效性。

Semantic Foam: Unifying Spatial and Semantic Scene Decomposition（语义泡沫：统一空间与语义的场景分解）

现代场景重建方法（如3D高斯溅射）虽能实现照片级实时新视角合成，但在交互式图形应用中受限于与传统3D资产相比的交互困难。现有语义分解工作在分割质量和跨视图一致性上仍有挑战。为此，我们提出语义泡沫，将Radiant Foam表示扩展至语义分解任务。该方法利用辐射泡沫的体积Voronoi网格的空间结构，并在单元层面增强显式语义特征场，从而通过直接空间正则化提升视图一致性，缓解点表示中常见的遮挡与监督不一致伪影。实验表明，该方法在目标级分割性能上优于Gaussian Grouping和SAGA等前沿方法。项目页面：http://semanticfoam.github.io/

ClawGym: A Scalable Framework for Building Effective Claw Agents（ClawGym：构建高效Claw智能体的可扩展框架）

Claw类环境支持对本地文件、工具及持久化工作区状态的多步骤工作流，但其规模化开发因缺乏系统性框架而受限，尤其是缺乏可验证训练数据的合成、智能体训练与诊断评估的集成方案。为此，我们提出ClawGym——一个支持Claw类个人智能体全生命周期开发的可扩展框架。具体而言，我们构建了ClawGym-SynData数据集，包含由个性驱动意图与技能基础操作合成的13.5K个经过过滤的多样化任务，并配备逼真的模拟工作区及混合验证机制。随后，我们通过在黑盒轨迹数据上进行监督微调，训练了一组名为ClawGym-Agents的Claw模型家族，并进一步通过轻量级管道（在每任务的沙箱中并行化轨迹生成）探索强化学习。为支持可靠评估，我们构建了ClawGym-Bench基准，包含200个经自动过滤与人类-大语言模型联合校准的实例。相关资源将发布于https://github.com/ClawGym。

Causal Learning with Neural Assemblies（神经集群的因果学习）

神经集群（同步放电并通过共激活增强的神经元群体）能否学习变量间的因果影响方向？虽然已被证明是分类、解析和规划的计算通用基础，但神经集群尚未被证明能内化因果方向性。我们证明神经集群的固有操作——投影、局部可塑性控制和稀疏胜者选择——足以进行方向性学习。我们引入DIRECT（方向性边缘耦合/训练）机制，该机制通过自适应增益调度共激活源集群和目标集群来内化有向关系。与基于反向传播的方法不同，DIRECT仅依赖局部可塑性，使得因果主张在机制层面可审计。研究通过双读出验证策略证实：（i）突触强度不对称性，衡量前向和反向连接间出现的权重差距；（ii）功能传播重叠度，量化方向性信号流的可靠性。在多个领域中，该框架在有监督的已知结构设置下实现了完美的结构恢复。这些结果确立了神经集群作为生物合理动力学与形式化因果模型之间的可审计桥梁，提供了“可解释设计”框架，其中因果主张可追溯至特定神经胜者和突触不对称性。

ATLAS: An Annotation Tool for Long-horizon Robotic Action Segmentation（ATLAS：面向长周期机器人动作分割的标注工具）

现有标注工具主要针对纯视觉数据设计，缺乏对机器人专有信号（如夹爪状态、力/扭矩）的同步可视化支持，且难以适配不同数据集格式。本文提出ATLAS，一种专为长周期机器人动作分割设计的标注工具，能够同步可视化多模态数据（多视角视频与本体感知信号），并支持动作边界、动作标签及任务结果标注。该工具原生支持ROS包和RLDS等主流数据集格式，直接兼容REASSEMBLE数据集，并可通过模块化数据抽象层轻松扩展至新格式。其键盘中心化设计显著减少标注工作量，提升效率。在接触密集型装配任务实验中，相比ELAN工具，ATLAS将单动作平均标注时间减少至少6%；通过融入时序数据，其时间边界与专家标注的对齐精度提升超2.8%，边界误差较纯视觉工具降低5倍。

Graph Construction and Matching for Imperative Programs using Neural and Structural Methods（使用神经与结构方法进行命令式程序的图构建与匹配）

本文聚焦于图构建，作为重用验证工件的结构性基础步骤。我们提出了一条流水线，可将命令式程序及其注释转换为带类型、带属性的图。实验涵盖了采用ACSL的C语言、采用JML的Java以及面向C#的Dafny等数据集。该流水线将抽象语法树解析与来自SentenceTransformer和CodeBERT等模型的语义嵌入相结合，从而生成能够同时捕获结构关系和语义上下文的图表示。结果表明，跨不同语言和注释风格可以构建出一致的图表示。本研究为后续实现语义丰富化及近似图匹配以完成可扩展的验证工件重用提供了实践基础。
