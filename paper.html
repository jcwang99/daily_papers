精确事件检测（PES）需要区分视觉相似但语义不同的相邻帧，这与图像分类和粗粒度动作识别有本质区别。尽管DINO等自蒸馏方法在图像上展现出强大的表示学习能力，但直接将其应用于PES效果不佳：在无监督引导下，细微但关键的运动线索常被作为噪声抑制，导致表示对精确事件边界不敏感。为此，本文提出时间特征蒸馏——一种半监督目标，对齐含时间信息的主干特征（而非投影头输出），以保留对运动敏感和边界感知的线索用于帧级定位。采用带渐进式调度的监督预热，确保在无标签蒸馏开始前学习到有意义的事件线索，从而稳定训练。我们还提出Transformer门控移位模块，一种多尺度门控移位模块，将运动感知的时间信息注入视觉Transformer。在四个细粒度体育基准上的实验表明，该方法在完全监督和半监督基线上均持续改进。在FSPerf数据集仅使用10%监督的情况下，我们的方法将mAP比最强对比方法提升了4.54个百分点；仅用80%标注数据，即在四个数据集中的两个上达到或超越了完全监督的100%基线。

基于非侵入式脑电图（EEG）的脑机接口（BCI）在解码连续三维（3D）运动想象（MI）时，因信号变异性和残差解码误差而面临挑战。卷积神经网络-长短期记忆（CNN-LSTM）模型虽能捕捉连续运动学解码的时空动态，但预测轨迹中仍存在系统性残差误差。本文提出一种两阶段解码框架，应用强化学习（RL）对CNN-LSTM解码器（CNN-LSTM-RL）的输出进行残差运动学修正。RL智能体在离线状态下训练，无需直接输入脑电图信号，仅基于预测的运动学轨迹优化运动精度。与单独使用CNN-LSTM相比，CNN-LSTM-RL在2D环境下将平均相关系数从0.5076提升至0.7181（p=0.0005），在虚拟现实（VR）环境下从0.6420提升至0.7780（p=0.0059），相对增益分别为41.5%和21.2%；对应均方根误差（RMSE）在2D环境中从0.0890降至0.0532（p<0.0001），在VR环境中从0.0714降至0.0441（p<0.0001），相对降幅分别为40.2%和38.2%。这些发现表明，该可扩展框架通过离线残差强化学习修正运动学误差，无需额外神经数据即可提升3D BCI运动想象解码性能，推动神经康复、假肢控制及虚拟交互技术的发展。

Production and Perception in LLMs: A Token Probability Approach（大语言模型中的生成与感知：一种词元概率方法）

本研究探讨大语言模型（LLMs）中是否存在类似人类语言生成与感知的不对称性。通过直接测量词元概率（而非使用元语言提示），研究人员使用Llama-3.1-8B模型进行实验：在生成提示下创作诗歌，然后用改写后的生成提示和感知导向提示对相同词元重新评分。实验结果表明，生成-感知距离始终且显著大于生成-生成距离，平均比值约为1.8。控制实验中的高相关性证实该效应源于交际框架差异，而非提示表面变化。该效应在五种开源模型（包括基础版和指令调优版）中均得到复现。时间分析显示，感知提示在序列开头影响最强，随着生成上下文积累，差异逐渐衰减，但衰减的具体形态因提示对而异。这些发现表明，即使在仅解码器架构中，仅通过提示框架就能引发LLM概率分布上的生成-感知区分。

Enhancing LLMs through human feedback: a journey towards self-improvement（通过人类反馈增强大语言模型：迈向自我提升的旅程）

本研究提出一种新方法，通过策略性地集成辅助反馈检索增强生成（RAG）系统，利用人类反馈持续改进主RAG系统的性能。核心是采用人在回路机制，持续收集、分类用户反馈并将其整合到推理流程中，使系统能够迭代学习和进化。为验证有效性，研究使用LLM作为裁判的评估策略，在三个不同的通用与自定义领域知识基准数据集上进行了严格测试。该框架突显了反馈驱动增强在RAG系统中的变革潜力，为自适应信息检索技术的未来研究树立了先例，标志着通过用户互动实现自主优化与精进的重要一步。

Latent-Identity Tuning in Text-to-Image Personalization Models（文本到图像个性化模型中的潜在身份调优）

本文提出一种针对文本到图像个性化模型的细粒度身份调优方法。不同于标准图像编辑，该方法通过修改特定身份的潜在表征来生成一致展现同一编辑后身份的多样化图像。为支持细粒度调优，研究探索了预训练冻结编码器的潜在空间，无需额外训练即可利用其现有架构揭示潜在语义方向。该空间由一组对捕捉身份不同方面起独特作用的潜在令牌构成，这些令牌通常对应特定的空间或语义人脸区域。实验证明，在此空间及选定令牌定义的子空间内可识别有意义的编辑方向，从而实现局部化、细粒度且语义一致的编辑。定性与定量实验验证了该方法在保持跨图像身份一致性的同时，实现了多样化的局部面部编辑。

Backpropagation as a Nilpotent Linear System（反向传播作为幂零线性系统）

本文提出一种全局算子理论——F-adjoint框架，将深度前馈网络的逐层后向递归重新表述为单一线性系统。核心观点包括：1) 全局后向算子严格块上三角且指数至多为L的幂零性，保证Neumann级数解在至多L项后精确终止，证明经典反向传播在数学上等价于块回代；2) 形式化定义F-对称性（反向传播完美镜像前向传播的条件），以正交权重矩阵为典型实例；3) 通过数值示例揭示严格前馈网络的单路径坍塌及其在残差架构中的消解；4) 利用这种组合结构严格推导残差网络（梯度高速路）和迁移学习（梯度截断）的机制。该框架将反向传播从算法配方提升为全局幂零算子形式。

CFR-Net: Collaborative Feature Refinement Network for Medical Image Anomaly Detection（协作特征精炼网络用于医学图像异常检测）

医学图像异常检测面临挑战：在自然图像上预训练的网络难以适应医学图像，因为异常表现为细粒度局部偏移、多尺度上下文不匹配及方向敏感的结构偏差。为此，本文提出协作特征精炼网络（CFR-Net），在解码前采用共享教师-学生特征精炼，并在解码后引入跨空间一致性。CFR-Net通过共享参数的多路径特征精炼模块（MPFRM）精炼冻结的教师特征与可训练的学生特征，将通用的多路径精炼规则施加于视觉参考与医学领域适应表示上，从而在建模局部、多尺度及方向敏感特征的同时缓解领域差异。一种方差敏感目标与动态“作业集”重组进一步支持逐层自适应一致性学习。在医学基准上的实验表明，CFR-Net在仅训练正常数据时，实现了具有竞争力的异常分类与强异常定位性能。

RAGU: A Multi-Step GraphRAG Engine with a Compact Domain-Adapted LLM（RAGU：具有紧凑域适应大语言模型的多步图检索增强生成引擎）

核心观点总结：RAGU是一个开源模块化图检索增强生成引擎，通过将知识图谱的提取与合并分离来解决现有系统的噪声实体和脆弱检索问题。
创新点：采用两阶段类型提取、DBSCAN去重、LLM摘要和Leiden社区检测的步骤；核心洞察在于紧凑提取器所使用的语言技能（理解、提取、上下文推理）与模型规模弱相关，因此训练了7B参数的Meno-Lite-0.1模型，其在知识图谱构建上比Qwen2.5-32B高出12.5%的调和均值，且在英文GraphRAG任务中表现相当。
性能表现：在GraphRAG-Bench（医学）上，RAGU在所有事实层级中检索到最完整的上下文（证据召回率最高达0.84 vs. ≤0.76），并在合成任务上超越HippoRAG2；在多跳事实问答中，HippoRAG2的明显优势主要源于答案格式差异。
可用性：可通过pip install graph_ragu安装，单GPU运行，基于MIT协议开源，代码和模型均已公开。

STEP: Career-Path Recommendation via Temporal and Educational Trajectory Modeling（STEP：通过时间与教育轨迹建模进行职业路径推荐）

职业路径蕴含数十年技能获取、角色转换与教育投入，大规模理解职业路径对劳动力规划、市场政策及工作推荐至关重要。简历作为职业路径的丰富信息来源，包含详细工作经验、教育与技能描述，但其非结构化、异质性和多语言特性长期阻碍了系统性分析。大语言模型的出现使从非结构化简历中提取包含时间与教育信号的高质量职业轨迹数据成为可能，从而为职业路径推荐提供了新机遇。基于此，我们提出STEP（序列就业轨迹预测），一种利用时间与教育信号预测职业轨迹中下一岗位的新型推荐系统。STEP集成了时间衰减门控循环单元以建模时间动态、基于教育程度的特征线性调制，以及基于注意力的序列池化以选择相关特征进行下一岗位预测。为改进STEP的职业内部表征，我们引入ROUTE两阶段对比方法：先通过无监督去噪自编码将多语言编码器适配至职业领域，再进行带引导负例选择的监督对比微调。在四个职业轨迹数据集（含改进版公开JobHop数据集）上的评估表明，STEP在下一岗位预测任务中优于现有最优基线。数据集与代码已公开，以支持可复现的职业轨迹研究。

Agent Hacks Agent: Autoresearch for Production-Agent Red-Teaming（代理攻击代理：针对生产级代理的红队自动化研究）

本论文研究针对生产级大语言模型代理（如Claude Code和Codex）的自动化红队测试。现有方法主要优化攻击成功率并保存攻陷成品（如基准、负载或攻击程序），但无法记录不安全行为背后的使能条件。本文提出AHA框架，一个可证伪的发现循环：提出漏洞假设→构造证伪器→实例化有效攻击→在沙盒中执行→反思轨迹→将确认发现提升为漏洞概念图（VCG）。每个概念通过声明、使能条件、证伪器、迁移预测及支持证据，将攻击面与不安全轨迹关联起来。在Claude Code和Codex上针对直接与间接攻击的三个场景中，所发现的概念揭示了跨模型与代理的可复用漏洞核心。冻结的VCG无需进一步搜索，在相同单次协议下比最强冻结发现基线高出14.2个百分点，且能跨场景与攻击通道迁移。生成的VCG为生产安全团队提供了可审计成品，用于检查漏洞、验证补丁并积累可复用安全知识。

与智能机器人的长期物理共存不仅需要强大的机器人策略。一个持久的机器人助手必须支持多样化的用户界面，维护关于人和偏好的长期记忆，协调不同机器人本体，并将人类意图转化为安全的物理执行。我们提出了PHILIA，一个基于机器人网关抽象构建的多机器人代理系统。PHILIA保留了OpenClaw丰富的交互和工具生态系统，同时通过统一的能力接口暴露机器人本地运行时、机载感知、导航、扬声器和机器人策略。这种设计将低频、高语义的代理推理与高频、低层次的机器人执行解耦，实现了用户界面、机器人本体和策略后端的即插即用集成。因此，用户体验变得可组合：用户界面、机器人本体、机器人策略、导航或交互算法的进步都能提升整体体验，而无需重新设计系统。我们在Astribot S1机器人上验证了该架构，同时设计机器人网关契约，通过共享能力接口（用于观察、任务执行、导航、语音播放、状态监控和任务取消）支持未来的异构机器人平台。我们展示了代表性的用例，其中代理记忆和场景理解基于机器人动作实现，涵盖从简单整理到具有挑战性的长期及灵巧服务任务（如打包背包和提起垃圾袋）的交互式家庭场景。我们着重强调了人机交互流程，其中对用户意图和偏好的上下文理解，以及在执行过程中引入人工确认或调整，对于有效协助至关重要。

HCRMap: Pressure-Aware Hot-Expert Residency Mapping for 3.5D MoE Chiplet Inference（HCRMap：面向3.5D MoE芯片推理的压力感知热专家驻留映射）

混合专家（MoE）大语言模型（LLM）在推理时仅激活少量专家，但令牌路由会导致持续性的专家热度不均衡：少数热门专家持续接收大部分令牌，而其余专家负载较轻。在3.5D多芯片系统中，这种不均衡不仅造成计算失衡，还会放大通信、内存带宽、I/O和执行队列上的压力。因此，核心问题并非简单地减少令牌移动，而是要在不同内存层级间动态放置和复用热门专家副本。本文提出HCRMap，一种针对3.5D MoE推理中压力感知的专家副本管理框架。基于专家热度、权重加载成本、迁移开销和运行时资源压力，HCRMap动态决定哪些专家应被提升、保留、降级或驱逐，并将路由的令牌组映射到合适的驻留副本，从而联合缓解通信、内存和队列瓶颈。实验结果显示，与Hydra相比，HCRMap在预填充和解码阶段分别将端到端延迟降低了43.6%和43.0%；与MoEntwine相比分别降低了34.5%和33.1%；与PIMoE相比分别降低了46.7%和46.0%。

本文针对人脸超分辨率这一病态问题，提出了一种基于参考图像的解决方案。核心创新点包括：第一，采用基于空间变换器的对齐模块，其稳定性优于常用的可变形卷积；第二，设计了一种聚合函数，能从参考图像中提取高质量信息，并在信息不可用时抑制该功能。实验表明，该模型参数量较小，但在多个数据集上均达到了最先进的性能。源代码已公开。

MicroCharNet: Less is More for License Plate Character Detection（MicroCharNet：少即是多的车牌字符检测）

本文提出MicroCharNet，一种专为车牌字符检测设计的超轻量模型。该模型采用紧凑的C2f模块为主干网络，集成CoordAtt模块以增强特征提取并保留空间信息；使用轻量C3k2基础颈部网络融合多级特征，并配合单级无锚检测头实现端到端预测。在UFPR-ALPR数据集上的实验表明，MicroCharNet仅需0.08M参数和0.096 GFLOPs即可达到有竞争力的检测精度，优于多个近期基于YOLO的基线模型。硬件级评估进一步验证了其在边缘设备上实时部署的效率。结果表明，精心设计的超轻量架构能有效平衡车牌字符检测中的精度与效率。源代码已开源。

Similarity-Guided Curriculum Fine-Tuning of LLMs for Neural Architecture Synthesis（相似度引导的LLM课程微调用于神经架构合成）

本文提出一个基于MinHash的相似度调度框架，为基于大语言模型的神经架构搜索构建渐进式课程。通过使用128排列MinHash签名处理标准化后的7-gram源码块，将参考池划分为相似度带，并按架构异质性递增的顺序呈现；每个阶段的最佳LoRA适配器被累积合并至骨干模型。在CIFAR-10图像分类的LEMUR基准测试中，基于OlympicCoder-7B模型，经过六个渐进微调步骤，每代生成N=15个候选架构。该课程在最高相似度级别达到60%的峰值成功率，且无需后处理修复。在最具多样性的课程级别进行的2×2消融实验（课程模型与基础模型对比，有无部分接口修复）显示：无修复时，基础模型（47%峰值成功率）显著优于课程模型（7%成功率）；而加入部分接口修复后，两者均达到53%成功率。这一模式与合并层级权重漂移逐渐擦除评估器接口先验的假设一致，表明接口修复与课程调度针对不同的失败模式。进一步在SVHN上的跨数据集迁移观察显示，无课程预热的基础模型直接生成，其峰值成功率为27%，但准确率（60.5%）显著低于CIFAR-10的对应结果，这与unq族锚定架构合成难度的增加一致。

Active Offline-to-Online Reinforcement Learning（主动离线到在线强化学习）

背景：离线强化学习（RL）允许从预先收集的大型数据集中训练有效策略，并随后通过有限的在线交互进行改进。这种“离线到在线”强化学习（O2O-RL）范式在交互成本高昂或存在风险的动态非平稳领域中尤其有前景。标准O2O-RL流程会离线训练多个候选策略，利用离策略或在线评估进行评价，然后部署并微调估计价值最高的策略。然而，与离线预训练一样，微调性能对算法和超参数的选择高度敏感，这使得选择单一策略存在风险。

目标：本研究旨在解决O2O-RL环境下，在有限交互预算内为微调进行主动策略选择的问题。据我们所知，这是首次针对该问题的研究。

方法：我们首先明确了该问题的核心权衡：将在线交互资源分配给策略评估（有助于识别高性能策略）与分配给微调（可提升策略性能）之间的取舍。为此，我们提出了一种方法，基于对未来性能的置信上限来主动选择微调策略，以平衡这一权衡。这些置信上限来自于对在线评估观测数据进行拟合得到的局部线性性能预测。

结果：在多种实验条件下，所提出的方法一致优于现有的O2O-RL基线方法。

结论：相比于投入单一策略或平均分配预算，主动选择并微调策略能更有效地利用有限的在线交互预算。我们的框架也推动了离线RL向在线交互成本高昂或存在风险的实际系统进行实用化部署。

Advancing Optimal Subset Oracle via Learning Relaxation of Neural Set Functions（通过学习神经集合函数的松弛推进最优子集预言机）

学习神经集合函数在人工智能驱动的药物发现和产品推荐等应用中至关重要。近期研究引入了最优子集预言机，在弱监督设置下通过平均场变分推断隐式学习集合函数。然而，这些框架依赖蒙特卡洛采样来估计变分分布更新时的证据下界梯度。反复采样导致巨大计算开销，且随机性会破坏优化稳定性。本文重新将证据下界解释为集合函数的连续松弛，并学习一个替代目标函数，在变分优化中替换基于采样的ELBO梯度估计。该替代函数在整个连续域中提供稳定高效的梯度，从而降低计算开销并加速推断。此外，我们在子模最大化条件下建立了该框架的近似保证，并刻画了其与变分自由能的联系。在多个真实任务上的实验表明，该方法持续优于现有基线。
