UnGAP: Uncertainty-Guided Affine Prompting for Real-Time Crack Segmentation（UnGAP：不确定性引导的仿射提示实现实时裂缝分割）

实时裂缝分割对于结构健康监测至关重要，但常受光照变化、模糊和纹理不确定性等偶然不确定性的影响。现有方法通常将不确定性估计作为被动终点用于事后分析，未能将其反馈用于优化特征表示。本文认为，独立的逐像素异方差建模特别适用于裂缝分割，因为裂缝由细粒度局部梯度定义，而非全局语义一致性。然而，该方法存在结构优化缺陷：高预测方差会削弱损失梯度，导致模型忽略困难样本并欠拟合复杂边界。为此，我们提出 UnGAP 框架，在不确定性估计与特征学习之间建立闭环机制。其核心是不确定性提示特征调制器（UPFM），将偶然不确定性视为主动视觉提示而非仅输出。UPFM 通过逐像素仿射变换动态校准特征分布，从而缓解异方差病理，将原本会抑制梯度的较高方差转化为模糊区域中更强特征校正的建设性信号。此外，引入边界感知检测头进一步约束预测精度。大量实验表明，UnGAP 在保持实时推理速度的同时实现了卓越的分割精度，验证了将不确定性从被动度量转化为主动校正工具的优势。

A Semantic Autonomy Framework for VLM-Integrated Indoor Mobile Robots: Hybrid Deterministic Reasoning and Cross-Robot Adaptive Memory（面向集成VLM的室内移动机器人语义自主框架：混合确定性推理与跨机器人自适应记忆）

Causal Software Engineering: A Vision and Roadmap（因果软件工程：愿景与路线图）

软件工程越来越多地需要在不确定性下做出高风险决策，利用来自代码、现场数据和社会技术过程的信号。尽管近期基于AI的支持（如异常检测、预测分析、AIOps及基于大语言模型的智能体）提升了工程师检测模式、综合内容与建议的能力，但许多关键问题涉及干预或反事实推理，例如：改变负载均衡策略的预期影响是什么？如果在不同发布计划下是否能避免系统宕机？相关性模型能回答“什么往往同时发生”，但难以回答“如果我们行动会发生什么”。我们提出因果软件工程（CSE）作为未来范式，通过因果模型和因果推理系统性地指导软件生命周期中的各项活动，用显式假设、考虑不确定性的效果估计和反事实诊断增强现有实践。本文概述：(i) 涵盖开发与运维的因果优先工作流视图，(ii) 工具与组织采纳的分阶段路线图，以及 (iii) 衡量进展的评估与基准议程。

Federated Reinforcement Learning for Efficient Mobile Crowdsensing under Incomplete Information（不完全信息下用于高效移动群智感知的联邦强化学习）

移动群智感知（MCS）是一种分布式感知架构，利用移动单元（MU）的现有传感器执行感知任务。由于MCS系统具有动态性（任务需求、MU可用性及其资源随时间变化），且最优策略需要现实中无法获得的完美非因果信息，因此主要挑战在于不完全信息下为MU寻找高效的任务参与策略。为此，本文提出了一种全新的完全去中心化联邦深度强化学习算法FDRL-PPO。该算法使每个MU能基于自身经验、可用资源和偏好学习任务参与策略，无需依赖系统的完美非因果信息。MU通过能量收集补充电池，导致其可用能量随时间变化，从而产生碎片化学习体验。为缓解这一挑战，该方法利用联邦学习，使MU在无需共享私有原始数据的情况下协作改进模型。仅通过交换已学习的模型，MU能集体弥补个体局限，找到更具可扩展性、鲁棒性和高效性的任务参与策略。在合成数据集和真实数据集上的综合评估表明，FDRL-PPO在任务完成率、任务完成公平性、能量消耗和冲突提案数量方面始终优于基准算法。

U-Define: Designing User Workflows for Hard and Soft Constraints in LLM-Based Planning（U-Define：为大语言模型规划中的硬约束与软约束设计用户工作流）

大语言模型越来越多地用于最终用户的规划任务，但其黑箱特性限制了用户确保可靠性与控制的能力。尽管现有系统引入了验证技术，用户如何有效应用这些刚性约束来表达意图或适应现实世界的变异性仍不清楚。以往研究发现，仅使用硬约束过于死板，而数字化的灵活权重又会让用户困惑。本研究探讨交互工作流如何更好地支持用户应用约束来引导大语言模型生成的计划，重点考察将严格性抽象为高级类型（即硬约束与软约束）并配以不同验证机制，是否能帮助用户更可靠地表达和调整意图。我们提出了U-Define系统，允许用户用自然语言定义约束，并将其分类为不得违反的硬规则或允许灵活性的软偏好。U-Define通过互补方法验证这些类型：对硬约束采用形式化模型检查，对软约束采用大语言模型作为评委的评估。通过技术评估以及针对普通用户和专家参与者的用户研究，我们发现用户自定义的约束类型在保持可用性的同时，提升了感知有用性、性能和满意度。这些发现为设计灵活且可靠的基于约束的工作流提供了见解。

A decoupled diffusion planner that adapts to changing cost limits by using cost-conditioned generation for safety and reward gradients for performance（一种通过成本条件生成保障安全、利用奖励梯度提升性能以适应动态成本限制的解耦扩散规划器）

离线安全强化学习通常需要策略在部署时适应动态变化的安全预算。现有基于扩散的规划器将奖励提升和约束满足视为竞争梯度目标，导致成本限制下安全性不可靠。本文提出安全解耦引导扩散（SDGD），重新将自适应安全轨迹生成解释为从受约束轨迹分布中采样：预算限制轨迹区域，奖励塑造区域内偏好。SDGD 使用成本限制条件化无分类器引导，使采样偏向满足指定限制的轨迹，同时利用奖励梯度引导精炼轨迹以获得更高回报。为抑制直接奖励引导导致的高成本趋势，引入可行轨迹重新标记（FTR）重塑奖励目标并阻止此类方向。首次提出采样时间分析表明：在前缀恢复对齐条件下，FTR 可抑制奖励引发的成本漂移。在 DSRL 基准上的广泛评估显示，SDGD 在 94.7%（36/38）的任务上满足约束，实现了最强的安全性，同时在 21 个任务上获得了安全方法中最高的奖励。

本文研究了认知规划中的信念修正问题，即当一个智能体获得关于某个状态属性的信念后，多智能体系统中所有智能体的信念将如何变化。基于多智能体克里普克模型对智能体信念的标准表征，我们将经典的AGM信念修正公设推广至多智能体环境，旨在为评估动态认知推理框架提供形式化基础——此类框架负责计算作为行动结果的所有智能体的信念。作为满足所有广义AGM公设的简单算子示例，我们提出了广义全交多智能体信念修正。此外，我们还定义了迭代修正标准公设的推广，提出了一个更复杂的、基于事件模型的修正算子，并讨论了在克里普克模型上定义能同时满足所有迭代多智能体信念修正广义公设的认知算子时可能存在的问题。

The Causal Description Gap: Information-Theoretic Separations Across Pearl's Hierarchy（因果描述差距：Pearl层级中的信息论分隔）

Pearl的因果层级表明，观测、干预和反事实查询在性质上存在差异。本文对此提出了一个量化版本：在已知较低层级答案的情况下，需要额外多少比特来指定较高层级的因果答案？我们通过查询类描述长度（即针对某一类查询，由结构因果模型诱导的答案预言机的柯尔莫哥洛夫复杂度）来形式化这一问题。主要构造给出了一种二元无环结构因果模型，其观测分布具有恒定的描述长度，而单变量干预答案预言机的描述长度则为Θ(n²)。一个度数敏感的上界表明，入度为d的有限门模式结构因果模型的观测-干预差距最多为O(nd log(en/d) + n log n)，这使得二次构造在密集情况下达到阶最优，而有根树构造在有限入度情况下达到阶最优。对于每一个固定的ε < 1/4，在ε-精确的全变差描述下，二次差距依然存在。在更高层级，完整的硬干预预言机仍可留下Θ(n)的反事实描述差距。一个通用的模糊度到比特定理及其香农类比表明，这些差距等于残差的高层级模糊度的对数，直至低阶项。

When Alignment Isn't Enough: Response-Path Attacks on LLM Agents（当对齐不足时：针对LLM智能体的响应路径攻击）

自带密钥的智能体架构允许用户通过第三方中继路由LLM流量，这造成了关键完整性漏洞：恶意中继可在对齐后的LLM响应生成后、但在智能体执行前对其进行篡改。我们形式化了这种后对齐篡改威胁，并证明在没有端到端完整性保障的情况下，中继能够观察、抑制或替换下游消息，使完全对齐的LLM也无法抵抗此类攻击。我们将此威胁实例化为中继篡改攻击（RTA），该攻击实施多轮策略性改写、最小安全关键编辑，并通过将篡改后的输出重新提交给上游LLM来实现隐秘恢复。在AgentDojo和ASB基准测试中，对6种LLM实施的RTA攻击成功率最高达99.1%，且开销适中地优于提示注入基线。对OpenClaw和Claude Code的案例研究验证了现实世界的可行性，对四种防御措施的评估表明，没有一种能完全阻止RTA。最后，我们提出了一种基于时间的检测防御方法，可在保持智能体效用的同时缓解RTA。

SCPRM: A Schema-aware Cumulative Process Reward Model for Knowledge Graph Question Answering（SCPRM：一种面向知识图谱问答的感知模式累积过程奖励模型）

大语言模型擅长复杂推理，但评估其推理中间步骤仍具挑战。现有过程奖励模型虽能提供逐步监督，却存在“风险补偿”效应：错误步骤被后续正确步骤抵消，导致对有缺陷的推理路径给出高分。在知识图谱推理中，因实体间存在多条路径，该问题更为严重。为解决这一问题，我们提出了SCPRM模型。该模型通过结合推理前缀与当前步骤到查询隐含目标的模式距离，提供累积及未来奖励以指导路径探索。进一步地，我们将SCPRM集成到蒙特卡洛树搜索中，构成SCPRM-MCTS，用于知识图谱上的多跳问答推理。在医疗、法律知识图谱问答及CWQ数据集上，SCPRM-MCTS的Hits@k指标平均比强基线模型提升1.18%，展现出更准确且风险敏感的推理评估能力。

Binary Rewards and Reinforcement Learning: Fundamental Challenges（二元奖励与强化学习：基本挑战）

使用可验证奖励的强化学习（RLVR）已成为提升语言模型推理能力的标准方法，但用RLVR训练的模型常出现多样性崩溃：单样本准确率提升，但多样本覆盖度下降，甚至低于基础模型。我们基于二元奖励的特性提供了对这一现象的结构性解释。二元奖励给策略梯度方法带来了根本性的退化：最大化期望奖励的分布数量无限，且无特指元素。KL控制通过极限β→0选择滤波模型p*（即基于有效性的条件化基础模型）来破解这种退化——这是KL散度上最接近基础模型的唯一全有效分布。这种选择通过非平凡的对称性实现：倾斜分布p[β]在β→0时向前向KL收敛于p*，但p*不能作为直接优化目标，因为对于任何全支持策略q，KL(q||p*)是无穷大。我们推导了超参数β与可解释目标有效性比率μ之间的显式公式。在模型设定错误（典型实际场景）下，β减小的压力会驱动优化器集中到少量有效输出上，且随着β减小而非朝向滤波模型，崩溃到更少的输出。我们通过一个自回归玩具实验说明了这一机制，并讨论了直接以p*为目标的替代散度（如Kruszewski等人2026年实证探索的）如何通过奖励覆盖p*的支持范围而非聚焦高有效性输出来避免这种失败模式。

Double Rectified Linear Unit-based Modular Semantics for Quantitative Bipolar Argumentation Framework（基于双修正线性单元的模块化语义用于量化双极论证框架）

定量双极论证框架（QBAFs）为计算双极论证框架（BAFs）中的论证可接受性提供了另一种方法。每个论证被赋予一个初始强度，然后通过考虑其攻击者和支持者的影响更新为最终强度。多年来，已有多种语义被提出用于计算QBAFs中的论证可接受性，然而即使在简单的无环案例中，它们也常常产生分歧或反直觉的结果。我们提出了新颖的QBAFs渐进语义，解决了这些局限性，产生的结果更符合直觉期望，同时满足了文献中既定的理性公设。此外，我们研究了其收敛行为，证明它不仅对无环QBAFs收敛，对更广泛的循环框架类别也收敛。
