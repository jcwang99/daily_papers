Robust and Fast Training via Per-Sample Clipping（基于每个样本裁剪的稳健快速训练）

本文提出一种基于每个样本梯度裁剪的稳健梯度估计器，并从理论和实证两方面分析其性质。研究表明，所提出的方法——每个样本裁剪的SGD（PS-Clip-SGD）——在重尾梯度噪声下的非凸优化问题中，实现了最优的期望收敛速度。此外，本文建立了高概率收敛保证，其收敛速度与期望速度在失败概率的对数多项式因子内相匹配。我们通过多项数值实验补充了理论结果。特别地，在使用CIFAR-100数据集训练AlexNet时，即使考虑了每个样本裁剪带来的额外计算时间，PS-Clip-SGD在性能上仍优于带动量的普通SGD和标准梯度裁剪。我们还通过实验表明，在梯度累积的情况下，在小批量级别应用裁剪可在几乎不增加额外计算成本的情况下提升训练性能。这一发现尤为值得关注，因为它与仅在完成所有累积步骤后才应用裁剪的常见做法相悖。

SpectraDINO: Bridging the Spectral Gap in Vision Foundation Models via Lightweight Adapters（SpectraDINO：通过轻量级适配器弥合视觉基础模型中的光谱差距）

传统自动规划假设规划任务的所有方面（初始状态、目标和可用动作）都需事先完全明确，适用于规则固定和确定性执行的领域。然而，现实规划往往需要灵活性，允许偏离原始任务参数以应对意外情况或改进结果。本文综述了自动规划中反事实推理的现有研究，按照改变的要素、推理触发时机、以及变更的原因与方式进行分类。最后，我们讨论了关键发现并概述了开放性的研究问题，以指导该领域的未来工作。

Rethinking the Need for Source Models: Source-Free Domain Adaptation from Scratch Guided by a Vision-Language Model（重新思考对源模型的需求：视觉语言模型引导下从零开始的无源域适应）

本文提出了一种更严格的无目标域适应设置VODA，完全消除对源域模型的依赖，仅依赖随机初始化的模型、视觉语言模型和无标签的目标域数据。研究观察到，不同源模型在相同目标域上的表现差异极小，表明源模型本身影响有限。为此，作者设计了TS-DRD两阶段框架：第一阶段利用视觉语言模型预热模型，第二阶段寻找视觉语言模型与自适应模型中的共同去噪区域，从而提供更干净的蒸馏监督。在Office-Home、VisDA和DomainNet-126上的实验表明，TS-DRD在VODA设置下达到了与现有使用源模型的方法相当或更优的性能，验证了其有效性和VODA设置的潜力。

STABLEVAL: Disagreement-Aware and Stable Evaluation of AI Systems（STABLEVAL：具有分歧感知能力的AI系统稳定评估）

人工评估是评估现代AI系统的主要标准，但标注者的分歧、偏差和可变性使得在标准多数投票聚合下系统排名不够稳健。多数投票忽略了标注者可靠性和项目层面的模糊性，通常导致跨标注者子集的比较不稳定。我们提出了STABLEVAL，这是一个具有分歧感知能力的评估框架，该框架对潜在的项目正确性和标注者特定的混淆模式进行建模，以生成后验期望的项目学分和校准的智能体级别分数。与Dawid-Skene等标签去噪方法不同，STABLEVAL明确设计用于稳定且具有不确定性感知的系统评估，而非恢复硬标签。我们将排名稳定性形式化为一个首要的评估目标，并分析聚合方法如何保留或扭曲底层的标注者行为。在受控的合成实验和多个真实世界的人工标注基准测试中，多数投票在标注者异质性和对抗性噪声下表现出分数误差增大和排名不稳定性，而STABLEVAL则能产生更稳定、统计基础更扎实的系统排名。这些结果表明，对分歧进行建模是实现稳健且可复现的AI评估的关键。

FitText: Evolving Agent Tool Ecologies via Memetic Retrieval（FitText：通过模因检索进化智能体工具生态）

为了解决用户描述任务与工具文档之间的语义鸿沟，以及静态查询检索无法在智能体执行过程中动态更新工具集的问题，本文提出了 FitText，一种无需训练的框架。FitText 将检索直接嵌入智能体的推理循环，通过生成自然语言伪工具描述作为检索探针，利用检索反馈进行迭代优化，并通过随机生成探索多样化方案。Memetic Retrieval（模因检索）在候选描述上施加进化选择压力，并由工具记忆引导以避免冗余搜索。在 ToolRet（43k 工具，4个领域）上，FitText 将平均检索排名从 8.81 提升至 2.78；在 StableToolBench（16,464 个API）上，平均通过率达到 0.73，相较于静态查询检索提升了 24 个百分点。当基础模型具备语义操作能力时，性能增益可迁移；在较弱的基础模型下，模因的进化搜索反而放大噪声而非提炼信号，这表明模型能力是进化工具探索的前提条件。

T$^2$PO: Uncertainty-Guided Exploration Control for Stable Multi-Turn Agentic Reinforcement Learning（T$^2$PO：不确定性引导的探索控制以实现稳定的多轮智能体强化学习）

近期多轮强化学习（RL）的进展显著提升了推理大语言模型（LLM）在复杂交互任务上的表现。尽管细粒度信用分配和轨迹过滤等稳定技术有所进步，但训练不稳定性依然普遍存在，常导致训练崩溃。本文认为，这种不稳定源于多轮场景中低效的探索策略——模型持续生成低信息量的动作，既未降低不确定性，也未推动任务进展。为解决此问题，我们提出了Token和轮次级别的策略优化（T$^2$PO），一种在细粒度层面显式控制探索的不确定性感知框架。在Token级别，T$^2$PO监控不确定性动态，并在边际不确定性变化低于阈值时触发思考干预；在轮次级别，T$^2$PO识别探索进展微弱的交互，并动态重采样这些轮次以避免无效推演。我们在WebShop、ALFWorld和Search QA等多种环境中评估了T$^2$PO，证明了其在训练稳定性、性能提升及探索效率方面的显著收益。代码已开源：https://github.com/WillDreamer/T2PO。

UnGAP: Uncertainty-Guided Affine Prompting for Real-Time Crack Segmentation（UnGAP：不确定性引导的仿射提示实现实时裂缝分割）

实时裂缝分割对结构健康监测至关重要，但常因光照变化、模糊和纹理不明等随机不确定性而效果不佳。当前方法通常将不确定性估计视为事后分析的被动终点，未能将其反馈以优化特征表示。本文认为，独立像素异方差建模特别适合裂缝分割，因为裂缝由细粒度局部梯度定义，而非通用物体分割所依赖的全局语义一致性。然而，该方法存在结构优化缺陷：高预测方差会衰减损失梯度，导致模型忽略困难样本并欠拟合复杂边界。为此，我们提出UnGAP框架，在不确定性估计与特征学习之间建立闭环机制。核心是不确定性提示特征调制器（UPFM），它将随机不确定性视为主动视觉提示而非简单输出。UPFM通过逐像素仿射变换动态校准特征分布。关键的是，该机制通过将高方差（否则会抑制梯度）转化为模糊区域更强特征修正的建设性信号，从而缓解异方差缺陷。此外，引入边界感知检测头以进一步约束预测精度。大量实验证明，UnGAP在实现卓越分割精度的同时保持实时推理速度，有效验证了将不确定性从被动度量转化为主动校准工具的优势。

A Semantic Autonomy Framework for VLM-Integrated Indoor Mobile Robots: Hybrid Deterministic Reasoning and Cross-Robot Adaptive Memory（面向集成大视觉语言模型的室内移动机器人的语义自主框架：混合确定性推理与跨机器人自适应记忆）

现有框架（如ROS 2 Navigation 2）能让机器人可靠地导航到度量坐标，但无法理解表达意图的自然语言指令。大视觉语言模型提供了所需的语义推理能力，但其推理延迟（消费级硬件上每次决策2-9秒）和会话间遗忘限制了实际部署。本文提出一个六层参考框架“语义自主堆栈”，并在配备现成边缘硬件的实物机器人上验证了一个完整实例，该实例融合了混合确定性-大视觉语言模型推理与跨机器人自适应记忆。一个七步参数解析器可在0.1毫秒内处理88%的指令，无需调用语言模型、摄像头或GPU；只有真正模糊的指令才会升级到大视觉语言模型推理。一个包含明确范围分类（全局环境知识、操作员偏好、机器人能力）的五类语义记忆框架，实现了跨会话学习和跨机器人知识迁移：在一个机器人上通过大视觉语言模型交互学到的偏好被提升为确定性解析，并通过共享编译摘要迁移到第二个机器人，实现了10.3万倍的测量延迟降低。在两台自制差分驱动机器人上进行的实验验证（涵盖82个场景级决策和三个会话）表明，语义迁移准确率为100%（33/33，95%置信区间[0.894, 1.000]），语义解析准确率为100%，且支持并发多机器人操作——所有这一切都在树莓派5平台上完成（无板载GPU），无需任何训练数据。

Causal Software Engineering: A Vision and Roadmap（因果软件工程：愿景与路线图）

软件工程越来越多地在不确定性下做出高风险决策，利用来自代码、现场数据和社会技术过程的信号。近期AI驱动支持（例如异常检测、预测分析、AIOps以及基于LLM的智能体）增强了工程师检测模式和综合内容及建议的能力，但许多关键问题是干预性或反事实性的：改变负载均衡策略的预期影响是什么？采用不同发布计划能否避免故障？相关性模型回答“什么倾向于共同发生”，而难以回答“如果我们行动会发生什么”。我们提出因果软件工程作为未来范式，在此范式下因果模型和因果推理系统性地为软件生命周期各活动提供信息，以显式假设、不确定性感知的效果估计和反事实诊断增强现有实践。我们概述了(i)涵盖开发和运营的因果优先工作流视图，(ii)工具和组织采用的分阶段路线图，以及(iii)衡量进展的评估和基准议程。

Federated Reinforcement Learning for Efficient Mobile Crowdsensing under Incomplete Information（信息不完整情况下用于高效移动群智感知的联邦强化学习）

针对移动群智感知（MCS）系统动态变化及信息不完整的挑战，本文提出了一种全新的完全去中心化联邦深度强化学习算法——FDRL-PPO。该算法使每个移动单元（MU）能基于自身经验、资源和偏好独立学习任务参与策略，无需依赖完美的非因果信息。为解决因能量采集导致的可用能量波动与碎片化学习问题，算法利用联邦学习让多个MU在无需共享原始数据（如个人经验）的前提下协同改进模型，仅通过交换已学习模型来弥补个体局限，从而找到更具扩展性、鲁棒性和高效的任务参与策略。在合成和真实数据集上的综合评估表明，FDRL-PPO在任务完成率、公平性、能耗和冲突方案数量上均持续优于基准算法。

U-Define: Designing User Workflows for Hard and Soft Constraints in LLM-Based Planning（U-Define：在基于大语言模型的规划中为用户设计硬约束与软约束的工作流程）

大语言模型越来越多地用于终端用户的任务规划，但其“黑箱”特性限制了用户确保可靠性和进行控制的能力。尽管现有系统引入了验证技术，但用户如何有效应用这些刚性约束来代表意图或适应现实世界的多变性尚不明确。例如，先前研究发现，仅使用硬约束过于僵化，而数值化的灵活性权重又使困惑用户。我们研究了交互工作流程如何更好地支持用户应用约束来指导大语言模型生成的计划，并考察了将严格性抽象为高级类型（即硬约束和软约束）并配合不同验证机制是否能帮助用户更可靠地表达和调整意图。我们提出了U-Define系统，该系统允许用户用自然语言定义约束，并将其分类为不可违反的硬规则或允许灵活性的软偏好。U-Define通过互补方法验证这些类型：针对硬约束采用形式化模型检查，针对软约束采用大语言模型作为评判者进行评估。通过技术评估以及针对普通用户和专家参与者的用户研究，我们发现用户定义的约束类型在不影响可用性的前提下，提升了感知有用性、性能和满意度。这些发现为设计灵活且可靠的基于约束的工作流程提供了洞见。

A decoupled diffusion planner that adapts to changing cost limits by using cost-conditioned generation for safety and reward gradients for performance（一种解耦扩散规划器：利用成本条件生成保障安全、利用奖励梯度提升性能以适应变化的成本限制）

离线安全强化学习通常要求策略在部署时适应不同片段间或单一片段内变化的安全预算。现有的扩散规划器常将奖励提升与约束满足视为竞争性梯度目标，导致在成本限制下安全性不可靠。本文重新将自适应安全轨迹生成解释为从约束轨迹分布中采样：预算限制轨迹区域，奖励塑造区域内的偏好。基于此提出安全解耦引导扩散（SDGD），通过条件化无分类器引导对成本限制进行偏置采样，确保轨迹满足指定限制；同时使用奖励梯度引导优化轨迹以获取更高回报。为应对直接奖励引导可能提升回报却导向高累计成本轨迹的问题，引入可行轨迹重标记（FTR）重塑奖励目标，抑制此类方向。进一步提供了一阶采样时间分析，证明在前缀恢复对齐条件下，FTR能抑制由奖励引发的成本漂移。在DSRL基准上的广泛评估显示，SDGD在基线方法中实现了最强的安全合规性，在94.7%的任务（36/38）上满足约束，并在21个任务上获得了安全方法中的最高奖励。

本文研究了认知规划中的信念修正问题，即当某个智能体获得关于某个状态属性的信念后，多智能体系统中所有智能体的信念将如何变化。基于通过单一多智能体克里普克模型来表示智能体信念的认知规划标准方法，我们将经典的AGM信念修正公设推广至多智能体环境，旨在为评估动态认知推理框架提供一个形式化框架——此类框架需计算作为行动结果的所有智能体的信念。作为满足所有广义AGM公设的简单算子示例，我们提出了广义全交多智能体信念修正。此外，我们定义了迭代修正标准公设的推广形式，提出了一种更复杂的基于事件模型的修正算子，并讨论了在克里普克模型上定义能够满足所有迭代多智能体信念修正广义公设的认知算子时可能存在的问题。

Foundation Models to Unlock Real-World Evidence from Nationwide Medical Claims（利用基础模型解锁全国医疗索赔中的真实世界证据）

本文提出ReClaim模型，是一个从零开始在超2亿患者、438亿医疗事件（2008-2022年MarketScan数据）上训练而成的生成式Transformer。该模型对诊断、手术、用药及支出的纵向轨迹进行建模，参数规模达1.4亿、7亿和17亿。

The Causal Description Gap: Information-Theoretic Separations Across Pearl's Hierarchy（因果描述差距：Pearl层级中信息论层面的区分）

Pearl的因果层级表明，观察性、介入性和反事实查询在性质上是不同的。本文提出了该问题的量化版本：在已知低层级答案的情况下，需要额外多少比特信息才能确定高层级因果答案？我们通过查询类描述长度（即针对某类查询，由结构因果模型诱导的答案预言机的Kolmogorov复杂度）来形式化这一问题。我们的主要构造给出了二元无环结构因果模型，其观察性分布具有恒定的描述长度，而单变量介入性答案预言机的描述长度为Θ(n²)。一个度数敏感的上界表明，入度为d的有限门模式结构因果模型的观察-介入差距至多为O(nd log(en/d) + n log n)，这使得二次构造在密集情形下达到阶最优，而有根树构造在有界入度情形下达到阶最优。对于每个固定ε<1/4，二次差距在ε-精确全变差描述下仍然存在。在更高层级，完整的硬介入预言机仍可能留下Θ(n)的反事实描述差距。一个通用的“歧义到比特”定理及其香农类比表明，这些差距等于剩余高层级歧义的对数，忽略低阶项。

When Alignment Isn't Enough: Response-Path Attacks on LLM Agents（当对齐还不够：针对LLM智能体的响应路径攻击）

论文揭示了自带密钥（BYOK）智能体架构中的关键安全漏洞：恶意中继可在模型生成响应后、智能体执行前篡改输出，从而破坏对齐LLM的安全性。作者形式化定义了这种“对齐后篡改”威胁，并证明中继能够观察、抑制或替换下游消息，使完全对齐的LLM也难以抵御。该威胁被具体化为中继篡改攻击（RTA），其通过多轮策略性重写、最小化安全关键编辑和隐蔽恢复（将篡改输出重新提交给上游LLM）来实施。在AgentDojo和ASB基准上，针对六种LLM的测试显示，RTA最高达到99.1%的攻击成功率，优于提示注入基线且开销适中。对OpenClaw和Claude Code的案例研究证明了现实可行性，而对四种防御措施的评估表明，没有一种能完全阻止RTA。最后，文中提出了一种基于时间的检测防御，可在保持智能体实用性的同时缓解RTA。

SCPRM: A Schema-aware Cumulative Process Reward Model for Knowledge Graph Question Answering（SCPRM：一种基于模式感知的累积过程奖励模型用于知识图谱问答）

大语言模型擅长复杂推理，但评估其中间步骤仍具挑战性。过程奖励模型虽提供了逐步监督，却常受风险补偿效应影响（错误步骤被后续正确步骤抵消，从而对有缺陷的推理路径给予高分）。在知识图谱推理中，该问题因实体间可能存在多条路径而加剧，风险步骤易导致推理路径出错。这在医疗、法律等高风险敏感任务中尤为棘手。为此，我们提出了一种模式感知累积过程奖励模型（SCPRM），通过在推理前缀条件下评估路径，并结合当前推理步骤与从查询中解析出的隐含目标之间的模式距离，提供累积与未来奖励以引导路径探索。我们进一步将SCPRM集成至蒙特卡洛树搜索（MCTS），形成SCPRM-MCTS，用于知识图谱上的多跳问答任务。在医疗、法律KGQA及CWQ数据集上，SCPRM-MCTS相比强基线平均提升了1.18%的Hits@k性能，展示了更准确且风险敏感的推理评估能力。

Binary Rewards and Reinforcement Learning: Fundamental Challenges（二元奖励与强化学习：基本挑战）

使用可验证奖励的强化学习（RLVR）已成为提升语言模型推理能力的标准方法，但训练后的模型常出现多样性崩塌：单样本准确率提升，而多样本覆盖度下降，甚至低于基础模型。本文基于二元奖励的特性对此现象提供了结构性解释。二元奖励为策略梯度方法带来了根本性退化：最大化期望奖励的分布集是无限的，且无区分要素。KL控制通过极限β→0选择滤波模型p_*（即基于有效性的条件基础模型）来解决退化，该模型是KL散度中最接近基础模型的唯一完全有效分布。这种选择存在非平凡不对称性：倾斜分布p_{[β]}在β→0时以前向KL收敛于p_*，但p_*不能直接作为优化目标，因为对任何全支持策略q，KL(q∥p_*)均为无穷大。我们推导了超参数β与更可解释的目标有效性比率μ之间的显式公式。在模型误设（典型实际场景）下，减小β的压力驱使优化器转向小量有效输出上的高度集中分布，随着β减小不断坍缩至更少输出，而非转向滤波模型。我们通过玩具自回归实验说明这一机制，并讨论直接以p_*为目标的替代散度如何通过奖励覆盖p_*的支持集（而非集中于高有效性输出）来避免此故障模式——正如kruszewski等人（2026）的实证探索所示。

Double Rectified Linear Unit-based Modular Semantics for Quantitative Bipolar Argumentation Framework（基于双整流线性单元的模块化语义用于定量双极论证框架）

定量双极论证框架（QBAFs）为计算双极论证框架中的论证可接受性提供了一种替代方法。每个论证被赋予一个初始强度，然后通过考虑其攻击者和支持者的影响，将其更新为最终强度。多年来，已有多种语义被提出用于计算QBAFs中的论证可接受性，但即便对于简单的非循环案例，它们也常常产生分歧或反直觉的结果。我们为QBAFs引入了新颖的渐进语义，解决了这些局限性，使结果更符合直觉预期，同时满足文献中既定的合理性公设。此外，我们研究了其收敛性，证明该语义不仅适用于非循环QBAFs，也适用于更广泛的循环框架类别。
