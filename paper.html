DiscoTrace: Representing and Comparing Answering Strategies of Humans and LLMs in Information-Seeking Question Answering（DiscoTrace：信息寻求问答中人类与大语言模型回答策略的表征与比较）

本文提出了DiscoTrace方法，用于识别回答者在回应信息寻求类问题时所使用的修辞策略。该方法将答案表征为一系列与问题相关的话语行为序列，并配以对原始问题的解读，这些标注基于修辞结构理论分析。应用该方法分析九个不同人类社区的答案后发现，各社区在构建答案时存在多样化的偏好。相比之下，大语言模型的答案则缺乏修辞多样性，即使被要求模仿特定人类社区的作答指南时亦是如此。此外，大语言模型系统性地倾向于追求广度，会回应人类回答者通常选择忽略的问题解读。本研究结果可为开发更实用的、能考虑问答情境并借鉴多种人类策略的大语言模型回答者提供指导。

GDPR Auto-Formalization with AI Agents and Human Verification（GDPR自动形式化：AI智能体与人工验证）

本研究探讨了在人机协同验证框架下，利用大语言模型自动形式化《通用数据保护条例》条款的整体流程。核心方法是采用角色专精的工作流，而非追求完全自主化：基于LLM的AI组件在多智能体环境中，通过迭代反馈生成法律场景、形式化规则和原子事实。该流程与独立的验证模块相结合，后者包含人工评审员对表征、逻辑及法律正确性的评估。通过此方法，我们构建了一个用于GDPR自动形式化的高质量数据集，并分析了成功与存在问题的案例。结果表明，结构化的验证和针对性的人工监督对于可靠的法律形式化至关重要，尤其是在处理法律细微差别和上下文敏感推理时。

Data Synthesis Improves 3D Myotube Instance Segmentation（数据合成改善3D肌管实例分割）

肌管是多核肌纤维，是研究肌肉生理、疾病机制和药物反应的关键模型系统。相关机制研究和药物筛选依赖于对其直径、长度和分支程度等形态的量化分析，这需要精确的三维实例分割。然而，由于缺乏大规模标注的肌管数据集，现有的预训练生物医学分割模型难以适用于此领域。

本文提出了一种几何驱动的合成流程，通过多项式中心线、局部变化的半径、分支结构以及基于真实显微镜观察得出的椭球端帽来模拟单个肌管。合成的三维数据渲染了真实的噪声、光学伪影，并采用了基于CycleGAN的域自适应技术。一个经过自监督编码器预训练的紧凑3D U-Net，仅使用合成数据训练，在真实数据上取得了0.22的平均IPQ分数，显著优于三种现有的零样本分割模型。这表明，生物物理学驱动的合成方法能够在标注稀缺的生物医学领域实现有效的实例分割。

On the Expressive Power and Limitations of Multi-Layer SSMs（多层状态空间模型的表达能力与局限）

本文研究了多层状态空间模型（SSMs）的表达能力及其固有局限。首先，研究揭示了多层SSM在处理组合性任务时存在根本性限制，这体现了SSM与流式模型之间存在本质差距。其次，论文分析了思维链（CoT）的作用，指出离线CoT无法从根本上提升模型表达能力，而在线CoT则能显著增强其能力，甚至使多层SSM的表达力达到与流式算法等同的水平。最后，研究探讨了模型宽度与计算精度之间的权衡，发现在基础模型中二者不可互换，但在引入在线CoT后，两者可实现清晰的等效关系。总体而言，本文从模型深度、有限精度和思维链三个维度，为理解SSM的能力与局限提供了一个统一的视角。

Rethinking Patient Education as Multi-turn Multi-modal Interaction（将患者教育重新构想为多轮多模态交互）

现有医学多模态基准多聚焦于图像问答、报告生成等静态任务。患者教育要求更高：系统需从图像中识别证据、引导患者关注点、用易懂语言解释发现，并处理患者的困惑或焦虑。然而，相关工作大多仍局限于纯文本。本文提出MedImageEdu基准，用于评估基于证据的多轮放射科患者教育。该基准模拟医生代理与具有隐藏背景（如教育水平、健康素养）的患者代理的交互。当问题需要视觉辅助时，医生代理可调用绘图工具生成基于报告和图像的标注图，最终结合图像与解释形成多模态回应。基准包含150个病例，从咨询过程、安全与范围、语言质量、绘图质量、图文回应质量五个维度进行评估。实验发现现有模型存在三大不足：语言流畅性常优于视觉证据的忠实性、安全性是普遍弱项、处理情绪紧张的交互比应对低教育/健康素养更困难。MedImageEdu为评估多模态代理能否“依据证据教学”而非“仅从文本回答”提供了可控测试平台。

GlobalSplat: Efficient Feed-Forward 3D Gaussian Splatting via Global Scene Tokens（GlobalSplat：通过全局场景令牌实现高效前馈3D高斯溅射）

本文核心提出了一种名为GlobalSplat的创新框架，旨在解决现有3D高斯溅射技术在表示紧凑性、重建速度与渲染质量之间难以权衡的问题。传统方法（无论是迭代优化还是前馈推理）主要依赖缺乏全局场景感知的局部启发式分配策略，导致表示冗余、尺寸膨胀和全局一致性脆弱。GlobalSplat采用“先对齐，后解码”原则，首先学习一个紧凑的、全局的潜在场景表示来编码多视图输入并解析跨视图对应关系，然后再解码为显式3D几何。该方法无需依赖预训练的像素预测主干网络或重用密集基线的潜在特征，即可实现紧凑且全局一致的重建。通过由粗到细的训练课程逐步增加解码能力，本征地防止了表示膨胀。在RealEstate10K和ACID数据集上，该模型仅使用16K个高斯（远少于密集流程所需）即可实现具有竞争力的新视图合成性能，模型轻量至4MB。此外，其单次前向传播推理时间低于78毫秒，显著快于基线方法。

MM-WebAgent: A Hierarchical Multimodal Web Agent for Webpage Generation（分层多模态网页代理：用于网页生成）

人工智能生成内容（AIGC）工具的快速发展，使得图像、视频和可视化内容能够按需生成，为现代网页UI/UX设计提供了灵活且日益普及的范式。然而，直接将此类工具集成到自动化网页生成中，常因元素孤立生成而导致风格不一致和全局协调性差的问题。本文提出MM-WebAgent，一个用于多模态网页生成的分层代理框架，它通过分层规划和迭代自我反思来协调基于AIGC的元素生成。该框架联合优化全局布局、局部多模态内容及其整合，从而生成协调一致、视觉统一的网页。此外，本文还引入了一个多模态网页生成基准测试集和一个用于系统评估的多层次评估方案。实验表明，MM-WebAgent在性能上超越了基于代码生成和基于代理的基线方法，尤其是在多模态元素生成与整合方面表现优异。

Unsupervised Skeleton-Based Action Segmentation via Hierarchical Spatiotemporal Vector Quantization（基于分层时空矢量量化的无监督骨骼动作分割）

本文提出了一种新颖的分层时空矢量量化框架，用于无监督的基于骨骼序列的时间动作分割。核心创新在于：

1. 分层量化设计：采用两级矢量量化，底层将骨骼数据关联到细粒度的子动作，高层进一步将子动作聚合成动作级表示，性能优于非分层基线。

2. 时空信息融合：扩展框架以同时利用空间（骨骼结构）和时间（时序）信息，通过重建输入骨骼及其对应时间戳进行多级聚类。

3. 卓越性能：在HuGaDB、LARa和BABEL等多个基准测试上的实验表明，该方法取得了最先进的性能，并减少了分割长度偏差。

StoryCoder: Narrative Reformulation for Structured Reasoning in LLM Code Generation（StoryCoder：用于大语言模型代码生成中结构化推理的叙事重构）

本文提出StoryCoder框架，旨在通过将代码生成问题重构为连贯的自然语言叙事来提升大语言模型的代码生成能力。该叙事包含任务概述、约束条件和测试用例三个部分，并受选定算法和风格指导。实验表明，该方法在多个基准测试上显著提升了模型性能（零样本pass@10平均提升18.7%），能有效引导模型采用正确的算法策略、减少实现错误并生成更模块化的代码结构。分析指出，叙事的一致性和风格匹配是关键，强调了结构化问题表征对代码生成的重要性。

StreamCacheVGGT: Streaming Visual Geometry Transformers with Robust Scoring and Hybrid Cache Compression（流式视觉几何Transformer：基于鲁棒评分与混合缓存压缩）

本文针对连续视频流中密集三维几何重建任务，在恒定内存限制下，解决现有“纯淘汰”范式因二值化令牌删除和局部单层评分噪声导致信息严重破坏的问题。提出了无需训练的StreamCacheVGGT框架，其核心创新在于两个协同模块：跨层一致性增强评分（CLCES）通过追踪Transformer层级间的令牌重要性轨迹，利用顺序统计分析识别持续的几何显著性，以降低激活噪声；混合缓存压缩（HCC）则利用上述评分，引入三级筛选策略，在关键向量流形上通过最近邻分配将中等重要性令牌合并至保留锚点，从而超越简单淘汰，保留关键几何上下文。在五大基准数据集上的实验表明，该方法在严格遵循恒定计算成本约束下，实现了最优的重建精度与长期稳定性，确立了新的技术标杆。

CausalDetox: Causal Head Selection and Intervention for Language Model Detoxification（因果排毒：面向语言模型脱毒处理的因果注意力头选择与干预）

大语言模型（LLMs）常生成有害内容，对其安全部署构成显著风险。现有缓解策略常损害生成质量或需昂贵人工标注。本文提出CAUSALDETOX框架，通过因果分析识别并干预导致有害生成的具体注意力头。利用“必要性与充分性概率”（PNS），我们分离出对毒性产生既必要又充分的最小注意力头集合。我们通过两种互补策略利用这些组件：（1）局部推理时干预，构建动态、输入特定的引导向量，实现上下文感知的脱毒；（2）PNS引导的微调，永久性消除有害表征。我们还引入了PARATOX新基准，包含对齐的有害/无害句对，支持可控的反事实评估。在ToxiGen、ImplicitHate和ParaDetox数据集上的实验表明，CAUSALDETOX相比基线方法毒性降低幅度最高提升5.34%，同时保持语言流畅性，并在注意力头选择上实现了7倍加速。

Implicit Neural Representations: A Signal Processing Perspective（隐式神经表示：信号处理视角）

隐式神经表示（INRs）标志着信号建模的根本性转变，从离散采样数据转向连续的函数表示。它将信号参数化为神经网络，为图像、音频、视频、3D几何等提供了统一的连续函数表示框架。此函数视角使得微分等信号操作可通过自动微分解析进行，而非离散近似。本文从信号处理视角审视INRs的演进，强调其频谱特性、采样理论和多尺度表示。研究追溯了从具有低频分量频谱偏置的标准坐标网络，到通过周期性、局部化和自适应等专用激活函数重塑近似空间的更先进设计。同时探讨了提升空间适应性与计算效率的结构化表示（如层次分解和哈希网格编码）。文章进一步凸显了INRs在医学与雷达成像逆问题、压缩及3D场景表示等广泛应用中的效用。通过将INRs解读为近似空间能适应底层数据的学习信号模型，本文阐明了该领域的核心概念发展，并概述了理论稳定性、权重空间可解释性及大规模泛化等方面的开放挑战。

Layered Mutability: Continuity and Governance in Persistent Self-Modifying Agents（分层可变性：持久性自修改智能体的连续性与治理）

本文提出了“分层可变性”框架，用于分析持久性语言模型智能体的行为演化过程。该框架将影响智能体行为的可变因素划分为五个层级：预训练、训练后对齐、自我叙事、记忆以及权重级适应。核心观点指出，当智能体的内部状态发生快速突变、下游耦合性强、可逆性弱且可观测性低时，治理难度会显著增加。这导致对人类最易审查的层级与最影响行为的层级之间出现系统性错配。作者通过简单的漂移、治理负荷和滞后量来形式化这一观点，并通过初步实验表明，即使在恢复智能体可见的自我描述后，其累积记忆仍会导致基线行为无法复原（身份滞后比估计为0.68）。研究的重要启示在于，此类智能体的关键失效模式并非突发性失调，而是组合性漂移：即局部合理的更新逐步累积，最终形成从未被明确授权的行为轨迹。

From Tokens to Steps: Verification-Aware Speculative Decoding for Efficient Multi-Step Reasoning（从词元到步骤：面向高效多步推理的验证感知推测解码）

推测解码通过轻量草稿模型生成输出、再由强目标模型验证，以加速大语言模型推理。但其以词元为中心的特性会导致错误步骤传播。现有方法依赖外部奖励模型来缓解此问题，但会引入额外延迟、计算开销并限制泛化能力。本文提出SpecGuard，一个仅利用模型内部信号进行步骤级验证的验证感知推测解码框架。在每一步，SpecGuard采样多个草稿候选，选择最一致的步骤，并使用两种轻量级模型内部信号进行集成验证：(i) 基于注意力权重的“基础评分”，衡量对输入和已接受步骤的归因；(ii) 基于对数概率的“置信度评分”，捕捉词元级置信度。这些信号共同决定一个步骤是被接受，还是需调用目标模型重新计算，从而有选择地分配计算资源。在一系列推理基准测试上的实验表明，SpecGuard在降低约11%延迟的同时，将准确率提高了3.6%，性能优于标准推测解码及奖励模型引导的推测解码。

Pushing the Limits of On-Device Streaming ASR: A Compact, High-Accuracy English Model for Low-Latency Inference（突破设备端流式语音识别极限：面向低延迟推理的紧凑高精度英语模型）

本研究旨在开发完全在CPU上运行、无需GPU加速的高质量设备端自动语音识别模型，需同时优化精度、延迟和内存占用。通过对编码器-解码器、传感器和基于大语言模型等多种前沿ASR架构在批量、分块和流式推理模式下的系统实证研究，在超过50种配置的基准测试中，确定NVIDIA Nemotron Speech Streaming为资源受限硬件上实时英语流式识别的最佳候选方案。随后，我们在ONNX Runtime中重新实现了完整的流式推理流程，并对多种训练后量化策略进行了对比评估，包括重要性加权的k-quant、混合精度方案和就近舍入量化，并结合了图级算子融合。这些优化将模型大小从2.47 GB压缩至最低0.67 GB，同时将词错误率保持在全精度PyTorch基线1%的绝对误差范围内。我们推荐的int4 k-quant变体在八个标准测试集上实现了8.20%的平均流式WER，在CPU上以0.56秒的算法延迟流畅运行，速度快于实时要求，为设备端流式ASR建立了一个新的质量-效率帕累托最优点。

Predicting Post-Traumatic Epilepsy from Clinical Records using Large Language Model Embeddings（利用大语言模型嵌入从临床记录预测创伤后癫痫）

目的：创伤后癫痫（PTE）是创伤性脑损伤（TBI）后的一种严重神经系统疾病。由于临床数据异质性、阳性病例有限且依赖资源密集的神经影像数据，其早期预测仍具挑战。本研究旨在探索仅使用常规收集的急性期临床记录，能否通过基于语言模型的方法支持早期PTE预测。

方法：基于TRACK-TBI队列的精选子集，我们开发了一个自动化PTE预测框架。该框架将预训练的大语言模型（LLMs）作为固定特征提取器，对临床记录进行编码。通过分层交叉验证，使用梯度提升树分类器评估了表格特征、LLM生成的嵌入以及混合特征表示的性能。

结果：与单独使用表格特征相比，LLM嵌入通过捕捉上下文临床信息实现了性能提升。最佳性能由结合表格特征与LLM嵌入的模态感知特征融合策略实现，其AUC-ROC达到0.892，AUPRC达到0.798。急性创伤后癫痫发作、损伤严重程度、神经外科干预和ICU住院是预测性能的关键贡献因素。

意义：这些发现表明，常规急性期临床记录中包含的信息，足以通过LLM嵌入结合梯度提升树分类器进行早期PTE风险预测。该方法是对基于影像预测的一种有前景的补充。

Physically-Induced Atmospheric Adversarial Perturbations: Enhancing Transferability and Robustness in Remote Sensing Image Classification（物理诱导的大气对抗扰动：提升遥感图像分类的迁移性与鲁棒性）

对抗攻击严重威胁遥感图像分类深度学习模型的可靠性。现有方法多依赖直接像素扰动，未能利用遥感图像固有的大气特性，也难以在真实图像退化中保持有效。本文提出FogFool，一种物理上合理、基于雾效的对抗扰动生成框架，通过迭代优化基于Perlin噪声的大气模式来建模具有自然不规则结构的雾形成。该方法生成的对抗样本不仅视觉上与真实遥感场景一致、具有欺骗性，而且通过利用大气现象的空间连贯性和中低频特性，将对抗信息嵌入到不同模型架构共享的结构特征中。在两个基准遥感数据集上的大量实验表明，FogFool性能卓越：不仅在白盒设置下表现优异，还展现出卓越的黑盒迁移性（攻击成功率高达83.74%），并对JPEG压缩、滤波等基于预处理的防御具有强鲁棒性。混淆矩阵和类激活图可视化等详细分析表明，我们的大气驱动扰动能引发模型注意力的普遍性偏移。这些结果证明，FogFool对遥感分类系统构成了实用、隐蔽且高度持久的威胁，为评估复杂环境下的模型可靠性提供了强有力的基准。

GenRec: A Preference-Oriented Generative Framework for Large-Scale Recommendation (GenRec：面向偏好的大规模推荐生成框架)

摘要：生成式检索为推荐系统提供了新范式，但大规模工业应用面临三大挑战：分页请求导致输出不一致、长用户行为序列编码成本高昂、生成策略与复杂用户偏好难以对齐。本文提出的GenRec框架部署于京东App，在单一解码器架构内解决了这些问题。其核心创新包括：1) 训练目标：提出页面级下一词预测任务，缓解单点训练歧义并提供更密集梯度信号；2) 输入处理：设计非对称线性令牌合并器，将输入长度压缩约2倍且精度损失可忽略；3) 策略对齐：提出GRPO-SR强化学习方法，结合组相对策略优化与负对数似然正则化确保稳定性，并采用混合奖励机制防止奖励攻击。在线A/B测试表明，GenRec使点击量和交易量分别提升9.5%和8.7%。

RL-STPA: Adapting System-Theoretic Hazard Analysis for Safety-Critical Reinforcement Learning（RL-STPA：面向安全关键强化学习的系统理论危险分析适配）

本文针对强化学习在安全关键领域应用时，传统评估方法难以系统识别神经网络策略黑盒性及训练-部署分布偏移所引发危险的问题，提出了RL-STPA框架。该框架通过三项核心创新，将传统系统理论过程分析适配于强化学习：1）结合时序阶段分析与领域知识进行分层子任务分解，以捕捉涌现行为；2）采用覆盖引导的扰动测试，探索状态-动作空间的敏感性；3）建立迭代检查点，通过奖励塑形与课程设计将识别出的危险反馈至训练过程。在无人机自主导航与着陆的安全关键测试案例中，RL-STPA揭示了标准评估可能遗漏的潜在失效场景。该框架为从业者提供了系统危险分析工具包、安全覆盖评估的量化指标以及建立操作安全边界的可行指南。尽管无法为任意神经策略提供形式化保证，但在详尽验证方法仍不可行的安全关键应用中，RL-STPA为系统评估与提升强化学习的安全性与鲁棒性提供了一种实用方法论。

Beyond Literal Summarization: Redefining Hallucination for Medical SOAP Note Evaluation（超越字面总结：重新定义医疗SOAP记录评估中的幻觉）

评估大语言模型在生成医疗SOAP记录等临床文档任务中的表现面临挑战。此类任务不同于标准摘要，需要临床抽象、口语规范化及基于医学的推理。然而，当前主流的评估方法（包括自动指标和LLM作为评判框架）依赖词汇忠实度，常将病历转录本中未明确出现的信息一概标记为“幻觉”。

研究表明，这些方法系统性地将临床有效的输出误判为错误，虚高了幻觉率并扭曲了模型评估。分析发现，许多被标记的“幻觉”实则为合理的临床信息转换，包括同义词映射、检查结果的抽象、诊断推理以及符合指南的护理计划。

通过基于医学术语库进行校准提示和检索，使评估标准与临床推理对齐后，结果发生显著变化。在词汇评估体系下，平均幻觉率为35%，严重惩罚了有效推理；而在具备推理意识的评估下，该比率降至9%，剩余案例才反映真实的安全隐患。这些发现表明，当前评估方法过度惩罚了有效的临床推理，可能测量的是评估设计本身的人为缺陷而非真实错误，强调在医学等高语境领域需要采用结合临床知识的评估框架。

CMTM: Cross-Modal Token Modulation for Unsupervised Video Object Segmentation（跨模态令牌调制用于无监督视频目标分割）

近期无监督视频目标分割的进展突显了融合外观与运动信息的双流架构的潜力。然而，要充分利用这些互补信息源，需有效建模其相互依赖关系。本文提出了一种新颖的跨模态令牌调制方法，旨在强化外观与运动线索间的交互。该方法在各模态的令牌间建立密集连接，通过关系变换器块实现高效的模态内与模态间信息传播。为提高学习效率，我们引入了一种令牌掩码策略，以解决仅依赖增加模型复杂度的局限性。本方法在所有公开基准测试中均达到了最先进的性能，超越了现有方法。

OmniGCD: Abstracting Generalized Category Discovery for Modality Agnosticism（OmniGCD：面向模态无关性的广义类别发现抽象）

本文提出了一种模态无关的广义类别发现（GCD）方法OmniGCD，其灵感来源于人脑的抽象类别形成机制。与以往局限于单一模态、需针对特定数据集进行微调的GCD方法不同，OmniGCD利用特定模态编码器处理视觉、音频、文本等多模态输入，并通过降维构建GCD潜在空间。在测试时，通过一个基于合成数据训练的新型Transformer模型将该空间转换为更适合聚类的表示。

研究引入了零样本GCD设置，禁止对特定数据集进行微调，从而实现真正的模态无关类别发现。OmniGCD仅需在合成数据上训练一次，即可在涵盖四种模态的16个数据集上执行零样本GCD，在已知类和新类分类准确率上均超越基线方法（在视觉、文本、音频和遥感模态上平均分别提升+6.2、+17.9、+1.5和+12.7个百分点）。

这项工作强调了强大编码器的重要性，同时将表示学习与类别发现解耦。模态无关方法的改进将惠及所有模态，使编码器开发独立于GCD任务。本研究为未来模态无关的GCD工作设立了基准，为可扩展的、类人的类别发现铺平了道路。

NG-GS: NeRF-Guided 3D Gaussian Splatting Segmentation（NG-GS：神经辐射场引导的3D高斯溅射分割）

本文针对3D高斯溅射（3DGS）中因高斯表示离散性导致物体边界分割不准确的问题，提出了NG-GS框架。该框架的核心创新在于：首先通过掩码方差分析自动识别边界处的模糊高斯；接着利用径向基函数插值构建空间连续的特征场，并采用多分辨率哈希编码进行高效多尺度表征；最后通过联合优化策略，结合对齐损失和空间连续性损失，将3DGS与轻量级NeRF模块对齐，从而确保分割边界的平滑性与一致性。在多个基准数据集上的实验表明，该方法在边界mIoU指标上取得显著提升，实现了最先进的性能。

H2VLR: Heterogeneous Hypergraph Vision-Language Reasoning for Few-Shot Anomaly Detection（H2VLR：用于小样本异常检测的异构超图视觉-语言推理）

作为经典视觉任务，异常检测在工业质检和医学影像中应用广泛，但常面临数据稀缺问题。小样本异常检测方案因此备受关注。近年来，超越传统视觉范式，视觉-语言模型被广泛探索以推动该领域发展。然而，现有基于VLM的方案几乎都仅通过成对特征匹配进行异常推断，忽略了结构依赖和全局一致性。为利用VLM进一步推动FSAD，本文提出异构超图视觉-语言推理框架。该框架将FSAD重新定义为视觉-语义关系的高阶推理问题，通过在统一超图中联合建模视觉区域和语义概念。实验对比验证了H2VLR的有效性与优势，其能在代表性的工业和医学基准上达到最先进的性能。

Personalized and Context-Aware Transformer Models for Predicting Post-Intervention Physiological Responses from Wearable Sensor Data（基于可穿戴传感器数据预测干预后生理反应的个性化情境感知Transformer模型）

消费者可穿戴设备能够持续测量与压力和恢复相关的生理数据，但将这些数据流转化为可操作的、个性化的压力管理建议仍具挑战。用户通常难以预知一项旨在减压的干预活动，会如何影响未来15至120分钟的心率、心率变异性或心跳间隔。本文提出一个框架，用于预测这些生理指标在干预后随时间变化的轨迹及变化方向。该方法结合了Transformer模型，以预测相对于干预前基线的多时间跨度百分比变化轨迹，以及各时间点的变化方向（正向、负向或中性），并利用叠加了用户标记事件与干预措施的可穿戴传感器数据进行了实证研究。该概念验证表明，个性化的干预后预测是可行的。我们鼓励未来在更大规模研究和适当监管审查后，将该框架整合到压力管理工具中，为个人提供量身定制的每日干预建议。

MapSR: Prompt-Driven Land Cover Map Super-Resolution via Vision Foundation Models (MapSR：基于视觉基础模型的提示驱动土地覆盖图超分辨率)

高分辨率土地覆盖制图常受限于密集标注的高昂成本。本文从地图超分辨率角度重新审视该问题，旨在将粗糙的低分辨率土地覆盖产品提升至输入影像分辨率的高分辨率地图。现有弱监督方法虽可利用低分辨率标签，但通常需以高计算成本重新训练密集预测器。本文提出MapSR，一个将监督与模型训练解耦的提示驱动框架。MapSR仅使用一次低分辨率标签，通过轻量级线性探针从冻结的视觉基础模型特征中提取类别提示，随后通过免训练的度量推理和基于图的预测细化进行高分辨率制图。具体而言，通过聚合线性探针识别的高置信度高分辨率特征来估计类别提示，并通过余弦相似度匹配及基于图传播的空间细化获得高分辨率预测。在切萨皮克湾数据集上的实验表明，MapSR在无需任何高分辨率标签的情况下实现了59.64%的平均交并比，与最强的弱监督基线方法表现相当，并超越了全监督基线。值得注意的是，MapSR将可训练参数减少了四个数量级，并将训练时间从数小时缩短至数分钟，从而在有限标注和计算预算下实现了可扩展的高分辨率制图。

Boundary-Centric Active Learning for Temporal Action Segmentation（边界中心的时间动作分割主动学习框架）

时间动作分割任务需要密集的时间标注，但标注成本主要消耗在识别和修正动作边界上，此处也是分割错误集中、微小时间偏移会严重影响评估指标的区域。本文提出的B-ACT框架，是一种在剪辑预算约束下的主动学习方法，其核心是明确地将标注资源优先分配给这些高影响力的边界区域。

B-ACT采用分层两阶段循环工作：(1) 基于预测不确定性对未标注视频排序并选择查询；(2) 在每个选中的视频内，从当前模型预测中检测候选边界，并通过一个融合了邻域不确定性、类别模糊性和时间预测动态的新边界评分，选择最重要的K个边界进行标注。关键的是，我们的标注协议仅请求边界帧的标签，但训练时使用以边界为中心的剪辑片段，以利用模型感受野中的时序上下文信息。

在GTEA、50Salads和Breakfast数据集上的大量实验表明，这种以边界为中心的监督策略实现了卓越的标注效率，在稀疏标注预算下持续超越了代表性的时间动作分割主动学习基线方法和先前最优方法，在边界定位主导编辑和基于重叠率F1分数的数据集上提升尤为显著。

RELOAD: A Robust and Efficient Learned Query Optimizer for Database Systems（RELOAD：面向数据库系统的鲁棒高效学习型查询优化器）

摘要：当前查询优化技术正从传统的基于规则和成本的方法转向机器学习驱动。其中，强化学习因其能通过学习查询规划策略来优化长期性能而备受关注。然而，现有基于强化学习的查询优化器在单查询级别常表现出性能不稳定，包括严重的性能退化，且需长时间训练才能达到专家级基于成本优化器的计划质量。这些缺陷阻碍了学习型查询优化器在实际生产数据库系统中的部署与应用。为解决这些问题，本文提出了RELOAD，一个面向数据库系统的鲁棒高效学习型查询优化器。RELOAD专注于两个核心目标：一是鲁棒性，通过最小化查询级性能退化并确保跨执行过程优化行为的一致性；二是效率，通过加速收敛至专家级计划质量。在Join Order Benchmark、TPC-DS和Star Schema Benchmark等标准基准上的大量实验表明，相较于最先进的基于强化学习的查询优化技术，RELOAD实现了高达2.4倍的鲁棒性提升和3.1倍的效率提升。

xFODE: An Explainable Fuzzy Additive ODE Framework for System Identification（xFODE：一种用于系统辨识的可解释模糊加性常微分方程框架）

深度学习（DL）的进步强化了数据驱动的系统辨识（SysID），但现有神经/模糊常微分方程（NODE/FODE）模型的状态常缺乏物理意义，且输入对状态导数的贡献难以解释。为此，本文提出可解释FODE（xFODE）框架。该框架通过增量形式定义状态以赋予其物理意义，并采用模糊加性模型逼近状态导数，从而增强每个输入的可解释性。此外，通过开发分区策略（PSs）来训练具有可解释性的模糊加性模型，确保对任意输入仅激活两个连续规则，这降低了局部推理复杂度并增强了前件空间的可解释性。为训练xFODE，本文提出了一个支持端到端优化的、包含参数化隶属函数学习的深度学习框架。在多个基准SysID数据集上的实验表明，xFODE在匹配NODE、FODE和NLARX模型精度的同时，提供了可解释的洞察。
