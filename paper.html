准确分割皮层下区域对神经外科规划和功能研究至关重要。大多数自动方法依赖于模板空间配准，但这可能损害患者特异性精度，尤其是在小结构中。本研究评估了原生空间方法是否具有可测量的优势，并在运动障碍背景下进行了检验。我们开发了两个基于UNet的分割管路，分别用于分割丘脑底核（帕金森病常见手术靶点）、邻近的红核和黑质。我们收集了五个公共数据集的7T和3T MRI数据，在原生空间中对照手动标签评估管路，并进一步研究了模板分辨率的影响。假设模型在更高场强下能更好学习目标边界，我们测试了7T训练模型向3T临床图像的迁移能力，以及通过解缠表征学习生成的合成3T训练数据是否能弥合这一领域差距。在保留的7T数据上，原生管路始终优于模板管路。对于丘脑底核，原生空间Dice达到0.775±0.055，而模板管路（1 mm模板）为0.713±0.051；HD95分别为0.79±0.24 mm和1.17±1.10 mm。红核和黑质也观察到类似优势。提高模板分辨率并未改善精度。应用于3T图像时，所有模型均出现显著性能下降。添加合成3T数据仅带来微小改善，但未降低7T性能。对于需要患者特异性解剖保真度的应用（如帕金森病手术规划），原生空间分割更优。弥合7T到3T的领域差距仍是一个开放挑战，这激发了针对皮层下结构的领域自适应未来研究。

Selective Capability Unlearning in End-to-End Spoken Language Understanding（端到端口语语言理解中的选择性能力遗忘）

现代口语语言理解（SLU）系统在实际部署中，常需因政策或安全约束移除特定功能（即意图及其对应的槽生成行为）。然而，在自回归模型中，抑制目标意图并不能消除该意图下生成槽的条件映射——当意图前缀由外部提供时，模型仍可重构原始的意图-槽结构。我们将此结构性缺陷称为“能力固化”。为此，我们提出绑定子空间（BSU），一种在表征层隔离并衰减该映射中意图条件方向的框架。在多个SLU基准测试中，BSU显著降低了强制前缀的可恢复性，同时保持了模型保留性能。

Can Scale Save Us From Plasticity Loss in Large Language Models?（规模能否拯救我们免于大语言模型的可塑性损失？）

可塑性损失（网络在学习旧信息后学习新信息的能力丧失）是构建具备持续学习能力的人工神经网络的根本挑战。为探究该问题在现代基于Transformer的LLM范式中是否依然存在，本研究在GPT风格Transformer模型上进行了多语言持续学习实验。实验发现，在参数规模从500万到3.14亿（非嵌入参数）的模型中，均观测到可塑性损失，具体表现为固定越南语探测任务的性能恶化。可塑性损失的出现遵循可预测的标度律，与模型规模呈次线性增长，表明更大模型虽能延迟可塑性损失的显现，但单纯增加参数量不足以完全阻止该现象。此外，在稳态多语言训练场景中也发现了可塑性损失，挑战了该现象仅存在于任务突变的持续学习中的观点。总体而言，研究结果表明，经过足够长时间的训练，无论是持续学习还是稳态设置下，即便是基于自然语言训练的大型Transformer模型，最终都将丧失高效适应新数据的能力。

Code Isn't Memory: A Structural Codebase Index Inside a Coding Agent（代码不是记忆：编码代理中的结构化代码库索引）

该研究在固定编码代理框架和固定模型（Claude Opus 4.7）下，通过三组实验（带索引框架、不带索引框架以及agentic-grep对比组），在SWE-PolyBench Verified和SWE-bench Pro基准上评估了结构化代码库索引对成本和解决率的影响。结果表明：在框架内消融实验中，索引带来了显著的定位提升和统计上可分离的解决率提升，且每单元无额外成本，每次解决的成本更低；跨框架对比显示，索引在解决率和定位上均未低于agentic-grep基线，同样无成本惩罚。因此，结构化代码库索引的部署问题不在于运行成本过高（各随机种子下，索引的每解决成本均低于agentic-grep），而在于工作负载是否包含需要结构排序的多文件变更。

Breaking Shortcut Learning for Cross-Trial EEG-Guided Target Speech Extraction via Two-Stage Training（通过两阶段训练打破跨试验脑电引导目标语音提取中的捷径学习）

近期用于脑电（EEG）引导目标语音提取的端到端模型取得了令人瞩目的成果，凸显了神经控制听力技术的潜力。然而，我们的分析表明，高试验内性能可能源于充当目标选择捷径的试验特定EEG结构，这导致模型在未见过的试验上泛化能力差。为克服这一局限，我们提出TRUST-TSE，一种两阶段框架以减轻捷径学习。通过引入基于听者负采样的对比预训练，我们促使EEG编码器捕捉细粒度的EEG-语音对齐，同时抑制试验身份线索。我们还采用基于EEG-源相似性的置信加权提取目标，利用所学表征指导提取。在KUL和DTU数据集上的实验表明，在严格的跨试验协议下，TRUST-TSE优于端到端基线方法，解决了现有方法的关键可靠性瓶颈。

EMA-FS: Accelerating GBDT Training via Gain-Informed Feature Screening（EMA-FS：通过增益信息特征筛选加速GBDT训练）

梯度提升决策树（GBDT，以LightGBM为例）的训练时间主要消耗在构建每个特征的直方图上（通常占65-70%）。现有方法（如随机特征子采样）在丢弃特征时未考虑其预测效用。我们提出基于EMA的特征筛选（EMA-FS），这是一种算法级优化方法：它在提升迭代过程中维护每个特征分裂增益的指数移动平均（EMA），经过短期预热后，仅对历史增益排名前K的特征构建直方图。与随机子采样不同，EMA-FS是一种有信息依据的方法：它保留高增益特征，同时筛除低增益特征。该方法在每棵树级别运行，完全兼容LightGBM的直方图减法技巧，无需修改核心例程。

我们在金融欺诈检测、广告点击率预测、工业质量控制和合成基准等数据集上进行了评估，特征维度从29到968不等。在密集的中高维数据上，该方法实现了显著加速：在500特征的合成基准上达到2.61倍加速，在432特征的IEEE-CIS欺诈数据集上以30%保留率实现1.45倍加速；以70%保留率运行时，AUC提升0.11点，同时实现1.34倍加速。在极度稀疏数据（Bosch数据集，缺失率>90%）上未见加速，因为LightGBM的稀疏分箱优化已能跳过空值。

我们进一步提出了随机EMA-FS（S-EMA-FS），该方法用由浓度参数β控制的增益加权随机采样替代确定性top-K选择，将确定性EMA-FS（β→∞）和随机子采样（β=0）统一在一个框架中。两种方法在LightGBM所有六个树学习器中仅用约120行C++代码实现，且完全向后兼容。

Training the Orchestrator: A Supervised Approach to End-to-End PDDL Planning with LLM Agents（训练编排器：一种基于监督学习的端到端PDDL规划方法，结合大语言模型智能体）

核心问题：将自然语言规划意图转化为经过验证的形式化规划（PDDL）是一大挑战；现有方法依赖大型语言模型作为编排器，每次细化步骤都需调用昂贵API。

提出方案：HALO（混合智能体-学习编排器）——从外部验证器证实的有效规划轨迹中训练编排器，覆盖11个PDDL领域。HALO采用小型QLoRA微调策略与3条硬编码规则（处理可简单判断的选择），在扩展的21个智能体动作空间中运行。

关键创新：不同于每步调用大模型或用稀疏回报训练，HALO利用验证器提供的强监督信号：每个被接受的轨迹均为可直接用于监督学习的正确（状态，智能体）决策序列。

实验结果：在PlanBench、Natural Plan及经典规划基准上，HALO在成功率上达到或超过GPT-5-mini基线，比更强基线Gemini-3-Flash低3个百分点以内；编排成本降低一个数量级以上（每任务成本从0.18美元降至0.004美元，约为GPT-5-mini的1/45、Gemini-3-Flash的1/15），每次任务的LLM调用次数减少40%-50%。

Finding the Time to Think: Learning Planning Budgets in Real-Time RL（寻找思考的时间：在实时强化学习中学习规划预算）

核心观点总结：标准强化学习（RL）中环境会无限等待智能体决策，但在实时场景中，环境在等待时仍在推进，因此深思熟虑需要付出时间代价。本文提出了一种可变延迟实时RL框架，智能体在每个决策点可自行选择思考时长。针对规划型智能体，作者训练了一个轻量级的门控策略，通过根据环境状态选择规划预算来解决“规划瘫痪”问题。在实时吃豆人、俄罗斯方块、贪吃蛇、Speed Hex和Speed Go等任务中，该门控策略在性能上优于固定预算和启发式基线，并能成功迁移到智能体与环境运行在不同GPU上的真实分布式场景中。

Projection-Volume Fidelity Divergence: Diagnosing and Controlling Optimization Drift in Sparse-View 3D Gaussian Tomography（投影-体积保真度分歧：稀疏视角三维高斯层析成像中的优化漂移诊断与控制）

稀疏视角计算机断层扫描是一个严重病态的逆问题。现有3D高斯泼溅方法虽为层析重建提供了高效显式表示，但研究发现，在此场景下投影域优化可能具有误导性：渲染投影持续改善，而重建体积却不断恶化。我们将这种故障模式定义为投影-体积保真度分歧（PVFD），即稀疏雷东约束下各向异性高斯形变与视角特定基元共适应导致的表示级优化漂移。为表征此现象，我们引入几何与体积级诊断指标，用以测量针状高斯退化和体素化密度场稳定性。基于这些观察，我们提出LADES——一种无需真实值的稀疏视角高斯层析优化控制器。该方法结合线性退火丢弃（早期训练中施加强随机掩码以阻断基元过早共适应，再逐步恢复全容量以巩固结构）与结构感知早停（根据高斯种群增长饱和而非验证PSNR终止密集化）。稀疏视角CT重建实验表明，LADES显著提升体积保真度，抑制结构退化，在保持投影精度前提下大幅缩短训练时间。这证明鲁棒的高斯层析重建需监测并控制体积结构，而非仅优化投影拟合。
