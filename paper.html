Rethinking the Need for Source Models: Source-Free Domain Adaptation from Scratch Guided by a Vision-Language Model（重新思考源模型的需求：视觉语言模型引导下的从零开始的无源域自适应）

传统无源域自适应（SFDA）虽无需源域数据，但仍需从源预训练模型初始化，并非真正无源。引入视觉语言（ViL）模型后，实验发现不同源模型对同一目标域的最终结果影响极小，说明源模型作用有限。为此，本文提出更严格的ViL-Only Domain Adaptation（VODA）设定，完全消除对源域的依赖，仅依靠随机初始化模型、ViL模型和无标签目标数据。我们分析了VODA的自适应动态，并引入Two-Stage Denoised-Region Distillation（TS-DRD）两阶段框架：先通过ViL引导预热模型，再寻找ViL与自适应模型共有的去噪区域，提供更干净的蒸馏监督。在Office-Home、VisDA和DomainNet-126上的实验表明，TS-DRD在VODA设定下达到了与仍需源模型的传统SFDA方法相当或更优的性能，证明了其有效性与VODA设定的潜力。

T$^2$PO: Uncertainty-Guided Exploration Control for Stable Multi-Turn Agentic Reinforcement Learning（T$^2$PO：不确定性引导的探索控制实现稳定多轮智能体强化学习）

UnGAP: Uncertainty-Guided Affine Prompting for Real-Time Crack Segmentation（UnGAP：不确定性引导的仿射提示实现实时裂缝分割）

实时裂缝分割对于结构健康监测至关重要，但常受光照变化、模糊和纹理歧义等偶然不确定性影响。现有方法通常将不确定性估计视为事后分析的被动终点，未能将其反馈至特征表示优化形成闭环。
本文认为，独立像素的异方差建模特别适用于裂缝分割，因为裂缝由细粒度局部梯度定义，而非依赖全局语义一致性。然而，该方法存在结构优化病态：高预测方差会减弱损失梯度，导致模型忽略困难样本并欠拟合复杂边界。
为此，我们提出UnGAP框架，在不确定性估计与特征学习之间建立闭环机制。核心是**不确定性提示特征调制器（UPFM）**，它将偶然不确定性视为主动视觉提示而非单纯输出。UPFM通过像素级仿射变换动态校准特征分布，并通过将高方差（否则会抑制梯度）转化为模糊区域更强的特征修正信号，缓解异方差病态。此外，引入边界感知检测头进一步约束预测精度。
大量实验证明，UnGAP在实现卓越分割精度的同时保持实时推理速度，有效验证了将不确定性从被动指标转化为主动校准工具的优势。

The Causal Description Gap: Information-Theoretic Separations Across Pearl's Hierarchy（因果描述差距：珀尔层级中的信息论分离）

珀尔的因果层级表明，观察性、干预性和反事实查询在性质上是不同的。本文提出了一个量化版本的问题：在已知低层级答案的基础上，指定高层级因果答案需要额外多少比特信息？我们通过查询类描述长度来形式化这一问题，即针对某类查询，由结构因果模型（SCM）引发的答案预言机的柯尔莫哥洛夫复杂度。我们的主要构造给出了二元无环SCM，其观察性分布的描述长度恒定，而单变量干预答案预言机的描述长度为 $Θ(n^2)$。一个度敏感的上界表明，入度为 $d$ 的有限门模式SCM的观察-干预差距最多为 $O(nd \log(en/d) + n \log n)$，这使得二次构造在密集情况下达到阶最优，而有根树构造在有界入度情况下达到阶最优。对于每个固定的 $\varepsilon < 1/4$，二次分离在 $\varepsilon$-精确的全变差描述下仍然成立。在更高层级，完整的硬干预预言机仍可留下 $Θ(n)$ 的反事实描述差距。一个通用的模糊到比特定理及其香农类比表明，这些差距等于残余高层级模糊度的对数，忽略低阶项。

When Alignment Isn't Enough: Response-Path Attacks on LLM Agents（当对齐不够时：针对LLM智能体的响应路径攻击）

SCPRM: A Schema-aware Cumulative Process Reward Model for Knowledge Graph Question Answering（SCPRM：一种模式感知的累积过程奖励模型用于知识图谱问答）

大语言模型擅长复杂推理，但评估它们的中间步骤依然困难。过程奖励模型虽提供逐步监督，却常受风险补偿效应影响，即错误步骤被后续正确步骤抵消，导致对有缺陷的推理路径也给予高分。在知识图谱推理中，由于起点和终点实体间存在多条路径，一个风险步骤就会使整条推理路径出错，这进一步加剧了问题。为解决此问题，我们提出了模式感知的累积过程奖励模型，它基于推理前缀评估路径，并融合当前推理步骤与查询中解析的隐含目标之间的模式距离，提供累积奖励和未来奖励来指导路径探索。我们进一步将SCPRM集成到蒙特卡洛树搜索中，形成SCPRM-MCTS，以进行知识图谱上的多跳推理来回答问题。在医学和法律KGQA以及CWQ任务上，SCPRM-MCTS相较于强基线，平均将Hits@k指标提升了1.18%，展示了更准确且风险敏感的推理评估能力。

Binary Rewards and Reinforcement Learning: Fundamental Challenges（二值奖励与强化学习：基本挑战）

使用可验证奖励的强化学习（RLVR）已成为提升语言模型推理能力的标准方法，但经RLVR训练的模型常遭遇多样性崩溃：单样本准确率提升的同时，多样本覆盖率却下降，甚至低于基础模型。我们从二值奖励的性质出发，对这一现象进行了结构性解释。二值奖励为策略梯度方法带来了根本性退化：最大化期望奖励的分布集是无限的，且无显著元素。KL-控制通过极限β→0选择了滤波模型p*（即基于有效性条件的基础模型），以此解决退化问题。该选择通过非平凡的不对称性实现：倾斜分布p[β]在β→0时正向KL收敛于p*，但p*不能作为直接优化目标，因为对于任何全支撑策略q，KL(q||p*)都是无穷大。我们建立了超参数β与更可解释的目标有效性比率μ之间的显式公式。在模型错误指定（典型的实际场景）下，降低β的压力会驱使优化器集中于少量有效输出上的高度集中分布，并随β降低而坍缩至更少输出，而非向滤波模型靠拢。我们通过玩具自回归实验说明了这一机制，并讨论了直接针对p*的替代散度（如kruszewski_whatever_2026的实证探索）如何通过奖励覆盖p*的支撑而非集中在高有效性输出来避免此失败模式。

Double Rectified Linear Unit-based Modular Semantics for Quantitative Bipolar Argumentation Framework（基于双整流线性单元的定量双极论证框架模块化语义）

定量双极论证框架（QBAFs）提供了一种计算双极论证框架中论点可接受性的替代方法。每个论点被赋予一个初始强度，然后通过考虑其攻击者和支持者的影响，将其更新为最终强度。多年来，已有多种语义被提出用于计算QBAFs中的论点可接受性，但即使针对简单的无环案例，它们也常常产生分歧或反直觉的结果。我们为QBAFs引入了新颖的渐进语义来解决这些局限性，使生成的结果更符合直觉预期，同时满足文献中已建立的理性公设。此外，我们研究了其收敛行为，证明了该语义不仅适用于无环QBAFs，也适用于更广泛的有环框架类别。
