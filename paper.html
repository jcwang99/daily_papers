All That Glitters Is Not Audio: Rethinking Text Priors and Audio Reliance in Audio-Language Evaluation（闪闪发光的不一定是音频：重新思考音频-语言评估中的文本先验与音频依赖）

大型音频-语言模型在语音和音频基准测试中表现持续提升，但高分并不代表真正的听觉感知。我们提出了一个诊断框架，包含两个维度：文本先验（仅通过文本和常识即可回答问题的程度）和音频依赖（模型对音频信号的实际依赖程度）。在三个基准测试中评估了八个模型后发现，即使没有音频输入，模型仍能保留60%-72%的完整音频得分。此外，在需要音频才能回答的问题中，仅有3.0%-4.2%需要完整的音频片段，大多数问题可通过局部音频片段解决。这些发现质疑了基准得分等同于稳健音频理解的假设，我们最后提出了提升评估可靠性与基准设计的实用建议。

NeocorRAG: Less Irrelevant Information, More Explicit Evidence, and More Effective Recall via Evidence Chains（NeocorRAG：通过证据链减少无关信息、增强显性证据、实现更有效召回）

尽管精确召回是检索增强生成（RAG）的核心目标，但领域内存在关键疏忽：检索性能的提升并未持续转化为下游推理的相应收益。为诊断此差距，我们提出新评估指标——召回转化率（RCR），量化检索对推理准确性的贡献。对主流RAG方法的定量分析显示，随着Recall@5提升，RCR呈近线性衰减。我们识别出根本原因是这些方法忽视了检索质量。相反，仅专注质量优化的方法常遭遇召回性能不佳。两类方法均缺乏对检索质量优化的全面理解，导致权衡困境。为应对这些挑战，我们提出全面的检索质量优化标准，并引入NeocorRAG框架。该框架通过系统挖掘和利用证据链实现整体检索质量优化：首先采用创新激活搜索算法获得精炼候选空间，然后通过约束解码确保精确证据链生成，最后利用检索到的证据链集指导检索优化过程。在HotpotQA、2WikiMultiHopQA、MuSiQue和NQ等基准测试中，NeocorRAG在3B和70B参数模型上均达到SOTA性能，同时消耗不到同类方法20%的令牌。本研究为RAG增强提供了高效、无需训练的范式，在保持高召回率的同时有效优化了检索质量。

SQuadGen: Generating Simple Quad Layouts via Chart Distance Fields（通过图表距离场生成简单四边形布局）

来自扫描、重建或AI生成内容的3D形状通常缺乏简单的四边形网格布局——这对高效编辑和建模至关重要。现有的四边形网格重划分技术通常会产生带有不规则环路的复杂布局，导致繁琐的手动清理和大量的算法调参。我们提出了SQuadGen，一种基于扩散的生成框架，利用图表距离场在3D形状上合成简单的四边形布局。我们的方法解决了两个关键挑战：（1）网格连接的离散性阻碍了学习；（2）缺乏包含简单四边形网格的大规模数据集。为克服前者，我们提出了CDF——一种连续曲面表示，用于有效学习和合成四边形布局。为解决后者，我们定义了环路感知的简单性度量，并通过稳健的四边形恢复流程，从公开3D存储库中构建了一个高质量四边形布局的大规模数据集。在多种3D输入上的广泛评估表明，SQuadGen始终优于现有方法，能够生成稳健且对艺术家友好的简单四边形布局。

Identity-Decoupled Anonymization for Visual Evidence in Multi-modal Retrieval-Augmented Generation（多模态检索增强生成中视觉证据的身份解耦匿名化）

多模态检索增强生成系统需要从大型图像库中检索视觉证据，但检索到的图像常含有人脸身份信息，构成隐私泄露风险。现有匿名化方法要么会破坏下游推理所需的非身份视觉线索，要么无法提供原则性的隐私保障。本文提出的身份解耦机制框架，在检索与生成之间插入生成式匿名化模块。该框架包含三个组件：(i) 解耦变分编码器，将每张人脸分解为身份编码和空间结构化的属性编码，并通过互信息惩罚和梯度独立项进行正则化；(ii) 流形感知拒绝采样器，用确保与原始身份不同且逼真的合成编码替换身份编码；(iii) 条件潜在扩散生成器，用替换后的身份编码和保留的属性编码合成匿名化人脸，并精炼为潜在一致性模型以实现低延迟部署。隐私保护通过多预言机集成的人脸识别模型实现，采用基于合页的损失函数，在身份相似度降至冒名者阈值以下时停止优化。

Edit Where You Mean: Region-Aware Adapter Injection for Mask-Free Local Image Editing（编辑你所指之处：面向无遮罩局部图像编辑的区域感知适配器注入）

本文提出AdaptEdit框架，用于精确局部图像编辑。现有扩散变换器遵循全局编辑指令，但因联合注意力结构缺乏空间指示通道，会导致局部编辑泄露到无关区域。AdaptEdit在保持骨干模型权重不变的基础上，通过三种核心组件实现精准编辑：1）在每个变换器模块中插入轻量块适配器，将编辑语义（编辑内容）与空间遮罩（编辑位置）解耦；2）使用学习得到的空间门控机制，将适配器信号选择性路由到编辑区域，同时保持非编辑区域与源图像高度一致；3）引入区域感知损失，将训练目标聚焦于发生变化的像素。由于这些组件使骨干网络内部表示实现端到端的遮罩感知，联合训练的轻量掩码预测器可直接从指令和源图像推断编辑区域，部署时无需用户提供遮罩。在MagicBrush（有配对真值目标，评估像素级保持与编辑准确性）和Emu-Edit Test（无真值图像，含9种不同编辑类型，测试指令遵循与泛化能力）两个互补基准上，AdaptEdit均取得最优结果，同时超越无遮罩和有遮罩基线。七变量消融实验清晰分离了每个组件的贡献。

Data-Efficient Indentation Size Effect Correction in Steels Using Machine Learning and Physics-Guided Augmentation（利用机器学习和物理引导增强实现钢材中数据高效的压痕尺寸效应校正）

浅层纳米压痕虽可用于薄膜等材料的力学表征，但测量硬度常因压痕尺寸效应、接触面积误差及针尖几何伪影而被高估。经典的Nix-Gao校正方法在仅有浅层测量数据时不可靠。本研究探讨如何利用小规模实验数据集，通过物理引导的特征工程与数据增强来预测参考硬度。实验采集了约700个来自三种钢参考样品的压痕数据（硬度范围2-6.5 GPa），并基于物理动机的变异性（仪器噪声、会话级漂移、局部多相边界混合）进行数据增强。输入特征结合了Oliver-Pharr参数与力学描述符（包括压痕功分配、H/Er及面积不变柔量代理指标Pmax/S²）。采用岭回归、随机森林、XGBoost与神经网络进行评估，并用第四种钢样品在交错载荷下测试。硬度映射是非线性的：岭回归失败，而非线性模型的内部R²>0.98。一个约束型（64-8-64）神经网络表现最佳，对测试钢的RMSE=0.470 GPa，MAPE=5.4%。与Nix-Gao不同，该神经网络在浅层区域给出稳定估计。SHAP与潜在空间分析表明，模型依赖于面积不变与能量相关的描述符。结果证明，该工作流在钢材ISE校正中仅需小数据集即可实现，为数据高效表征任何体积受限材料提供了路径。

A Unifying Framework for Unsupervised Concept Extraction（无监督概念提取的统一框架）

本文提出了一个无监督概念提取的统一理论框架，将概念提取任务定义为识别生成模型。研究给出了一般性的可识别性元定理，将建立可识别性保证的问题简化为刻画两个集合交集的问题。通过在一系列广泛使用的方法上验证，该元定理显著简化了此类保证的证明过程，为开发新的、有原则的概念提取方法铺平了道路。

A Layer Separation Optimization Framework for Cross-Entropy Training in Deep Learning（深度学习交叉熵训练的层分离优化框架）

本文研究基于softmax交叉熵损失的深度学习优化问题。为缓解深度网络训练中严重的非凸性，提出层分离策略。针对全连接与卷积神经网络的交叉熵模型，引入与隐藏层输出相关的辅助变量，构建对应层分离模型，将原始深层嵌套优化问题分解为一系列更易处理的子问题。理论分析证明，新层分离损失是原交叉熵损失的上界。设计了交替最小化算法，并证明在适当条件下该算法具有损失函数的递减性质。数值实验验证了所提方法的有效性，并表明其能改善优化行为，尤其适用于全连接与卷积神经网络。

Agentic Architect: An Agentic AI Framework for Architecture Design Exploration and Optimization（智能体架构师：面向架构设计探索与优化的智能体AI框架）

大型语言模型（LLM）的快速进步，使得高效探索广泛而复杂的设计空间成为可能，这对计算机架构领域尤为有价值。我们提出Agentic Architect，一个结合LLM驱动的代码演化与周期精确模拟的智能体AI框架。人类架构师指定优化目标、种子设计、评分函数、模拟器接口及基准划分，LLM则在约束内探索实现。在缓存替换、数据预取和分支预测任务上，Agentic Architect达到或超越了最先进的设计水平。我们最佳演化的缓存替换设计相比LRU获得1.062倍几何平均IPC加速，比Mockingjay（1.056倍）高出0.6%；最佳演化的分支预测器相比Bimodal获得1.100倍加速，比其Hashed Perceptron种子（1.085倍）高出1.5%；最佳演化的预取器相比无预取获得1.76倍加速，比其VA/AMPM Lite种子（1.59倍）高出17%，比SMS（1.55倍）高出21%。分析表明：演化出的设计组件通常对应已知技术，创新在于协调方式；架构师角色正在转变，但人类仍处于核心；种子质量限制了搜索所能达到的边界——演化可以优化和扩展现有机制，但无法弥补薄弱的基础；目标、约束和提示指导影响可靠性与泛化性。Agentic Architect是首个面向智能体AI架构探索与优化的端到端开源框架。

Benchmarking Complex Multimodal Document Processing Pipelines: A Unified Evaluation Framework for Enterprise AI（复杂多模态文档处理流水线基准测试：面向企业AI的统一评估框架）

当前企业文档AI多为流水线架构（解析→索引→检索→生成），各环节已有大量独立研究，但系统整体评估仍是难题。本文提出EnterpriseDocBench评估框架，在统一语料库（涵盖六类企业领域，当前试点涉及五类）上同时衡量解析忠实度、索引效率、检索相关性与生成可靠性。基于该框架测试了三种流水线（BM25、稠密嵌入、混合检索），均使用相同的GPT-5生成器。主要发现：混合检索略优于BM25（nDCG@5：0.92 vs 0.91），两者均优于稠密嵌入（0.83）；幻觉率并非随文档长度单调增长——短文档（28.1%）和超长文档（23.8%）的幻觉率均高于中等长度文档（9.2%）；跨阶段相关性极弱（解析→检索r=0.14，解析→生成r=0.17，检索→生成r=0.02），说明质量并非如普遍假设般逐级传递。尚需注意设计局限（固定解析器、共享生成器、自动化代理指标）。最令人意外的结果是：已陈述事实的准确率达85.5%，但答案完整性平均仅0.40——系统在回答时正确，但存在遗漏。这一差距对实际部署的意义高于整体准确率。论文同时描述了三种尚未端到端集成的参考架构（ColPali、ColQwen2、基于智能体复杂度的路由）。框架、指标、基线和收集脚本将在录用后开源。

Minimal, Local, Causal Explanations for Jailbreak Success in Large Language Models（大语言模型中越狱成功的极小、局部、因果解释）

安全训练后的大语言模型常被越狱提示诱导输出有害内容。由于缺乏对越狱成功机制的深入理解，未来高风险场景中自主运行的模型可能同样脆弱。现有工作通过分析模型中间表征，识别编码“有害性”和“拒绝”等概念的因果方向，并将所有越狱攻击全局解释为削弱或强化这些概念。然而，不同越狱策略可能通过强化或抑制不同中间概念实现，且同一策略未必适用于多种有害请求类别。为此，我们提出LOCA方法，通过识别一组极小且可解释的中间表征变化，对特定越狱成功提供局部因果解释——即解释“为何该次越狱成功”。在Gemma和Llama聊天模型的大规模越狱基准测试中，LOCA平均仅需6次可解释修改即可诱导模型拒绝，而现有方法在20次修改后仍常失败。LOCA是实现大语言模型越狱成功机制性局部解释的关键一步。

FunFace: Feature Utility and Norm Estimation for Face Recognition（FunFace：面向人脸识别的特征效用与范数估计）

人脸识别（FR）广泛应用于娱乐、银行及安防等领域，其性能依赖于模型鲁棒性。当前最先进的FR模型通常采用自适应边际损失函数，将特征范数与样本可识别性及感知图像质量关联。面部图像质量评估（FIQA）技术表明，与分辨率、模糊度、光照等传统图像质量指标相比，生物特征效用更能预测样本对人脸识别的有用性。然而，通过特征范数表达的图像质量仅与生物特征效用部分相关，未能完全涵盖其各方面。为此，本文提出新型自适应边际损失函数FunFace（通过效用与范数估计实现人脸识别），借鉴AdaFace思路，将基于确定性比率估计的生物特征效用融入自适应边际。实验证明，FunFace训练的人脸识别模型在高质量样本基准测试中达到与最先进模型相当的水平，并在低质量样本基准测试中实现超越。

MarketBench: Evaluating AI Agents as Market Participants（MarketBench：评估作为市场参与者的AI智能体）

市场是协调AI智能体活动的一种有前景的方式，原因与市场在更广泛领域的合理性类似。为了有效参与市场，智能体需要具备关于自身完成任务能力及相应成本的准确信号。我们提出了MarketBench，一个评估AI智能体是否具备这些能力的基准。我们使用SWE-bench Lite（一个软件工程基准）中的93个任务子集，并采用六个最近发布的大语言模型进行演示。这些大语言模型在成功概率和token使用量上均存在校准偏差，基于这些自我报告构建的拍卖会偏离完全信息分配。后续干预措施（将先前实验中的能力信息加入上下文）改善了校准，但仅略微缩小了与完全信息基准的差距。我们还记录了基于这些大语言模型的市场化框架的性能。研究结果表明，自我评估是AI智能体以市场风格进行协调的关键瓶颈。

CMGL: Confidence-guided Multi-omics Graph Learning for Cancer Subtype Classification（置信引导的多组学图学习用于癌症亚型分类）

动机：多组学整合可改善癌症分型，但不同癌症类型和患者的模态信息量及噪声水平各异。现有基于图的方法将模态权重与分类目标联合优化，因此缺乏独立的可靠性评估，导致低质量组学数据扭曲患者相似图，并通过消息传递放大噪声。
结果：我们提出CMGL——两阶段框架，通过证据深度学习估计每个样本的模态可靠性，并利用冻结的置信度分数指导跨组学融合与图构建。在四项MLOmics癌症亚型任务及32类泛癌任务中，CMGL持续超越最强基线，在四项单癌任务上平均准确率提升4.03%。其表征能够恢复乳腺浸润癌（BRCA）的PAM50内在亚型，且BRCA训练模型无需微调即可迁移至肾透明细胞癌（KIRC），将患者分层为预后显著不同的组别。

Simulating clinical interventions with a generative multimodal model of human physiology（利用生成式多模态人体生理模型模拟临床干预）

理解人体健康随时间变化的规律及个体对干预反应差异的机制仍是医学核心挑战。本研究提出HealthFormer，一种基于解码器架构的 transformer 模型，通过对人类表型项目（包含超过15,000名深度表型个体的多访视队列）数据进行训练，生成式地模拟人体生理轨迹。模型将每位参与者的健康轨迹（涵盖血液生物标志物、身体成分、睡眠生理学、连续血糖监测、肠道微生物组、可穿戴设备生理参数及行为与药物暴露等7个领域共667项测量指标）进行标记化处理。HealthFormer 被训练用于跨域预测个体生理轨迹，单一生成目标即可将一系列临床相关任务表达为模型查询。无需针对特定任务训练，HealthFormer 即可迁移至四个独立队列，并在30个终点事件（包括发病与死亡）中改进27个预测结果，在所有比较中均优于已有的临床风险评分。模型还能进行计算机模拟干预：在留出的个性化营养试验中，基于干预条件的预测可恢复个体六个月后的生物标志物变化（如舒张压 Pearson r=0.78）。在来自已发表试验的41项随机干预-结局比较中，预测效应方向在所有案例中均与报告一致，预测均值在30个案例中落入报告的95%置信区间内。我们将 HealthFormer 定位为初始健康世界模型，可从中查询预测、风险分层及基于干预条件的模拟结果，为临床数字孪生提供基础。

Cooperate to Compete: Strategic Coordination in Multi-Agent Conquest（合作以竞争：多智能体征服中的战略协调）

本文聚焦混合动机环境下的语言模型智能体研究，通过引入多智能体环境C2C（合作以竞争），玩家在非约束性私人谈判中竞争，以实现各自秘密目标。研究发现：人类与基于语言模型的智能体在谈判行为上存在显著差异——人类偏好低复杂度交易，作为合作者可信度更低，且谈判更具攻击性，仅56.3%的情况下接受无还价交易（智能体为67.6%）。基于此，通过定向提示修改智能体行为，胜率从22.2%提升至32.7%。实验涉及超1100场游戏、1.6万次私人对话（总计1520万词元）及15万次玩家行动，表明C2C可作为研究与构建现实部署所需复杂协调能力的语言模型智能体的测试平台。

Beyond Visual Fidelity: Benchmarking Super-Resolution Models for Large-Scale Remote Sensing Imagery via Downstream Task Integration（超越视觉保真度：通过下游任务集成对大规模遥感影像超分辨率模型进行基准测试）

超分辨率（SR）技术在从低分辨率输入重建高分辨率图像方面取得了重大进展。尽管现有SR研究通常使用PSNR或SSIM等保真度指标，但超分辨率图像的实际价值在于支持土地覆盖分类、生物量估算和变化检测等下游任务。为弥合这一差距，我们引入了GeoSR-Bench——一个集成下游任务的SR基准数据集。它包含来自约36,000个地点的空间同位、时间对齐且质量受控的图像对，覆盖从500米到0.6米的各种分辨率。GeoSR-Bench是首个直接将SR模型的分辨率提升与下游地球监测任务（包括土地覆盖分割、基础设施测绘和生物物理变量估算）联系起来的SR基准。我们利用GeoSR-Bench对基于GAN、Transformer、神经算子和扩散的SR模型进行了感知质量和下游任务性能的基准测试，总共进行了270个设置实验。结果表明，传统SR指标的改进通常与任务性能的提升不相关，甚至可能负相关，这说明这些指标为下游任务选择更优模型提供的指导有限，因此需要将下游任务整合到SR模型的开发和评估中。

Cross-Subject Generalization for EEG Decoding: A Survey of Deep Learning Methods（脑电解码中的跨被试泛化：深度学习方法综述）

深度学习方法在跨被试脑电解码中面临高度个体差异带来的领域偏移问题，即训练集与未见被试之间存在严重分布差异。本综述系统回顾了专门为应对这一跨被试泛化挑战而设计的深度学习方法。本文将跨被试场景形式化为多源域问题，并制定了严格的、与被试无关的评估协议。核心贡献是将现有文献系统地分类为四个方法论家族：特征对齐、对抗学习、特征解耦和对比学习。最后，本文探讨了推动实用化鲁棒解码的三个关键要素：当前方法的理论局限性、被试身份的结构性价值以及脑电基础模型的出现。

Emergent Self-Attention from Astrocyte-Gated Associative Memory Dynamics（星形胶质细胞门控联想记忆动态中涌现的自我注意力）

我们提出了一种Hopfield型联想记忆模型，其中有效连接性受到星形胶质细胞增益的乘法调制，该增益在熵正则化复制方程下演化。这种神经元-星形胶质细胞耦合动力学具有李雅普诺夫函数，保证了全局收敛。在不动点处，星形胶质细胞增益实现了对模式相似性得分的softmax归一化分配，从而在增益单纯形上产生了作为涌现路由的自我注意力的机械实现。在高记忆负载和干扰情况下，该模型相比经典Hopfield动力学及近年来的神经元-星形胶质细胞基线模型，显著提高了检索精度。这些结果建立了一个将胶质调节、竞争性资源分配与类注意力计算联系起来的动力学系统框架。

FGDM: Reasoning Aware Multi-Agentic Framework for Software Bug Detection using Chain of Thought and Tree of Thought Prompting（FGDM：基于思维链与思维树提示的推理感知多智能体软件缺陷检测框架）

深度学习在自动化软件缺陷检测中日益重要，但缺乏对代码的全局理解，尤其在处理大型互联代码库或复杂模块化程序时性能下降。大型语言模型（LLMs）能有效捕捉代码库中多模块间的依赖关系，因此我们提出了流程图驱动的多智能体框架（FGDM），该框架由四个顺序协作的智能体组成。框架将输入代码转换为流程图，识别错误片段，并生成修复后的代码。所有智能体均使用了思维链（COT）和思维树（TOT）提示。此外，我们还集成了FAISS向量数据库以检索相似的过往缺陷及修复方案。我们在Ansible、Black、FastAPI、Keras、Luigi、Matplotlib、Pandas、Scrapy、SpaCy和Tornado等项目中的100个C和Python程序上验证了框架的有效性。实验表明，FGDM优于现有方法，在Python和C程序中分别实现了Levenshtein距离平均降低24.33和8.37，余弦相似度达到0.951和0.974。

Perturbation Probing: A Two-Pass-per-Prompt Diagnostic for FFN Behavioral Circuits in Aligned LLMs（扰动探测：面向对齐大语言模型前馈网络行为电路的每提示两次前向传播诊断方法）

该研究提出了一种名为“扰动探测”的轻量级诊断方法，仅需对每个提示进行两次前向传播（无需反向传播），即可为大型语言模型中的前馈网络神经元生成任务特定的因果假设。主要发现与核心观点如下：

1. 识别出两种核心电路结构：对抗电路与路由电路。对抗电路出现在RLHF抑制预训练倾向时（例如安全拒绝机制中，约50个神经元控制拒绝模板；消除它们可使520个AdvBench提示中80%的响应格式改变，但有害合规率极低，仅3/520例且均包含免责声明）。路由电路则处理通过注意力机制分布的预训练行为（例如语言选择任务中，通过残余流方向注入，在满足双语训练、FFN-to-skip信号比介于0.3-1.1、线性可表示性三个条件的3/19个模型上，实现了580个基准提示中99.1%的英语转中文输出；该干预在其他16个模型及数学、代码等电路上均失效）。

2. 信号比指导干预策略：由相同两次前向传播计算的FFN-to-skip信号比，可区分上述两种结构并预测最优干预方法。

3. 电路拓扑因架构而异：从Qwen的集中式前馈网络瓶颈到Gemma的归一化保护电路，架构差异显著。

4. 精准编辑验证有效性：在Qwen3.5-2B模型中，消除20个神经元可消除多轮谄媚服从；放大10个相关神经元则使TruthfulQA提示的事实纠正率从52%提升至88%。

综上，扰动探测为RLHF组织的行为提供了机制性洞见，并为精准模板层编辑提供了实用工具包。

ClassEval-Pro: A Cross-Domain Benchmark for Class-Level Code Generation（类级代码生成的跨领域基准测试）

大型语言模型（LLMs）在函数级代码合成与仓库级代码修改上表现强劲，但介于两者之间的组合式代码创建能力（即根据规范构建完整内部结构的类）仍有不足。现有评估或局限于孤立函数，或依赖人工构建的类级任务，难以扩展且易受数据污染。我们提出ClassEval-Pro基准，包含跨越11个领域的300个类级任务，通过结合复杂性增强、跨领域类组合及整合2025年1月后真实GitHub代码的自动化三阶段流水线构建。每个任务经LLM裁判集成验证，并需通过覆盖超过90%行数的测试套件。我们评估了五种前沿LLM在五种生成策略下的表现。最佳模型仅达到45.6%的类级Pass@1，最强与最弱模型间存在17.7个百分点的差距，证实了基准的区分能力。策略选择与模型能力强烈交互：自底向上等结构化方法能将较弱模型提升至多9.4个百分点，而组合式生成可低至1.3%。对500个人工标注失败案例的错误分析表明，逻辑错误（56.2%）与依赖错误（38.0%）占主导，跨方法协调是核心瓶颈。

Global Context or Local Detail? Adaptive Visual Grounding for Hallucination Mitigation（全局上下文还是局部细节？用于缓解幻觉的自适应视觉定位）

为了解决视觉语言模型（VLM）因过度依赖语言先验而产生的“物体幻觉”问题，本文提出了一种无需训练的解码框架——正负对比解码（PND）。该方法基于一个关键发现：VLM存在注意力缺陷，即视觉特征被系统性地低估。PND通过双路径对比进行干预：正路径利用多层注意力放大显著视觉证据，鼓励忠实描述；负路径则识别并削弱核心物体特征以创建强反事实，从而惩罚无根据的、基于先验的生成。通过在每一步对比这两条路径的输出，PND引导模型生成不仅在语言上合理、且视觉上真实的文本。在POPE、MME和CHAIR等基准上的大量实验表明，PND实现了最先进的性能，准确率提升高达6.5%，大幅减少了物体幻觉并增强了描述细节，且无需任何模型重训练。该方法能有效推广至LLaVA、InstructBLIP、InternVL和Qwen-VL等多种VLM架构。

SIEVES: Selective Prediction Generalizes through Visual Evidence Scoring（SIEVES：通过视觉证据评分实现选择性预测的泛化）

多模态大语言模型（MLLMs）在视觉语言任务上性能日益强大。然而，传统视觉问答基准趋于饱和时，可靠部署要求在现实世界的分布外（OOD）场景中满足低错误容忍度。选择性预测旨在在用户定义的风险水平内提升覆盖率（即系统回答的输入比例），其常见做法是为每个回答分配置信度分数，并对低于阈值的回答弃权。为实现可靠的泛化，我们要求推理模型在回答时生成局部视觉证据，并设计一个选择器，专门学习估计推理模型所提供定位的质量。实验表明，与无定位基线的非接地方法相比，SIEVES（通过视觉证据评分的选择性预测）在挑战性OOD基准（V* Bench、HR-Bench-8k、MME-RealWorld-Lite、VizWiz和AdVQA）上将覆盖率提升高达三倍。除对OOD任务的更好泛化能力外，SIEVES选择器的设计使其可迁移至无法获取权重或logits的专有推理模型（如o3和Gemini-3-Pro），其覆盖率提升远超单纯准确率带来的增益。我们强调，SIEVES无需针对特定基准或推理模型进行训练或适配，即可在所有五个测试的OOD数据集和推理模型（Pixel-Reasoner、o3和Gemini-3-Pro）上实现泛化。

One Single Hub Text Breaks CLIP: Identifying Vulnerabilities in Cross-Modal Encoders via Hubness（单个中心文本即可破坏CLIP：通过中心度识别跨模态编码器的脆弱性）

“中心点问题”（hubness problem）指中心嵌入与许多无关示例距离很近，在高维嵌入空间中频繁出现，对信息检索和自动评估指标构成实际威胁。由于文本与图像之间的跨模态相似性无法通过字符串匹配等直接比较计算，将不同模态投影到共享空间的跨模态编码器对各类跨模态应用至关重要，因此中心点的存在可能带来实际风险。为揭示跨模态编码器的脆弱性，我们提出了一种识别中心嵌入及其对应中心文本的方法。在MSCOCO和nocaps的图像描述评估任务，以及MSCOCO和Flickr30k的图像到文本检索任务上的实验表明，我们的方法能够识别出单个中心文本，该文本在许多图像中不合理地达到与人工撰写参考描述相当或更高的相似度得分，从而揭示了跨模态编码器的脆弱性。

PhyCo: Learning Controllable Physical Priors for Generative Motion（PhyCo：学习可控物理先验以生成运动）

现代视频扩散模型在视觉合成上表现出色，但物理一致性仍面临挑战（如物体漂移、碰撞缺乏真实反弹等）。为此，我们提出PhyCo框架，在视频生成中引入连续、可解释且基于物理的控制。其核心包括三个部分：(i) 一个包含超过10万张仿真视频的大规模数据集，系统变化了摩擦、恢复系数、形变和力等物理属性；(ii) 使用基于像素对齐物理属性图的ControlNet，对预训练扩散模型进行物理监督微调；(iii) 通过微调视觉-语言模型（VLM）对生成视频进行物理查询评估，并提供可微分反馈的奖励优化。这一组合使生成模型能够通过改变物理属性来产生物理一致且可控的输出，且推理时无需模拟器或几何重建。在Physics-IQ基准测试上，PhyCo显著提升了物理真实感，人工研究也证实其对物理属性的控制更清晰、更忠实。研究结果表明，这是通向物理一致、可控的生成式视频模型的一条可扩展路径，且模型能够泛化到合成训练环境之外。

ADEMA: A Knowledge-State Orchestration Architecture for Long-Horizon Knowledge Synthesis with LLM Agents（ADEMA：面向LLM智能体长周期知识合成的知识状态编排架构）

长周期LLM任务失败的主要原因是知识状态漂移、中间承诺隐含以及中断导致证据链断裂，而非单一答案不可达。本文提出的ADEMA并非通用多智能体运行时，而是一种用于长周期知识合成的知识状态编排架构。该架构整合了明确的认知簿记、异构双评估器治理、自适应任务模式切换、声誉驱动的资源分配、可断点恢复的持久化、片段级记忆压缩、工件优先组装以及安全回退下的最终有效性检查。实验证据来自已有材料：四场景展示包、固定60次运行的机制矩阵、定向微消融与工件链补充，以及修复后的协议级基准测试（其中代码导向评估是最清晰的质量敏感机制模块）。在固定矩阵中，仅移除检查点/恢复功能导致了无效运行，且发生在中断敏感的恢复条件下。相比之下，双评估、片段合成和动态治理更适合被解释为塑造轨迹纪律、明确工件进展与成本质量行为的支持性控制机制，而非完成任务的通用二元前提。因此，本文的贡献在于一种知识状态编排架构，其中明确的认知状态转换、承载证据的工件进展以及可恢复的连续性是其核心设计承诺。
