关于大语言模型能否检测和报告自身内部状态的问题，一些研究给出了肯定答案。然而，基于人类元认知研究的经验，我们认为这一结论可能为时过早：要令人信服，必须区分真正的内省与基于表面线索的模式匹配。此外，仅凭行为证据本身不足以确立强有力的内省主张。我们重新审视了两个近期提出的评估范式。第一范式中，模型需检测其内部状态是否被篡改，但结果发现它们无法可靠地区分对其内部状态的干预与对输入的操纵，这表明原始研究中的成功反映的是模型检测异常的一般能力，而非特定于内部状态的干预。第二范式中，模型被要求预测源自自身隐藏状态的标签。我们发现，仅能访问输入的分类器取得了与模型自身上下文内预测相当的性能，这表明原始结果并未确凿证明模型对其内部表征具有特权访问。我们还引入了一个重新标记的控制设置，在此任务中模型无法依赖语义，而必须依赖内部表征，结果其表现接近随机水平。综上所述，这些结果表明现有证据不足以证明大语言模型具备元认知监控能力。

现有关于AI编程代理失败的分析主要依赖基准测试轨迹，忽略了开发者的实际错位体验。本研究对来自1639个仓库的20574次编程代理会话进行了观察性分析，将错位操作化为开发者反驳所暴露的故障，并从形式、原因、代价和解决方案四个维度进行标注。研究识别出七种常见的错位形式，涵盖代理如何阅读项目、理解开发者意图、遵循规则、约束行为、实现与执行代码以及报告进度。90.50%的错位事件带来努力和信任成本，而非不可逆的系统损坏，但91.49%的显性解决方案仍需用户明确纠正。错位模式在IDE和CLI场景中有所不同，在相邻会话中持续存在，并随时间变化：虽然总体发生率下降，但违反约束和不准确自我报告的比例却在上升。研究结果为设计训练、评估和界面提供了依据，以保持编程代理与真实开发者工作流程的一致。

Focal Reward: Balanced Reinforcement Learning under Rubric-Based Rewards（焦点奖励：基于评分标准的平衡强化学习）

大语言模型（LLMs）的开放式生成通常需要多维评分标准来充分评估质量并指导强化学习的改进。然而，该训练范式中存在一个关键困境：不同评分维度上的奖励极化不平衡。在此瓶颈下，即便LLMs在训练后获得较高平均奖励，仍可能在特定维度存在严重缺陷，直接导致用户体验恶化。为解决此问题，我们提出Focal Reward，这是一种在基于评分标准的奖励下自动平衡强化学习训练的新目标。具体而言，我们首先利用逆向奖励投影机制估计每个评分标准的饱和程度，以此作为校准奖励方向的基础；随后，通过为每个标准自动设定重新加权系数来构建最终目标，实现细粒度平衡。涵盖三种模型规模和六个基准的广泛实验表明，在所有18个模型-基准对比中，Focal Reward方法均优于最强的静态聚合基线。滚动生成、机制分析和消融实验进一步证明，这些增益源于在线、饱和感知的奖励再分配，且该分配指向仍有改进空间的评分标准。

Adaptive Reservoir Computing for Multi-Scenario Chaotic System Forecasting（面向多场景混沌系统预测的自适应储层计算）

本文提出了一种自适应储层计算框架，用于CTF-4-Science Lorenz基准测试。该测试包含跨五个不同场景（基准预测、噪声信号重建、噪声下预测、少样本学习及参数泛化）的十二项任务。核心贡献包括：（1）精确的储层状态同步，消除了短时预测中的预热近似误差；（2）直方图引导的候选选择，直接优化长时间遍历评估指标；（3）多种子储层搜索，应对训练数据极度稀缺的少样本场景；（4）序列多序列训练，解决参数泛化任务中的状态分布失配问题。该框架在公开基准排行榜上取得74.91分，证明了经过精心适配的储层计算是一种具有竞争力且计算高效的方法，适用于多样化的混沌系统建模挑战。

Native Audio-Visual Alignment for Generation（用于生成的原生音视频对齐）

本文针对联合音视频生成任务，提出NAVA（原生音视频对齐框架）。现有开源方法存在局限：双塔设计削弱了细粒度音视频协同进化，而统一三模态设计则混淆了语义条件与低级同步。NAVA通过“对齐-融合”MMDiT架构，先在专用交互空间建立音视频对应关系，再使用外部上下文（如文本）调节联合去噪过程。此外，引入上下文音色条件机制，将参考音色线索与对应语音片段关联，实现可控语音音色。实验表明，仅用6.3B参数，NAVA在视频质量、音视频同步、音频质量及参考音色可控性上均表现优异。

Smoothed Score Queries and the Complexity of Sampling（平滑得分查询与采样的复杂度）

本文研究了使用梯度信息从高维高斯分布中采样的查询复杂度。在标准预言机模型中，确切梯度仅暴露与精度矩阵的矩阵-向量乘积，导致多项式近似障碍及对条件数κ的√κ依赖。研究表明，当允许采样器查询“平滑得分”（即高斯卷积密度的对数梯度）时，该障碍消失。对于精度矩阵为Λ的高斯目标，在噪声水平τ下的平滑得分查询可访问解析算子(Λ+τ⁻¹I)⁻¹。通过将几何级数噪声水平与sinc-正交有理近似相结合，对于总变分误差δ_TV，所提采样器仅需q=O((logκ+log(e√d/δ_TV)) log(e√d/δ_TV))次平滑得分查询，将条件数依赖从√κ改进为对数依赖。此外，研究还涉及有限位梯度预言机：通过坐标量化变换后的平滑得分响应并辅以最终抖动步骤，所得采样方案的总梯度信息通信量在κ上为多对数级；特别地，在固定维度与精度下，比特复杂度为O(log²κ)。为补充上界，本文引入了用于采样下界的信道综合（逆向香农）逆推技术，将总变分模拟保证转换为通信需求，并得到所需梯度信息量的Ω(logκ)下界。综上，这些结果证实平滑得分是用于采样的可证明信息更丰富的预言机，并为其有限比特复杂度提供了近乎匹配的上下界。

MRMMIA: Membership Inference Attacks on Memory in Chat Agents（针对聊天代理记忆中的成员推理攻击）

成员推理攻击（MIAs）用于测试目标数据是否属于系统的私有数据，已成为衡量机器学习系统隐私泄露的标准工具。以往研究主要关注训练语料库或检索数据库，但针对代理记忆的MIA较少受到关注，尽管此类记忆可能包含敏感的用户-代理交互、检索到的事实及用户偏好。为此，本文聚焦于聊天代理记忆MIA，攻击者推断候选记忆单元是否属于代理记忆存储。我们提出了多召回记忆MIA（MRMMIA），一种统一攻击方法，利用对代理的多次召回探针，在黑盒、灰盒和白盒设置中提取成员信号。实验表明，MRMMIA始终优于基线方法。我们的结果揭示了代理中的隐私风险，并为聊天代理记忆系统中的成员泄露提供了初步评估框架。

SilentRetrieval: Hijacking Retrieval-Augmented Generation via Semantically-Preserving Adversarial Data Poisoning（无声检索：通过语义保留的对抗性数据投毒劫持检索增强生成）

检索增强生成（RAG）可减轻大语言模型幻觉，但引入了语料库完整性的关键漏洞。本文提出SilentRetrieval，一种两阶段数据投毒攻击，通过对抗性构造的流畅文档劫持RAG系统。第一阶段采用协调波束搜索，一种多词元联合优化方法，配合流畅性-相似度目标，在保证投毒宿主文档可检索性的同时限制困惑度。第二阶段利用上下文自适应触发器生成，一种由冻结大语言模型驱动的轻量触发器融合步骤，将操纵触发器集成到文档内容中。在基于合成目标答案的每查询单投毒文档评估下，SilentRetrieval在Natural Questions和MS MARCO上分别达到84.6%/81.3%的HR@10和57.5%/54.8%的ASR-LLM，同时保持接近良性的困惑度。跨模型评估显示，在固定触发器生成器下对四个目标大语言模型均有显著有效性；对包括ColBERT和商业嵌入模型在内的未知检索器的迁移测试，在相同注入语料库协议下平均HR@10达64.7%。在采样版维基百科规模评估中，SilentRetrieval在0.016%投毒比例下保持74.2%的HR@10。结合检索侧与生成侧防御可大幅降低攻击成功率，但会带来延迟权衡。人工评估显示，其标记率显著低于不流畅基线，但在当前样本量下数值上仍比良性内容更可疑。

Slogans or Stance? A Label-Light Diagnostic for Entrepreneurial-Discourse Measurement on Chinese SOE Speeches（口号还是立场？中国国企演讲中创业话语测量的轻标签诊断方法）

本研究聚焦于企业演讲中“创业精神”等构念的测量方法，而非提出新的提取模型。基于80篇央企领导演讲稿语料，利用24对同公司不同演讲者和5对同公司同演讲者的自然实验，检验各方法每文档指标是否随领导者身份变化（公司不变）。结果显示：LDA失败（Cohen d=0.20）；字典评分器d=0.81，中文句子编码器在文档向量距离约10^-3量级上d=0.65；零样本9B开源LLM（Qwen3.5:9b）将配对对比d提升至1.09（精确置换p1=0.034）。据此，我们下调三项主张：黄金F1衡量的是与LLM自身提示规则的一致性，而非外部构念恢复；文档级风格残差化将LLM的d降至0.43（p1=0.22），约半数效应源于领导者个人语风；置信度加权校准以方差换取Delta，自动挖掘的口号词典在消融实验中几乎无效。我们开放了2,190段评分语料、170段落试点、口号词典、两族LLM评分及评估工具。

Provably Communication-Efficient and Privacy-Preserving Federated Graph Neural Networks（可证明的通信高效且隐私保护的联邦图神经网络）

图神经网络（GNN）在关系数据上表现优异，但真实世界的图常分布在多个组织间，受隐私和政策限制无法共享原始数据。现有联邦GNN方法要么忽略跨客户端链接导致精度下降，要么需要频繁交换嵌入向量，造成巨大的通信和隐私成本。我们提出CE-FedGNN——一种通信高效且隐私保护的联邦GNN框架，用于学习此类耦合图。该方法不共享原始数据或每轮嵌入，而是通过不频繁交换聚合后的节点表示来工作。为处理跨客户端依赖及数据陈旧问题，我们引入移动平均估计器，持续追踪节点表示并支持其在多轮训练中稳定复用。为给发布的表示提供正式隐私保证，我们采用度量差分隐私（metric-DP）框架，该框架基于学习到的嵌入空间中的距离度量隐私，而非最坏情况下的输入扰动，从而在标准差分隐私过于保守的噪声水平下仍能提供有意义的保障。我们证明了模型以$O(1/\sqrt{T})$的速率收敛到驻点，通信复杂度为$O(T^{3/4})$。此外，在公共队列威胁模型下，通过Rényi差分隐私组合推导出$(\varepsilon,\delta)$-度量差分隐私保证。在合成银行间反洗钱基准和引文网络上的实验表明，CE-FedGNN在显著降低通信开销的同时保持了强性能，并在隐私保护噪声下具有鲁棒性。

Evi-Steer: Learning to Steer Biomedical Vision-Language Models through Efficient and Generalizable Evidential Tuning（通过高效且可泛化的证据调优控制生物医学视觉语言模型）

针对视觉语言基础模型的参数高效适配对于精确理解生物医学图像至关重要，但现有方法具有确定性，且在领域偏移或图像-文本对齐模糊时表现不佳。这一问题在临床中尤为关键，因为模型需要在低数据量和领域偏移下保持鲁棒性。本文提出Evi-Steer，一种面向BiomedCLIP的证据性跨模态低维控制框架，通过仅更新0.11%的总模型参数实现具有不确定性感知的参数高效微调。该方法在视觉和文本编码器中进行轻量级低维令牌更新，同时估计认知不确定性；这些不确定性估计用于更新门控残差，使模型在证据不足时保守调整。此外，基于Dempster-Shafer理论引入跨模态置信度融合，使视觉适配以文本置信度为条件，抑制冲突或不确定的跨模态更新。在涵盖8个器官和8种成像模态的15个生物医学影像数据集上进行了少样本学习和领域泛化评估。Evi-Steer在少样本学习和领域偏移场景下持续优于现有最先进方法，为在真实临床环境中部署视觉语言模型提供了实用且稳健的路径。代码已开源。

Conditional KRR: Injecting Unpenalized Features into Kernel Methods with Applications to Kernel Thresholding（条件核岭回归：将无惩罚特征注入核方法及其在核阈值化中的应用）

本文研究了条件核岭回归（Conditional KRR）的统计性质。该方法基于条件正定核，其核心在于：将估计的回归函数按其原生空间范数的平方进行惩罚。该方法可视为经典线性回归（特征由函数类$\mathcal{F}$指定）与对目标变量残差部分应用标准KRR的组合。主要理论结果表明，该方法的行为可简化为使用另一个固定核（称为残差核）的标准KRR，代价是期望测试风险中多出一个由$\mathcal{O}(1/\sqrt{N})$界定的附加项（其中$N$为样本量，隐藏常数取决于$\mathcal{F}$和输入分布）。这一简化使我们能够分析两种情境：一是$K$为正定核且$\mathcal{F}$由$K$的Mercer分解中前$k$个主特征函数构成；二是$\mathcal{F}$由$K$的随机特征表示中的$k$个随机特征构成。理论分析与实验均证实，当回归函数的$\mathcal{F}$分量比残差部分更显著时，条件KRR在这些情境下优于标准KRR。

Entity-Collision: A Stratified Protocol for Attributing Retrieval Lift in Agent Memory（实体碰撞：用于归因智能体记忆检索提升的分层协议）

现有端到端智能体记忆基准仅报告单次hit@k指标，混淆了词汇泄漏（未控制的查询/黄金/干扰实体重叠）与标签混合（偏好、服务、工具等特征的平均化）。本文提出实体碰撞协议——一种与系统无关的方法，通过构造使每个干扰项与答案共享实体token来固定BM25基线下限，并按判别器标签对查询进行分层，从而使任何超出BM25的提升均可归因于嵌入器。在开源智能体记忆测试平台上（5个标签×3个嵌入器×5种碰撞程度，成对自助法95%置信区间），该协议揭示出双轴模式：256维哈希三元组仅在深度碰撞的闭词汇标签上有效；MiniLM-384在两个轴上均占优；2.7倍参数的BGE-large并未全面超越MiniLM——它在意图型查询上胜出，但在词汇型查询上失败。编码器容量并非唯一约束。合成意图标签零假设在LongMemEval（n=500）上复现为单会话偏好召回悬崖。LoCoMo上的自适应向量权重路由为测量上的零假设：存在11.7个百分点的oracle提升空间，但测试信号均未能恢复。所有26张结果表和37个重现脚本均经过版本控制并由公共注册表验证；该协议在确定性控制的记忆测试平台（事件溯源决策日志、DAG状态机模式生命周期）上执行，因此每个报告的置信区间均可从数据摄入流中逐字节重现。

RW-TTT: Batched Serving for Request-Owned Test-Time Training State（请求自有测试时训练状态下的批量服务）

测试时训练（TTT）通过在生成过程中读取和更新请求自有状态（如快速权重、低秩增量或流式学习器状态）来适配大语言模型（LLM）。这打破了假设共享静态权重的批量LLM服务模式：串行执行虽正确但缓慢，而朴素批处理可能导致请求状态损坏。我们将此问题形式化为读写TTT服务，并提出RW-TTT：该方法为每个解码步骤标记其拥有者、版本及读/写效果，仅对兼容阶段进行批处理，并仅向拥有者提交更新。在单GPU上运行八个快速权重InPlace-TTT流时，RW-TTT达到274.61聚合tok/s，相较于串行服务加速9.31倍，在相同内存预算下比每流副本加速3.44倍。该方法在长上下文基准RULER上保持了性能，并通过了拥有者/版本校验。
