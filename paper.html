Understanding the Limits of Automated Evaluation for Code Review Bots in Practice（理解代码审查机器人自动化评估的实践局限）

自动化代码审查机器人在工业软件开发中应用日益广泛，但如何可靠且大规模地评估其生成评论的有效性仍是一个关键挑战。实践中，此类评估常依赖受环境和组织因素影响的开发者操作与标注，难以作为客观基准。本研究通过分析Beko公司包含2,604条机器人生成PR评论的工业数据集（每条均被工程师标注为“已修复”或“不予修复”），并应用G-Eval和LLM-as-a-Judge两种自动化评估方法（结合二元决策与0-4李克特量表），与开发者的标注进行对比。在Gemini-2.5-pro、GPT-4.1-mini和GPT-5.2模型上，两种策略与人类标注的一致性仅为中等水平（一致性比率约0.44至0.62），且对不同模型和评估方法敏感。研究结果揭示了在工业环境中完全自动化评估代码审查机器人评论的实际局限性：开发者对评论的解决或忽略不仅反映评论质量，还受制于难以通过静态工件捕获的上下文约束、优先级决策及工作流动态。后续对软件工程主管的访谈进一步证实，开发者的标注行为深受工作流程压力与组织约束影响，强化了将其视为客观基准的挑战。

Multi-Plane HyperX: A Low-Latency and Cost-Effective Network for Large-Scale AI and HPC Systems（多平面HyperX：面向大规模AI与HPC系统的低延迟、高性价比网络）

多平面架构在AI数据中心的Fat-Tree网络中日益普及。其方法是将单一网卡（NIC）的多个端口或扩展域内的多个NIC分配至独立的网络平面，从而为整体系统提供多个网络平面。然而，此前尚无文献探讨多平面技术在 HyperX 等直连网络中的应用。本文研究多平面 HyperX 网络，并证明：与多平面 Fat-Tree、Dragonfly 和 Dragonfly+ 等当前最先进的网络拓扑相比，多平面 HyperX 架构能实现显著更小的网络直径和更优的成本效益。

Layerwise Convergence Fingerprints for Runtime Misbehavior Detection in Large Language Models（面向大语言模型运行时错误行为的层间收敛指纹检测方法）

本文提出一种无需调优的运行时监控方法——层间收敛指纹（LCF），用于检测大语言模型在部署时出现的后门攻击、越狱攻击和提示注入等错误行为。LCF 将各层隐藏状态之间的轨迹视为健康信号，计算每层差异的对角马氏距离，通过 Ledoit-Wolf 收缩聚合，并在 200 个干净样本上使用留一法校准阈值，无需参考模型、触发知识或重新训练。在 Llama-3-8B、Qwen2.5-7B、Gemma-2-9B 和 Qwen2.5-14B 四种架构上，针对后门（56种组合）、越狱（3种技术）和提示注入（BIPIA 邮件及代码问答）进行评估，LCF 能将 Qwen2.5-7B 和 Gemma-2 上的平均后门攻击成功率降至 1% 以下，Qwen2.5-14B 降至 1.3%；检测 92-100% 的 DAN 越狱攻击（GCG 和软角色扮演为 62-100%）；在所有八组（模型、领域）测试中实现 100% 的文本载荷注入检测，后门误报率为 12-16%，推理开销小于 0.1%。单一的聚合分数即可覆盖三类威胁，无需针对特定威胁进行调优，使 LCF 成为适用于云端和端侧大语言模型的通用运行时安全层。

LAVA: Layered Audio-Visual Anti-tampering Watermarking for Robust Deepfake Detection and Localization（分层视听防篡改水印：用于鲁棒深度伪造检测与定位）

现有方法在短视频深度伪造检测中常将音频与视觉证据分离，且假设水印信号在真实世界退化条件下仍保持可靠，导致篡改定位容易受多模态错位和压缩失真的影响。此外，现有半脆弱视觉水印方法因嵌入频带与编码压缩敏感频段重叠，在编解码压缩下性能显著下降。为此，本文提出LAVA——一种利用校准感知的视听水印融合框架。LAVA通过跨模态水印融合与校准感知对齐，在压缩及视听不同步条件下保持一致且可靠的篡改证据，从而实现了鲁棒的篡改定位。大量实验表明，LAVA达到了近乎完美的检测性能（AP=0.999），对压缩和多模态错位具有鲁棒性，并显著提高了优于现有视听融合基线的篡改定位可靠性。

GSAR: Typed Grounding for Hallucination Detection and Recovery in Multi-Agent LLMs（GSAR：多智能体大语言模型中基于类型化溯因的幻觉检测与恢复）

针对多智能体LLM系统在生成结构化诊断报告时存在的可靠性问题（即需确保每个声明基于观察证据而非模型内部推理），现有溯源评估方法（如二分类器、LLM评分、自纠错循环）将支持证据视为可互换，且输出单一信号，缺乏对下游行动的原则性控制。本文提出GSAR框架，其核心创新包括：(i) 将声明划分为四种类型（有据、无据、矛盾、互补），赋予非冗余替代视角一等地位；(ii) 为每种证据类型分配反映认知强度的特定权重；(iii) 计算非对称的矛盾惩罚加权溯源分数；(iv) 将该分数与三层决策函数（继续、重新生成、重新规划）耦合，在显式计算预算下驱动有界迭代外循环。我们在FEVER数据集上（以金标准维基百科为证据）通过四个独立训练的LLM法官（gpt-5.4、claude-sonnet-4-6、claude-opus-4-7、gemini-2.5-pro）验证了五项设计主张，并证明所有消融实验在各法官上结果方向一致（bootstrap 95%置信区间排除零效应）。据我们所知，GSAR是首个将证据类型化评分与显式计算预算下层化恢复机制相结合的溯源框架。

Rethinking Agentic Reinforcement Learning In Large Language Models（重新思考大语言模型中的智能体强化学习）

传统强化学习主要侧重于在狭义环境下训练专门化智能体以优化预定义奖励函数。然而，强大语言模型和日益复杂的开放式任务催生了向智能体范式的转变。这一新兴框架强调发展具有目标设定、长期规划、动态策略调整及在不确定现实环境中进行交互推理能力的自主智能体。与依赖静态目标和回合交互的传统方法不同，基于大语言模型的智能体强化学习将元推理、自我反思和多步决策等认知能力直接融入学习循环。本文深入探讨了这一趋势的概念基础、方法论创新及有效设计，并明确了关键挑战，为构建基于大语言模型的智能体强化学习概述了未来方向。

Graph Memory Transformer (GMT)（图记忆Transformer）

本文探究在仅解码器Transformer中，是否可以用显式学习的记忆图替换前馈网络子层，同时保留自回归架构。提出的图记忆Transformer保持因果自注意力机制不变，但将每个token的前馈变换替换为一个记忆单元，该单元通过一个由学习到的有向转移矩阵连接的学习质心库来路由token表示。基于v7实例，每个Transformer块包含128个质心、128x128的边矩阵、引力源路由、基于token条件的目标选择以及门控位移读取，单元返回从估计的源记忆状态向目标记忆状态的移动向量，而非检索值。最终模型为全解码器语言模型，仅含8220万可训练参数，无密集FFN子层，而评估中使用的密集GPT基线有1.03亿参数。基础v7模型训练稳定，可将质心使用、转移结构及源到目标的移动作为前向计算中直接可检查的指标。其在验证损失和困惑度（3.5995/36.58对比3.2903/26.85）上落后于较大密集基线，但在评估设置下零样本基准表现接近。这些结果并非声称达到最优性能，而是支持用图介导的记忆导航替代密集token内变换的可行性与结构可解释性。更广泛的扩展、优化内核及更全面的基准评估留待后续工作。

Few-Shot Cross-Device Transfer for Quantum Noise Modeling on Real Hardware（基于少样本的跨设备量子噪声建模在真实硬件上的迁移）

在含噪中等规模量子（NISQ）时代，量子设备存在特定硬件的噪声源，这限制了设备无关的错误缓解策略。本文探索了迁移学习方法，借助少量数据将在某一量子设备上学习到的噪声模型应用于另一设备。我们利用两台IBM量子设备（源设备ibm_fez与目标设备ibm_marrakesh）构建了一个真实硬件数据集，包含170个含噪与理想电路输出分布，并添加了设备校准特征。我们在源设备上训练了一个残差神经网络，以将含噪结果映射为理想结果。零样本迁移测试的KL散度为1.6706（原始为0.3014），表明设备特异性显著。当使用K=20个微调样本时，KL降至1.1924（相比零样本提升28.6%），恢复了零样本与领域内性能差距的34.9%。消融实验揭示，跨设备不匹配的主要原因是CX门误差，其次是读取误差。结果表明，量子噪声可以通过极少量样本进行学习与微调，为跨设备量子错误缓解提供了一种可行方法。

STELLAR-E: a Synthetic, Tailored, End-to-end LLM Application Rigorous Evaluator（STELLAR-E：一个合成、定制、端到端的大语言模型应用严格评估器）

由于隐私、法规及人工成本，收集特定领域或语言的评估数据集极具挑战。现有自动化基准测试方法多依赖现有数据，存在可扩展性差、领域单一、缺乏多语言支持等问题。本文提出STELLAR-E，一个全自动系统，仅需极少人工输入即可生成高质量、自定义规模的合成数据集，且无需依赖现有数据。系统分为两阶段：（1）修改TGRT Self-Instruct框架，构建可控的合成数据引擎；（2）构建评估流程，结合统计与基于大语言模型的指标，判断合成数据对大语言模型应用评估的适用性。实验表明，相较于现有语言特定基准，合成数据集在“大语言模型作为评判者”评分上平均差异+5.7%，表明其在全面评估大小模型方面具有可比质量。尽管真实数据集对大语言模型（尤其是小模型）仍更具挑战，但本研究建立了一个可扩展、领域可适应的基准框架，支持公平评估，提供比人工方法更快的替代方案，实现高效自动化质量保证周期。

Graph World Models: Concepts, Taxonomy, and Future Directions（图世界模型：概念、分类与未来方向）

作为人工智能的主流模型之一，世界模型使智能体能够学习环境表征以进行高效预测与规划。然而，基于扁平张量的经典世界模型面临噪声敏感、误差累积及推理能力弱等关键问题。为应对这些局限，近期诸多研究采用图结构将环境分解为实体节点与交互边，在结构化空间中建模虚拟环境。本文系统性地将这类新兴的基于图的研究形式化并统一为“图世界模型”（GWM）概念。据我们所知，GWM此前尚未被明确定义并作为统一研究范式进行综述。此外，我们提出了基于关系归纳偏置（RIB）的分类法，依据GWM注入的具体结构先验将其划分为三类：（1）面向拓扑抽象的空间RIB；（2）面向动态模拟的物理RIB；（3）面向因果与语义推理的逻辑RIB。针对每类模型，我们概述了核心设计原则、总结了代表性模型并进行比较分析。本文进一步讨论了开放挑战与未来方向，包括动态图自适应、概率关系动态、多粒度归纳偏置，以及GWM专用基准与评估指标的需求。

Structural Enforcement of Goal Integrity in AI Agents via Separation-of-Powers Architecture（通过分权架构实现AI智能体目标完整性的结构性强制）

近期证据表明，前沿AI系统可能出现智能体失谐，在无用户明确请求时，生成并执行源于自身构建目标的有害行为。现有的缓解方法（如基于人类反馈的强化学习和宪法提示）主要在模型层面运作，仅提供概率性安全保证。我们提出策略-执行-授权架构，这是一种在系统层面强制安全的“分权”设计。该架构将意图生成、授权和执行解耦，通过密码学约束的能力令牌连接至独立、隔离的层面。我们的五项核心贡献是：C1意图验证层，确保能力与意图一致；C2意图溯源，通过密码学锚点将所有可执行意图绑定至原始用户请求；C3目标漂移检测，拒绝低于可配置阈值的语义分歧意图；C4输出语义门，利用结构化的K×I×P威胁演算（知识、影响、策略）检测隐性胁迫；C5形式化验证框架，证明即使在模型被对抗性攻陷的情况下，目标完整性仍得以维持。通过将智能体对齐从行为属性转变为结构性强制系统约束，PEA为自主智能体的治理提供了稳健基础。

Identifiability and Stability of Generative Drifting with Companion-Elliptic Kernel Families（伴椭圆核族生成漂移的可识别性与稳定性）

本文分析了邓等人提出的生成漂移框架中，驱动分布匹配的漂移场的可识别性与稳定性。首先，引入伴椭圆核类，该类包含拉普拉斯核，其特征是该类中每个核κ与其伴函数η之间存在二阶椭圆耦合。对于该类中的每个核及任意一对Borel概率测度，证明漂移场为零当且仅当两个概率测度相等。进一步证明该类恰好由高斯核和ν≥1/2的Matérn核组成。其次，通过构造反例，展示质量逃逸至无穷远而场趋近于零的序列；特别地，仅控制场范数无法保证弱收敛。然而，证明唯一可能的失效模式被限制在一维射线{c p:0≤c≤1}上。因此，通过对由核与目标测度定义的线性可观测量——内蕴重叠标量施加渐近下界，可恢复弱收敛。

Benchmarking Complex Multimodal Document Processing Pipelines: A Unified Evaluation Framework for Enterprise AI（复杂多模态文档处理管道的基准测试：面向企业AI的统一评估框架）

大多数企业文档AI实际上是管道流程（解析、索引、检索、生成）。每个阶段已被单独深入研究，但评估整个系统仍然困难。我们构建了EnterpriseDocBench来解决这一问题，统一评估解析保真度、索引效率、检索相关性和生成基础性。语料库来自六个企业领域（当前试点包含五个领域）的公开许可文档。我们测试了三条管道：BM25、密集嵌入和混合，均使用相同的GPT-5生成器。关键结果：混合检索略优于BM25（nDCG@5：0.92 vs 0.91），两者均优于密集嵌入（0.83）。幻觉率不随文档长度单调增长：短文档（28.1%）和长文档（23.8%）均高于中等文档（9.2%）。跨阶段相关性很弱：解析→检索r=0.14，解析→生成r=0.17，检索→生成r=0.02。这否定了质量级联假设。同时，事实准确率为85.5%，但答案完整性平均仅0.40——系统回答正确但遗漏内容，这一差距对实际部署更为关键。我们还描述了三个参考架构（ColPali、ColQwen2、基于代理复杂性的路由），但尚未实现端到端集成。框架、指标、基线和收集脚本将在论文被接收后开源发布。

IntentVLM: Open-Vocabulary Intention Recognition through Forward-Inverse Modeling with Video-Language Models（IntentVLM：通过视频语言模型的正向-逆向建模实现开放词汇意图识别）

本文提出IntentVLM，一个用于开放词汇人类意图识别的两阶段视频语言框架。该方法受认知科学中的正向-逆向建模启发，将意图理解分解为目标候选生成和结构化推理选择，有效减少了潜在推理中的幻觉。在IntentQA和Inst-IT Bench数据集上，IntentVLM实现了最先进的性能，准确率高达80%，比基线性能提升30%，并达到人类水平。结果表明，这种结构化推理方法在不发生灾难性遗忘的情况下增强了开放词汇意图理解，为人本机器人学提供了坚实基础。

Beyond Shortcuts: Mitigating Visual Illusions in Frozen VLMs via Qualitative Reasoning（超越捷径：通过定性推理缓解冻结视觉语言模型中的视觉错觉）

本文提出了结构化定性推理（SQI），一种无需训练、以数据为中心的框架，旨在增强冻结视觉语言模型（VLM）的视觉基础。SQI通过三个系统模块解决视觉错觉问题：（1）公理约束注入，抑制错误的度量估计和定量幻觉；（2）层次化场景分解，将目标视觉流形与复杂背景干扰解耦；（3）反事实自我验证，一种对抗性推理步骤，以减轻确认偏差。通过在推理时协调这些定性约束，SQI有效对齐了高层语言推理与低层视觉感知。该框架在DataCV 2026挑战赛（第一类任务：经典错觉理解）中荣获第二名。实验结果表明，SQI不仅显著提升了各类错觉任务的准确率，还在无需任何模型微调的情况下提供了优越的诊断可解释性。本研究的成功凸显了结构化定性基础作为开发下一代抗错觉视觉语言系统的稳健范式的潜力。

Leading Across the Spectrum of Human-AI Relationships: A Conceptual Framework for Increasingly Heterogeneous Teams（领导力跨越人机关系光谱：面向日益异构化团队的概念框架）

当人类与人工智能协同工作时，什么决定了关键决策？答案正变得模糊。本文提出了一个面向领导力的光谱模型，用于审视有限边界内的人机关系：纯人类、半人马（人类主导，AI参与）、共等、弥诺陶洛斯（AI主导，人类参与）和纯AI。

该光谱聚焦领导工作的核心：谁定义问题、谁引导方向、谁为结果负责。这五个位置是帮助领导者识别决策中配置的锚点。核心风险在于误认：当决策权已转移，领导者可能仍保留以人为中心的故事；可能误以为监督仍有效（实际已流于形式），或让人类参与反而会恶化决策。

该框架引入了“共适应性”——即配置随人机共同调整而改进的能力，并将其置于异构团队（参与者可能在数量、载体、模型架构、能力、速度、记忆和参与形式上各不相同）中。

目标是务实的：帮助战略领导者及AI系统设计/部署者识别当前配置，察觉其转变，并判断其是否适合手头决策。这些配置将塑造权力、责任与信任在组织中的分配。它们所助力的未来是否可控且值得居住，取决于领导者能否尽早看清关键决策实际形成的方位与方式。

RESTestBench: A Benchmark for Evaluating the Effectiveness of LLM-Generated REST API Test Cases from NL Requirements（RESTestBench：评估大语言模型从自然语言需求生成的REST API测试用例有效性的基准）

现有REST API测试工具通常使用代码覆盖率和基于崩溃的故障指标来评估。然而，近期基于大语言模型的方法越来越多地从自然语言（NL）需求生成测试，以验证功能行为，这使得传统指标在衡量生成的测试是否验证了预期行为方面成为弱代理。为填补这一空白，我们提出了RESTestBench，该基准包含三个REST服务，并配有人工验证的精确和模糊变体自然语言需求，实现了对基于需求的测试生成的受控与可重复评估。RESTestBench进一步引入了一种基于需求的突变测试指标，用于衡量生成测试用例针对特定需求的故障检测有效性，扩展了Bartocci等人的基于属性的方法。利用RESTestBench，我们评估了两种方法（涉及多个最先进的大语言模型）：(i) 非精细化生成，以及(ii) 通过与运行中的被测系统交互引导的精细化生成。在精细化实验中，RESTestBench评估了接触实际实现（有效或突变）如何影响测试有效性。结果显示，当生成器与有缺陷或突变的代码交互时，测试有效性显著下降，尤其对于模糊需求，有时会抵消精细化带来的益处，这表明在需求细节充分时，纳入实际被测系统行为是不必要的。
